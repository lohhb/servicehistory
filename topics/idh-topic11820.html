Service History

Subject: NS-6040 LD/ULD motor cannot turn, control panel no response

Handler Model: NS-6040 (NS60-25, S/N: 181610)

Controller: SRC520

Date: 28 & 29 Apr 2010


 

Symptom

UTAC  NS60-25, s/n:18610(off warranty) LD/ULD motor cannot turn, control panel no response.


 


 

Action

28 Apr 2010

1. LD/ULD induction motor unable to turn when triggered from SPEL95.

 Checked and found address on IO module incorrect.

2. Index Head 5 thermo fuse blown.

3. Shuttle 2 missing 1x roller stay A and 2x mini cam-follower.

4. Operational panel no response after home completed or cycle finished. Suspect UTAC member used incorrect recovery cd version 2.0 to recover the HDD and restored with old system data from the floppy diskette. To confirm hardware configuration of this handler(CPU upgraded) and perform another recovery with version 2.1 cd.


 

29 Apr 2010

1. Perform recovery with Recovery cd version 2.1 (FD04S015).

2. Upgrade MMI to 2.0c12.

3. Restored DU CT gain and CT offset.

4. Handler able to home and cycle run without hang up.

5. Teach all robot pose(input/output arm, shuttles, index arm and empty tray arm).


 


 

Cause

LD/ULD induction motor unable to turn when triggered from SPEL95 due to physical address on IO module incorrect.


 

Operation Panel no response maybe due to wrong recovery cd or incorrect/incompatible SPEL System backup.


 


 

Remarks

Recovery CD FD04S015 is for (EasyRestore) Win98, MMI 2.0c1, SPEL 95 2.53i

To use with PC-686BX CPU board.
