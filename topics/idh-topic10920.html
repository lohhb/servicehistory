Subject: RS232 board setup in SPEL, NS-3000 ATC

Handler Model: NS-3000 ATC (NS303)

Controller: RC520

Date: 7 Sep 2009


 

Symptom

RS232 board setup in SPEL for NS-3000 ATC (NS303).

need your advise on Contec COM-4(LPCI)H RS232 board installed for NS-3000 ATC.

 

I installed the Contec COM-4(LPCI)H RS232 Communication board in the PCI slot of RC520.

Using the COM Setup Disk Ver. 1.9 CD, I was able to install driver for this board.

In Windows 2k Device Manager, under [Ports(COM & LPT)], Contect Communication Port COM4, COM5, COM6 & COM7 was automatic added.

Also the [Multi-function adapter], "CONTECT C0., Ltd,-COM-4(LPCI)H" was added automatically, see picture below.

 

For "CONTECT C0., Ltd,-COM-4(LPCI)H" the I/O address in Windows 2k is E400H, see picture below.

For "CONTECT C0., Ltd,-PO-64L(PCI) the I/O address in Windows 2k is E000H, same address as in Output SPEL I/O Board #4 (7Segment LED), see picture below.

For the IFUN100 address, I checked in Device Manager, no conflict.

 

1)  I am not sure for the SPEL CT side,

a) do I change the setting of Output SPEL I/O Board #4 (7Segment LED) to same address as "CONTECT C0., Ltd,-COM-4(LPCI)H" that have the I/O addess E400H?

or b) do I add a Output SPEL I/O #5 for this "CONTECT C0., Ltd,-COM-4(LPCI)H"?

No, ATC program scan COM4 and above automatically.


 

 

Or if you have the procedure on the software installation and setup of the "CONTECT C0., Ltd,-COM-4(LPCI)H" in SPEL, please send to me.

 

2) I only need to setup on the Output I/O using HWConfe.exe right?

There is no need to add Input I/O for "CONTECT C0., Ltd,-COM-4(LPCI)H" right?

Yes no need to add, but you need check IFUNT100 higher address at Windows 2k and SPEL.

 

3) AS for Input Bit I/O label for Input Bit 40, 41, 42 and 43, the Input I/O label "ACT_COL1", "ACT_COL2", "ACT_COL3", "ACT_COL4", do I need to key in manually in SPEL? or it automatically add in the Input labels after I install the ATC installtion CD?

Pls install ATC program from ATC installation CD.

 


 


 

Action

PCI card installation, Only effect remaining PCI cards.

Pls check IFUNT100 and CONTEC I/O address(check whether the addresses were changed by Windows 2000.

If no changes to these address by Windows 2000 then no need to change SPEL setting IFUN100 and CONTEC I/O.


 

Contec PO-64L(PCI) address in Windows 2000 must be the same as SPEL Hardware Output I/O Board for 7segmentLED


 


 


 

Cause


 

Remarks

Contec COM-4(LPCI)H is Communication card for ATC temperature controllers.

Contec COM-4(LPCI)H appear as "PCI Simple Communication Controller" under [other devices] in Windows 2000 Device Manager before the correct driver is installed

Use COM Setup Disk ver. 1.9 (CD by Contec) to install driver for Contec COM-4(LPCI)H found in CD ROM path  \pci\comdrv\com2k_pi.inf


 

Contec PIO-64L in RC520  is to control 7 segment LED panel only(for binning display).
 This card only output signal from RC-520
