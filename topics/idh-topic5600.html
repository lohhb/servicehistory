Subject: NS-6040 CPU Board Upgrade

Handler Model: NS-6040 (UTAC, NS60-20T)

Controller: SRC520

Date: 9 Jan 2008


 

Symptom

CPU upgrade on NS60-20T(PC-686BX/Win98).

Handler hang after tried running with backup setup files.

Upgrade MMI version from 1.3a to 2.0c12, encountered convert fail error at end of installation. Suspect MMI already corrupted which caused problem in upgrading MMI.

Unable to launch MMI so unable to export setup files through MMI.

Perform setup files backup using Backupsan (MMI & SPEL closed).

Perform CPU Board upgrade, windows recovery, restore system and project and import setup files through backupsan. Handler hang after tried running with backup setup files.


 


 

Action

Suspect files corruption, perform recovery and recreate new setup file.

Perform binning check using dummy tester and verify handler pick and place alignment with 12x12 BGA devices with changekit.


 


 

Cause

Setup files already corrupted before CPU Board upgrade.
