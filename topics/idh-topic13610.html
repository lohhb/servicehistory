Subject: HMI P10.11-SAS, NS-8000 during Initial Start, never check for device in socket issue

Handler Model: NS-8040 / NS-8080 using HMI P10.11-SAS

Controller: RC520

Date: 6 Oct 2010


 

Symptom

ASE HMI P10.11-SAS, NS-8000 during Initial Start, never check for device in socket issue.


 


 

Action

Regarding NS-8000 during Initial Start, never check for device in socket issue actually is for Continuous Start:

-during offline without device, found out handler after one cycle + skip start , observed handler in Continous Start Index Arm did not go down to socket and check for device.(Index Arm standby above shuttle 1).

 Whereas for NS-7000/ NS-6040 during Continous Start, Index Arm 2 will go down to socket and check for device.

-according to Hidaka-san, only with device, then Index Arm will check for device in socket.
