Service History

Subject: EPSON typeI GPIB tester interface driver

Handler Model: NS-8040(Infineon, NS-100, S/N: 181213)

Controller: RC520

Date: 7 Jan 2009


 

Symptom

Customer informed temperature status, temperature setpoint , site map and ID commands cannot work on NS-100. "EPSON type1" was selected for GPIB interface.

This does not happen for NS-001.


 


 

Action

Found that customer did not select “EPSON typeI” for GPIB tester I/F in NS-100.

Change GPIB tester I/F to “EPSON typeI” , ok.

Refer How to change Tester Interface name for GPIB in NS-8000 series.


 


 

Cause

Found that customer did not select “EPSON typeI” for GPIB tester I/F in NS-100


 


 

Remarks

"EPSON typeI" is required for temperature status, temperature setpoint , site map and ID commands.

Infineon requested for tester interface setting to rename “EPSON typeI” to”Infineon typeA” to avoid confusion.

HMI F10.12-SIS
