Subject: NS-8040 index arm hang when doing contact height calibration

Handler Model: NS-8040(S/N: 181801, NS80-34)

Controller: RC520 (S/N: 03-15014)

Date: 5 Mar 2012


 

Symptom

UTAC - NS80-34 index arm hang when doing contact height calibration.


 


 

Action

- Check error history, no error.
- Check index to shuttle z-height calibration parameter, ok.
- Tried height calibration offline, ok.
- Observed index compliance double devices sensors’ height not adjusted properly.
- customer will fine tune sensor position and release handler and monitor online setup.


 


 

Cause

Index compliance double devices sensors’ height not adjusted properly caused handler hang up during Autocontact height.
