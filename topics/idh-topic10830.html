Subject: ASE - Running Infineon LF2BGA189 BGA 8x8 M2900 low yield

Handler Model: NS-8080 / NS-7080

Controller:

Date: 1 Sep 2009


 

Symptom

ASE - Running Infineon LF2BGA189 BGA 8x8 M2900 low yield.

Took some photos of the contactor w/ nest, dockplate and sockets. Currently ASE using NS-7080 to run this product, however they encounter high contact failures if index arms run at high speed. Right now their setup is running index arms at 40%. We had tried in Ns-8080, also same issue. They feedback Infineon gave them open socket, as I observe there is no device guide in the socket, purely rely on dockplate pins' guidance.

 

Would like to find out from you, last time when u buyoff this product in infineon using NS-8040 did you encounter this issue?

Seems like only the contact nest area is not manufactured by OEM (KA)

The rest of the kit (shuttle/hotplate..etc) is from KA


 


 

Action

3 Sep 2009 email from Iwata

Found that ASE use 3rd party contact blade.

We ask ASES to use KA’s one so that we support to find the root cause.


 

10 Sep 2009

1) Solution for setup M2900

 with open concept socket unable to run high speed for index arm, resulted in high conti failures. - K.A to provide new plunger with bellow cups.


 

2) NS-8080 index chattering error

 K.A propsed all existing NS-6040/NS-7000 contactors require modification on contactors to resolve vacuum leakage problem.


 

3) NS-8080 QFN 4x4.5 having flying device in initial setup

Adjusted individual index counter air purge and require to lower down index and shuttle speed to 80%. - Goto-san to provide recommended setup for various packages from EPSON.
