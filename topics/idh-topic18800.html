Subject: NS-8080 High UPH 8 Hands unable to Home due to 2ack error on Output Hands

Handler Model: NS-8080 High UPH 8 Hands (S/N: 182120, H24-NS88) 

Controller: RC520 (S/N: 03-30057)

Date: 23 Jan 2013


 

Symptom

23 Jan 2013 - ASESG - H24-NS88 NS-8080 8IO S/N: 182120 Handler unable to Home due to 2ack error on Output Arm.


 


 

Action

1) Tried Home in Z-Teach program.
2) Input Arm OK but output arm encountered 2ack error.
3) Manually pull down individual hands on output arm and felt vibration for hand c.
4) Replaced SKP 480/ Driver Board and SKP 481/ Resolver Board.
5) Tried Home output arm in Z-Teah program ok.
6) Replaced output hand c, suspect hand c causing the problem to come back.
7) Performed master calibration on output arm.
8) Cycle run handler and release to customer.

Raised warranty replacement S0538 for the following:
1) SKP 480/ Driver board
2) SKP 481/ Resolver Board
3) Z-axis module Output Hand C
