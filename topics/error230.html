Error 230   MCORG has not been completed.

Causes

After a controller has been initialized using DIP Switch 1-1, MCORG is required before you can execute MCAL. If you want to restore the previous MCOFS settings without executing MCORG, then follow these steps:

1.    From the Setup menu, select Robot Controller Configuration.

2.    Select the MCOFS tab.

3.    Change any value with the original value and click OK. For example, if home sensor logic type is 4, then delete the 4 and type in 4. This is done so that EPSON RC+ will send the MCOFS parameters to the controller when you click OK. If nothing is changed, then when you click OK, EPSON RC+ will not send anything to the controller.

You can now continue without getting error 230.
