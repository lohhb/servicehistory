Ans

The modification is only single shuttle support software.

I don't think improved to detect controller error and also get feedback from thermal controller when having error.

Because the specification sheet does not include the matters.


 

In that case, what is the differences to the current v2.6 HMI? Now also

disable shuttle 1 in setup and run only shuttle 2.


 

Note: Even if Index Arm 2 is disabled by Shuttle selection, With Device, during or after One Cycle, Index Arm 2 will still go down to test site to check for device in socket.
