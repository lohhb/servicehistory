Service History

Subject: NS-8080 Index unable to pick up during height cal and output shuttle 1 detection sensor error

Handler Model: NS-8080 (S/N: 181748, NS8080-04)

Controller: RC520 (S/N: 03-05023)

Date: 14 Dec 2011


 

Symptom

NS8080 Index unable to pick up during height cal and output shuttle 1 detection sensor error.


 


 

Action

Setup BGA 27x27 with 1x2 single site.

Found wrong dummy unit used.

Correct unit was thicker.

Also set compliance unit in setup to 55mm instead of 40mm.


 

- Found output shuttle 1 emitter height lower in position so when device in pocket was in line with side sensor, reading on amplifier was 0.

Adjusted the height of emitter higher.

 Reading with device in pocket less than 3000 and more than 10, ok.

- Cycle run with dummy units ok. No error.


 


 

Cause

Index Arms unable to pickup device during Auto Contact height due to wrong dummy device used.

Output Shuttle 1 detection sensor error due to Emitter height was too low causing timing sensor amplifier reading is 0 when there is device in pocket.
