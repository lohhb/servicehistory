Service History

Subject: Microsoft jet engine error when loading HMI.

Handler Model: NS-8040 (NS-105, S/N: 181394)

Controller: RC520

Date: 3 Mar 2011


 

Symptom

Microsoft jet engine error (ODBC) when loading HMI.


 


 

Action

Reinstall HMI, NG

Copy & paste handler.mdb.dsn and handler.mdb into C:\NS8000\Env

Reboot handler, ok

Offline run with devices, ok


 


 

Cause

Error log file corrupted maybe due to illegal shutdown of handler or power trip.
