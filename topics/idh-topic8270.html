Service History

Subject: Device floating on input Shuttle

Handler Model: NS-8080(ST Muar, S/N: 181416)

Controller: RC520

Date: 22 Dec 2008


 

Symptom

During production, encounter Input shuttle device floating error.

NS-8080 is using optical fiber sensor.


 


 

Action

-Adjusting the IC detecting sensor of the loading side shuttle

-Make sure that no device is placed in the pocket of loading side shuttle.

-Switch the channel selector switch of the loading side shuttle.

-Hold the mode key on the sensor amp for more than three seconds.

The output of the sensor amp is adjusted automatically.

When the adjustment completes, the digital display shows approximately 1000.

-Production monitoring for 5X5 package


 


 

Cause
