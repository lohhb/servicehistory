Service History

Subject: NS-6040  encountered”condition file access error” during launching MMI

Handler Model: NS-6040 (S/N: 15682, NS-6040-14)

Controller: SRC520

Date: 8 Aug 2012


 

Symptom

SEP – NS-6040-14(S/N: 15682) encountered ”condition file access error” during launching MMI.

Only NS-taskbar appear with the ”condition file access error” dialog box.
 CPU Board with Pentium MMX 200MHz. 2.5" Removeable IDE HDD.
Win95(J), MMI 1.4a-JPN, SPEL95 2.53i.


 

Was informed by customer error happened after customer copy  & paste setup file using windows explorer.


 


 

Action

Uninstall MMI.
In SPEL, Project Close. Project Delete.
 Re-install MMI will also install back Project.

Copy Pos data back to SPEL Project.
 Create setup file name and import setup file one by one.

Reload IFUNT100 driver by Loader.exe
Home handler.
Dry run handler.


 

Remarks:

No need to touch on SPEL because only setup file issue.
