Service History

Subject: J0405 Outtput shuttle 1 detection sensor error

Handler Model: NS-7000 Loan handler S/N:18825

Controller: RC520

Date: 19 Jun 2008


 

Symptom

NS-7000 Output Shuttle 1 detection sensor error.


 


 

Action

During troubleshooting found out that output shuttle 1 photoelectric switch faulty.

Part was replaced using SCS spare and dummy run carried out.

No abnormalities found.


 


 

Cause

Output shuttle 1 photoelectric switch faulty.


 


 

Remarks

Check points for Output Shuttle Sensor Detection Error:

- photoelectric switch(timing sensor)

- resit connector for timing sensor

- any uneven slide plate of output shuttle

- tunnel block missing on the output shuttle slide plate

- any hole on tunnel block

- Gate Sensor Board

- Input IO Module for Timing sensor

- Timing sensor positions( X direction and height)

- Use of retrofit kit, tick "Use Side Device Exist Sensor" in the Device Exist Sensor Window
