Subject: ASE NS-8080(H32-NS88; S/N: 182233) Output Arm placement of device drift in -Y direction for Auto1 to Fix3

Handler Model: NS-8080(H32-NS88; S/N: 182233)

Controller: RC520

Date: 1 Oct 2012


 

Symptom

ASE NS-8080 Output Arm placement of device drift in -Y direction for Auto1 to Fix3.


 


 

Action

Replaced 400W Servo Motor for Index Output Y Axis.

In SPEL CT CT Command Execution Windows, execute the following commands:

( ">" indicates commands are typed in the Command Execution Windows)

>selrb 1

>MCORG 2   ' (for Index Robot 1, Y Axis)


 

Dry run.

Running check with dummy devices in low and high speed.

Run production.

Warranty raise S0527.


 


 

Cause


 


 

Remarks

The calibration of motor can also be carried out using

C:\NS8000\Tool\AutoTeach.exe => select [Calibration] tab
