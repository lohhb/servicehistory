Service History

Subject: NS-8040 Tray End button no response

Handler Model: NS-8040

Controller: RC520

Date: 10 Oct 2011


 

Symptom

UTAC – NS80-45(transferred handler) Tray End button no response(phone support).


 


 

Action

-Checked Tray End button SPEL Input bit 296 signal from Front and Rear Operation Panel ok.
-Checked (240)EIN30 I/O module 16CH below loader found connector not inserted properly.

Re-sit ok.


 


 

Cause

Loose connection on Input I/O Module (240) EIN30 below loader.

Later stage, this I/O module relocate to near vacuum valve for Empty Tray Arm.


 


 

Remarks

Email by Hidaka, 25 May 2011:

To change EIN30 label of 240  Input Module 16 CH (located at Loader area)

EIN30    -> OP IN

EOUT30 ->OP OUT


 

Refer 0331-NS8K.


 

IO Bit Number = 48 + IO label number

288= 48+240

  

EIN30   16CH (OP IN)  240

 

EOUT30   16CH  (OP OUT) 240
