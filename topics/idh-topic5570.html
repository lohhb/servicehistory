Service History

Subject: NS-8040 Double Devices Detection Sensor Adjustment

Handler Model: NS-8020/NS-8040

Controller: RC520

Date: 12 May 2008


 

Action

The Double-device detection sensor is for NS8020/8040.

This sensor has two purposes:

1) for IC stack condition detection in socket (Double-device),

2) for Index arm height calibration (Contact height detection).


 

Install Socket Guide Base jig and Contactors.

The sensor position has to be adjusted to contact height calibration position.

Contact height calibration position is from full stroke position(touch Socket Guide Base jig) above 1mm.


 

<Method>

Eg. Arm 1 case:

1) selrb 1

2) motor on

3) mcal

4) motor off

5) Manually move Index Arm Down to full stroke position by hand(with break release).

6) Move Up index arm 1mm from full shrink position by SPEL CT.

Eg. ARM1 case:

SELRB 1

MOTOR ON

GO P*+z(1)     ; for Arm 2 is   Go P*+u(1)

7) Activate the E/P regulator by supply air to expand compliance unit using I/O bit  (Out 240 - 249) for E/P regulator, eg On Output Bit 246.

8) Adjust(using size 13 spanner) the double-device detection sensor height from turn off condition to turn on condition.

The point of setting, full stroke position is double-device detection sensor turn off and

from full stroke position to 1mm up position is sensor turn off condition.

From 1mm up position to 2mm full expand position is double-device detection sensor turn on condition. Compliance unit gap fully open is 2mm and double-device detection sensor turn on.

Adjustment is that's all.


 

<Motion>

For index arm height calibration.

Index arm detect double-device detection sensor turn on(Orange LED ON) condition from turn off condition.

This is from full stroke position from 1mm up position.

After that, index arm calculate more 0.5mm up position as contact height.

So, index arm compliance unit shrink 0.5mm when contact.


 

For double-device detect function.

Index arm detect double-device condition when device thickness over 1.0mm.

This function effective when device thickness from 0.5*** to under 1.0mm.

Eg.  Device thickness is 0.6mm. When double-device situation is device thickness is 1.2mm. 0.2mm is over 1.0mm.

Double-device sensor detect that socket has double-devices.


 

Note:

Arm 1 all heads ( head 1 and 2) or Arm 2 all heads( head 4 and 5) has to be adjusted to same sensor position, which mean that the same Index Arm Double Devices detection sensor turn off at the same time.

If one Head is lower (less than 1mm up position), contact height has priority to near (lower) sensor position.
