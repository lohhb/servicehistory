Subject: NS-8040 SRC error when input arm set to open close pitch in Z assignment. Input arm run in random when input arm set to fixed pitch in  Z assignment.

Handler Model: NS-8040 (S/N: 181590)

Controller: RC520

Date: 11 Oct 2010


 

Symptom

Reported by EEMS during production NS-8040 encountered SRC error when input arm set to open close pitch in Z assignment and input arm run in random when input arm set to fixed pitch in Z assignment.


 


 

Action

-Found that tray form x-pitch set to 42mm will have SRC error but below 42mm pitch running normal when set to open/closed pitch in Z assignment.

-Confirmed that input arm running random when input arm set to fixed pitch in Z assignment .

-Tired Epson setup file was same situation.

-Tired out on NS70000 no issue.

-Tired out on NS8080 and NS8040 same issue.
