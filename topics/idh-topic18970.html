Service History

Subject: Device pickup error at Input hand. Device pickup error on output Shuttles

Handler Model: NX1032XS (S/N: 182347)

Controller: RC520

Date: 3 May 2013


 

Symptom

Device pickup error for Input Hand G occurred frequently.

Device pickup error at output Shuttle 1, but there was no devices on the Output Shuttle 1 & Output Hands.


 


 

Action

Cycle run 5x5 devices in ambient & hot mode for 32 sites

Device pickup error for Input Hand G occurred frequently, increased the vacuum check wait time to 0.15sec.

Device pickup error at output Shuttle 1, but there was no devices on the Output Shuttle 1 & Output Hands, adjusted the receivers position, ok.


 


 

Cause

Device pickup error for Input Hand G due to device size is small so need to increase the Vacuum Check Wait Time.(However, later stage, the air tubing was found broken).

Device pickup error at output Shuttle 1, but there was no devices on the Output Shuttle 1 & Output Hands due to output device detection sensor not adjusted properly.
