Service History

Subject: NS7KW-03 Detection Sensor Error for Output Shuttle 1

Handler Model: NS7080W ( NS7KW-03, Xilinx)

Controller: RC520

Date: 27~29 Feb 2008


 

Symptom

Output shuttle 1 having “Detection Sensor Error” upon starting of handler.


 


 

Action

27 Feb 2008

Found output shuttle jig(Changekit by A.P) screw protruding out when tightened. Remove screw OK – Inform to Alan.


 

Adjusted output shuttle 1 top(emitter) and bottom(receiver) beam sensors with reference to shuttle jig.


 

Monitor production, still often have “Detection Sensor Error” at Output Shuttle 1.

Due to tight production schedule, Alan unable to down handler so temporary disable shuttle 1 and continue the following day.


 

28 Feb 2008

Scheduled handler offline by customer. Setup FBGA 23x23 using K.A octal changekit. Verified with dummy units, found problem same as (day before) production setup. Adjusted Output Shuttle 1 1-S condition sensor(emitter) position and continue verification. Verified with 2000 units in Ambient setup, no more occurrence of “Detection Sensor Error”, released to customer.


 

29 Feb 2008

Reported by customer, production setup FBGA 23x23 high temp last night having many “Detection Sensor Error” for output shuttle 1.


 

Feedback by customer, handler converted to BGA 19x19. Verified handler while tester having PM. Cycle run with dummy units, found out “Detection Sensor Error” still happen in hot mode. Check condition of output shuttle 1 slide plate uneven flatness when shuttle jig is place on top. Suspect problem lies with the output shuttle slide plate which misread by shuttle condition sensor. Informed to customer require to have new replacement for shuttle slide plate.. Temporary added 0.03mm shim under sensor block and requested K.G to send quotation to customer. Release to customer and monitor for any improvement of problem until new part arrive.


 


 

Cause

Output shuttle 1 slide plate uneven flatness when shuttle jig is place on top.

Suspect problem lies with the output shuttle slide plate which misread by shuttle condition sensor.


 

Remarks

Slide Plate B  H01-F12W-012

Mentioned part for NS-7080W, part name and part code taken from NS-7080W/NS-7160 Maintenance Part List Rev.1 Drawing number 7NSW05.


 

Check points for Output Shuttle Sensor Detection Error:

- photoelectric switch(timing sensor)

- resit connector for timing sensor

- any uneven slide plate of output shuttle

- tunnel block missing on the output shuttle slide plate

- any hole on tunnel block

- Gate Sensor Board

- Input IO Module for Timing sensor

- Timing sensor positions( X direction and height)

- Use of retrofit kit, tick "Use Side Device Exist Sensor" in the Device Exist Sensor Window
