Subject: Remove “SKIP” option when device stuck (JAM0315) occurred

Handler Model: NS-6040(ASE(M))

Controller: SRC520

Date: 3 Sept 2008


 

Symptom

Remove “SKIP” option when device stuck (JAM0315) occurred.


 


 

Action

For the Jam error code 0315 ("Device stuck"), "RETRY" and "SKIP" will be displayed upon seeing this error.

Customer requested not to have this "SKIP" lighted up when this jam error occurred.

Thus, I had made the necessary alteration to this jam error code.

I had also taught ASE engineer on how to alter these error code, as they need to change all handlers not to have this "SKIP" lighted up when error code 0315 came on.
