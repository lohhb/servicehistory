Subject: NS-8080 8 hands Input Arm pickup error on loader Tray

Handler Model: NS-8080  (NS88-04 s/n: 181655) 8 Hands High UPH

Controller: RC520

Date: 2 Nov 2010


 

Symptom

EEMS NS-8080 8hands (NS88-04 s/n: 181655) Input Arm pickup error on loader Tray.


 


 

Action

Customer feedback pickup error on loader getting worse.

Checked handler running condition. Found Hand E and F pickup on loader N.G.

Verified hand pitching with shuttle alignment jig, carried out tune tune from lower hand adapter.

Verified handler running condition again, found out X-way drift when input arm pick up towards end of X-way. Edited SPEL point of -0.3mm.

Found out another issue with loader tray lock cylinder. Tray lock cylinder unable to position tray properly before input arm pick up units from tray.

Suggest customer to request additional tray lock pusher from KGS.
