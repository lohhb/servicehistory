Service History

Subject: NS8080-01  8 Hands output arm hand up sensor error

Handler Model: UTAC NS-8080 (NS8080-01, S/N: 181665)

Controller: RC520

Date: 16 Feb 2011


 

Symptom

UTAC NS8080-01  8 Hands output arm hand up sensor error.


 


 

Action

16 Feb 2011

- Informed by Ray at 9am hand up sensor error on output arm. Last activity for hand up sensor error issue was on 13 Jan 2011.

- Advised Ray to check for abnormalities on every hands and perform hand height calibration. After that, he encountered z-axis servo error also.

- Checked handler at 11am but no more z-axis servo error. However, still got frequent output hand up sensor error.

- Checked output arm z-axis hands movement ok. Checked the arm assembly for any loose parts/connections, ok. Push and slightly shake the top cover of arm assembly.

- Resumed production and monitored.

- From 1130am to 530pm(end lot), total 4 counts of hand up sensor errors.

- Feedback to Hidaka san, he said Japan will release software for handler to detect which hand servo error but hand up sensor error not sure when can get solution.


 

17 Feb 2011

- Encountered hand up sensor issue again and also RS485 2ack error and RS485 unknown error.

- According to Hidaka san, no advise from Japan for hand up sensor error. But Japan released new HMI ver.1.8 for improved detection of z-axis servo error on particular hand.

- Installed HMI ver.1.8 and SPEL CT 3.11dh-sp. Tested on handler and able to prompt which hand z-axis servo error. Also included individual hand home sensor IO check in HMI and zteach.

- Continue monitoring.


 

18 Feb 2011

- Checked error history and encountered few counts of hand up sensor error on hand G for night shift.

- Started to encounter hand G sensor error again at 12pm. When error happened, press retry start will lead to output arm z-axis alarm code=6 error and RS485 unknown error. Had to clear error with pause reset start.

- Also observed when error happened, by pressing the hand G sensor connector, lightly knock on home sensor or adjust/resit sensor wiring, the HMi hand G I/O will light up.

- At 2pm, disable site 7 for hand G. Run till 330pm no hand up sensor error, but 1 count RS485 unknown error.

- At 330pm, swapped hand G and hand E sensor to see if problem migrate to hand E. Resume production at 4pm.

- Continue running till 615pm no occurrence of hand up sensor error.


 

19 Feb 2011

Last action swapped hand G and hand E home sensor. No problem during the weekends. So root cause still unknown.
