Service History

Subject: Silabs - NS-7000 QFN 3x3 floating on input shuttle due to Input Hand D pitch out

Handler Model: NS-7000(S/N: 18732)

Controller: RC520

Date: 9 Apr 2009


 

Symptom

QFN 3x3 device floating at Input Shuttle due to Input Hand D pitching out.


 

Action

Use shuttle alignment jig to align back hand pitch.

Fine tuning and running check with device for ambient and hot mode.


 


 

Cause

QFN 3x3 device floating at Input Shuttle due to Input Hand D pitching out.


 

Remarks

Model : NS-7000

Serial: 18732

MFG: Mar 2004

PC-686BX (PC)-LV PIII 850MHz

Factory : EPSON-M

CU  S/N : 01-01379

DUBOX 010 S/N: 230176 NF

Silabs is using 3 PG Boards C-864v1


 

Remarks

This is related to 0133-NS7K, 0151-NS7K and 0161-NS7K


 

PG Board C-863 and C-864v1 are not compatible.


 

PG#1 control Z motion of Input Hand A, B, C and D.

PG#2 control Z motion of Output Hand A, B, C and D.

PG#3 control Open/Close motion of Input Hand and Output.
