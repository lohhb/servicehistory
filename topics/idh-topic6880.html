Service History

Subject: Jam 0405 Detection sensor error at Output shuttle 1

Handler Model: NS-7080W

Controller: RC520

Date: 30 May 2008


 

Symptom

Jam 0405 Detection sensor error at Output Shuttle 1.


 


 

Action

Found out that Jam 0405 is “Detection sensor error at Output shuttle 1” and confirm that the error occurred when the device wasn't sit properly by trying out using NS7080W.


 


 

Cause


 


 

Remarks

Check points for Output Shuttle Sensor Detection Error:

- Index Arm counter-air too strong

- photoelectric switch(timing sensor)

- resit connector for timing sensor

- any uneven slide plate of output shuttle

- tunnel block missing on the output shuttle slide plate

- any hole on tunnel block

- Gate Sensor Board

- Input IO Module for Timing sensor

- Timing sensor positions( X direction and height)

- Use of retrofit kit, tick "Use Side Device Exist Sensor" in the Device Exist Sensor Window
