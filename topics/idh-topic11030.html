Service History

Subject: NS-7000 Input Arm(selrb 2) cannot HOME, Input Arm cannot move and no error issued.

Handler Model: NS-7000

Controller: RC520

Date: 5 Nov 2009


 

Symptom

Marvel(M) NS-7000 Input Arm(selrb 2) cannot HOME, Input Arm cannot move and no error issued.


 


 

Action

Checking:

- in SPEL CT able to home Index, Output Arm, Empty Tray Arm, Shuttles but cannot home Input Arm.

- check Encoder cables connections behind Drive Unit, ok

- check PG Board connections, ok

- Check MMI I/O monitor for Input Hand A, B, C & D, all HOME LED on.

- Power On at Front Operation Panel, pull Input Hand A and release, Input Hand A spring back up. There was no torque holding the stepping motor for Hand A.

Input Hand B, C & D after pull down but will hold at its' current position( Hand B, C & D never spring back), ok

 

- Swapped connector for Input Hand B stepper motor to Input Hand A stepper motor.

Now Input Hand A with motor torque, Input Hand B does not have motor torque.

Swapped back connectors

- Swapped stepper drivers for Input Hand A&B.

Problem migrate to Input Hand B for no torque.

 

Conclusion:

Need to replace stepper driver to confirm.


 


 

Cause

Faulty stepper driver.
