Item requested: Linear Guide (For Index Arm 1 Z Axis)

Part Code: R35N700014700 (HSR12R2UUME+270LM)


 

Symptom

SRC520 error at Index Arm 1 when running production. Index Arm 1 Z Axis linear guide found stuck and with rough movement.

Upon home, heard a sound from index arm 1 and the error came on. Thus, tried to release brake S1, so as to move the index arm 1 manually but looks jammed in Z axis.  Tried exerting force to lower the arm 1, but could not move at all.  Did a check on the ball screw and Z-axis motor, found that all is working.  Further check found that Z-axis linear guide was causing the problem.  A point noted is that there was no grease on this axis (very dry).  This grease-less issue might have cause stiffness to this axis, thus causing some roughness to surface on the Z axis linear guide. Linear guide was replaced and all linear guide was checked for grease.  All index area linear guides was dried and grease had been applied to them.  Setup LQFP  20x20 package and ran 10 trays of dummy without problem.
