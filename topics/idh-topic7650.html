Service History

Subject: NS-7000 Hand up sensor error on input arm hand 'C'

Handler Model: NS-7000

Controller: RC520

Date: 8~10 Sept 2008


 

Symptom

NS7000 (Getech Office) – Hand up sensor error on input arm hand 'C'.


 


 

Action

Remove cover of input arm and found no abnormalities.  Dry run without cover also have this problem.  Reduce speed to 90%, no more errors. Suspect stepper motor problem.


 

Hand 'C' replaced and dry run at 100% still have the same problem.  But problem improved. Previously, problem always happened when input arm pick from loader tray.  Now, problem happened less frequent.


 

After much comparison with the other 3 hands on the input, found out that the cable layout looks different on hand 'C' as compared to the other three hands, thus, cable rearranged and pulley system re-laid due to friction.  Dry run for 8000 contact, no error occurred.


 


 


 

Cause

When cover of input arm is removed, no problem. After cover input arm with cover then have problem.

Cable lug for grounding wire at input hand cause blocking to Hand.
