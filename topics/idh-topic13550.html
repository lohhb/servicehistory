Service History

Subject: SDT encountered "Target temperature cannot be set" message

Handler Model: NS-7000

Controller: RC520

Date: 20~24 Dec 2010


 

Symptom

SDT encountered "Target temperature cannot be set" message.

All temperature shows --- dash.


 


 

Action

1.   Re-install MMI but still same error.

2.   Checked yamatake board and rs232c connector/pin. No problem.

3.   Changed setup file, same error.

4.   Checked AMB only but same error.

5.   YAMATAKE board led status OK, same as good handler.

6.   HDD from #21, same error.

7.   Checked CPU Board com port conenctor/wiring. No problem.

8.   Checked COM port panel, found 1 pin loose.

Replaced new COM port panel, OK. No more error.


 


 

Cause

At RC520,Com port panel 1 pin loose cause  "Target temperature cannot be set" message.
