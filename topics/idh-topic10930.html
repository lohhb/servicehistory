Subject: AMD ATC with NS-3000, keyboard unable to use

Handler Model: NS-3000

Controller: RC520

Date: 27 Oct 2009


 

Symptom

Just to update on AMD NS3000 keyboard issue. Suzuki-san said that the Y-split cable now using is from NI or TI something like that and not compatible with our PS2. AMD tried to use UBS Y-Split cable and can use.

Originally, we suggested to use USB keyboard.

But after discussing with AMD, this is not good solution for USB keyboard because if during HD recovery, USB keyboard cannot use and need to go BIOS for some setting.

AMD do not like idea of going BIOS to change setting during HD recovery just to use keyboard.

 

Therefore:

1) If we buy locally, we do not know which type is compatible. AMD bought themselves 3 and all 3 cannot use.

2) Better to get from EPSON. No need for special shipping now. Can ship together with any spare parts.
