Subject: NS-8040 HMI incomplete load up. Only load NS-Task bar.

Handler Model: NS-8040 (NS-007, S/N: 181834)

Controller: RC520

Date: 21 Feb 2011


 

Symptom

HMI incomplete load up. Only load NS-Task bar.


 


 

Action

End Task by Ctr-Alt-Del.

Manually start SPEL, ok.

Manually start HMI, ok.

Select individual setup file, until select (LTX)_WF2SGA121-1_S_AMB, SRC file load error (Getkey)!!: C:\NS8000\Run\Contact.dat


 

Delete setup file (LTX)_WF2SGA121-1_S_AMB.

Import setup file  (LTX)_WF2SGA121-1_S_AMB.

Restart HMI, ok

Dry Run handler, ok.

Reboot handler, dry run, ok.


 


 

Cause

Setup file corrupted caused incomplete launch of HMI.

Corruption maybe due to force shutdown of handler.
