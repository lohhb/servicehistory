Service History

Subject:  W03290 "High Contact Force Attachment is not set properly" error

Handler Model: NX1032XS (S/N: 182347)

Controller: RC520 (S/N: 00504), DUBOX030 (S/N: 100004), PSUNT353 (P892)

Date: 24~25 Oct 2012


 

Symptom

During Loadcell Setup, encountered W03290 "High Contact Force Attachment is not set properly" error (HCF brackets).


 


 

Action

HMI 1.41 released to patch  this software bug.


 


 

Cause

HMI 1.4 software bug.


 


 

Remarks

EPSON stated +/- 18N for contact tolerance per site, Xilinx converted to +/- 1.5Kgf.


 

HMI 1.41 fixed (26 Oct 2012)

bug of the "W03290 High Contact Force Attachment is not set properly" error.


 

HMI 1.4 fixed (23 Oct 2012)

1. Don't execute Short Index motion(Socket Y Pitch bigger than 63.5mm).

2. Modified the bug of the rotation stage control parameters.

3. Modified the bug of the Time Stamp Error.

4. Modified the bug of the Output Arm step out when access fix tray. (placement of devices outside of Fix trays)
