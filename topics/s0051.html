Item requested: Stepping Motor Driver

Part Code: R35N700003600 (PMD07c-RP701567)


 

Symptom

Output Arm Hand D cannot HOME, stepper motor driver never generate power. Stepper motor driver also feels hot.


 

Remarks

This is related to Warranty Parts Requisition S0052, S0053 and S0054.

Technically, R33ZTA2080200 is the code for NS6000 and R35N700003600 is the code for NS7000.

However, to avoid any confusion, KG(S) use R33ZTA2080200 to prepare the item with their system.

R35N700003600 is not a mistake and can use it with no problem.

According to Epson both are for stepping motor driver (PMD07C), but only "R33ZTA2080200" is actually used. So pls use R33ZTA2080200, not R35N700003600.
