Service History

Subject: NS-8040 Abnormal humming noise from Input Arm Y-Axis motor & Output Arm Y-Axis motor when Homing, vibration from both Y-Axes motors

Handler Model: NS-8040 (NS-103, S/N: 181671)

Controller: RC520

Date: 2 Dec 2009


 

Symptom

Abnormal humming noise from Input Arm Y-Axis motor &  Output Arm Y-Axis motor when Homing, vibration from both Y-Axes motors.

During dry run 50% also humming noise and vibration can be felt from the front of handlers when touching the covers for the motors.

At 80% speed the humming sound were even louder.


 


 

Action

Checked the CT Data in SPEL CT, also remove Drive Unit 2 and check servo driver CT Data labels, ok.

Power & signal(encoder) cables connection checked ok. Motor position marking checked, ok.


 

Adjustment to both input/Output Arm Y motor timing belts tension still not OK.

Epson will send tuning software. To try out following day.


 

3 Dec 2009

1) Installed Custom manipulator tuning program Ver 1.4 for SPEL CT.

2) Adjustment to Servo Parameters, joint 2, Proportional gain and Torque filter time constant value. Input Arm joint 2 torque filter time constant from 

1750 to 850 and Output Arm from 1500 to 850.

3) Humming noise for Output Arm seems Ok when running in 70% speed. However Input Arm encountered Overcurrent.

4) Interchanged Motor drivers in DU for Input/Output Y motor. Problem migrated to Output Arm.

5) Requested to Goto-san for new motor driver replacement


 

9 Dec 2009

1) Replaced DU slot 2 with new DMB board and new motor drivers for input/output arm Y axis. Restored back original manipulator values. Results: Noisy.

2) Changed back original slot 2 with new motor drivers for input/output arm Y axis.

3) Replaced input/output arm Y axis motors. Results: Noisy.

4) Re-adjust motor belts tension. Results: Noisy.

5) Adjustment on manipulator Torque filter Time constant and Proportional Gain. Cycle run handler in 50% speed, input arm seems ok but output arm problem same. Increase input arm speed to 70%. results ok not noisy.

6) Continued adjustment on output arm manipulator values, still unable to achieve good result.

7) Requested by ijima-san to change AC/DC board in DU. Replaced AC/DC Board from another handler. Result still same.

8) Pending decision by Hidaka-san and Goto-san.


 

11 Dec 2009

1) Assisted Hidaka-san on checking of handler.

2) Measured resistance of  DU slot 2 motor supply power connector to motor drivers.

Results no short-circuited.

3) Measured both Y-axis motor incoming power connectors to power connectors to DU. No cross pathed.

4) Measurement on another handler. Results and reading same.


 

14 Dec 2009

NS-8040 transported to Getech for troubleshooting.


 

12 Jan 2010

With Kamijo & Ogata-san troubleshooting for Y Axes noise issue:

-use Servo Parameter Editor check Input Arm X(Joint 1) & Y(Joint 2) Axes, found some values are zero with reference with data provided by EPSON

-change Input Arm Joint 1 & 2 values using values provided by EPSON

-use Servo Parameter Editor check Output Arm X(Joint 1) & Y(Joint 2) Axes, found some values are zero with reference with data provided by EPSON

-change Output Arm Joint 1 & 2 values using values provided by EPSON

-replaced Input Arm Y motor & pulley assembly

-check & adjust short timing belt(170¡À15Hz) & long timing belt(18¡À2Hz) according to EPSON spec using the frequency tension meter

- calibration of Input Arm Y Axes motor by AutoTeach-Calibration tool

-replaced Output Arm Y motor & pulley assembly

-check & adjust short timing belt(170¡À15Hz) & long timing belt(18¡À2Hz) according to EPSON spec using the frequency tension meter

- calibration of Output Arm Y Axes motor by AutoTeach-Calibration tool

- Run handler in 10% & 100% speed, ok, no abnormal noise.


 

18 Jan 2010

NS-8040 return to Infineon for buyoff.
