Service History

Subject: UTAC- ESD Class 0 installation on NS70-21 and LAN conflict will parallel port in BIOS cause RC520 hang up(Blue screen)

Handler Model: NS70-21

Controller: RC520

Date: 2~4 Sep 2009


 

Symptom

UTAC- ESD Class 0 installation on NS70-21( 2nd set) and LAN conflict will parallel port in BIOS cause RC520 hang up(Blue screen).


 


 


 

Action

Hardware installation:

- 4x ION ionizer fans with grounding/close loop and power cables.

- close loop sensor board

- chamber ionizer controller

- chamber relay unit

- 4x ionizer fans power adaptors


 

Hardware and software installation:

- Installed MKS controller

- install input, chamber and output sensors

- install new MMI version, pci card driver and MKS software


 

Encountered blue screen again after installation of MKS software, load back last known good configuration of window 2k ok.

Ethernet unable to recognize and had to disable parallel port in BIOS and install ethernet driver again.


 

MKS software unable to find address of com port.

Found out that pci card dip switch did not set.

Remove card and set correct dip switch position ok.


 

Hardware and software installation completed. Awaiting STM schedule for static and dynamic buy off.


 


 

Cause

LAN setting conflict with parallel port in BIOS cause PC hang upon installing LAN driver.
