Service History

Subject: NS8080-01 input arm hand c and d servo and RS485 error

Handler Model: NS-8080 (S/N: 181665, NS8080-01)

Controller: RC520 (S/N: 03-02140)

Date: 31 Oct 2011 & 1 Nov 2011


 

Symptom

UTAC - NS8080-01 input arm hand c and d servo and RS485 error.


 


 

Action

31 Oct 2011

- Found somebody took out hand C motor cable and did not secure back properly.
- Input arm vac threshold was set to –20,-25,-70. Set back to default –35,-40,-70.
- Did hand height calibration and ran dry cycle for 2hrs.
- Handler encountered RS485 2ack error and unable to home and do hand height calibration during setup at 4pm.
- Input hand C and D moved up and down when homing and hand height calibration.
- Found SPEL input bit #48 led light did not turn off when pulled down hand C and D. Rest of the hands when pulled down, SPEL input bit #48 able to turn off.
- Disconnect hand C and D home sensor connector from distribution board then can turn off SPEL input bit #48.
- Checked resistivity of hand C and D home sensor wires, same as other hands.
- After resit connectors a few times able to cycle run.
- But when start again to try and run dummy units, encountered hand C servo error. Feel both hand C and D no torque.
- Open spel IO, pulled down hand C and D, input bit #48 still ON (abnormal).
- Swapped hand C and hand A home sensor connectors.
-> SPEL Input Bit #48 (Input Hands still ON when pull down hand C.
-> SPEL Input Bit  #48 turned OFF when pulled down hand A.
- Suspect home sensor of hand C and D to have problem.
- Feedback to Hidaka san and waiting for advise.


 

1 Nov 2011

- Heat up to 85 deg and cycle run. One cycle and start handler again, encountered input arm hand C servo error.
- Checked and found hand C and D no torque.
- Found SPEL input bit #48 led light did not turn off when pulled down hand C and D. Rest of the hands when pulled down, SPEL input bit #48 able to turn off.
- Swapped hand C and A home sensor. Problem migrated to hand A.
- Replaced both hand A and D home sensor with spare.
- Home and performed hand height calibration ok.
- Continue dry cycle running. Released to Fuzari and under monitoring.
