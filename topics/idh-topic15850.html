Subject: NS8080-01 index arm vacuum unit 2D and 2E caused initial start error

Handler Model:  UTAC NS-8080 (NS8080-01, S/N: 181665)

Controller: RC520

Date: 25 Apr 2011


 

Symptom

UTAC - NS8080-01 index arm vacuum unit 2D and 2E caused initial start error.


 


 

Action

Line resolved by swapping vacuum units with NS8080-19.

Customer confirmed vacuum unit settings correct.

Checked on vac unit 2D found setting incorrect, P1 set to 35 instead of –35.

Vac unit 2E always display ‘8’. Reset vacuum unit by pressing up+down button. Ok.

Swapped both units to running sites 2A and 2B, ok.

Release handler.
