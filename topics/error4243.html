Error 4243   Improper speed reference is generated in the High-power state.


 

The  main  CPU  detected  that  the  Manipulator's operating speed has exceeded the specified setting in the High-power state.

Check the Manipulator

(1) if the arm moves around smoothly when it is moved around by hands,

(2) if the arm's movement is tight and secure when it is moved around by hands,

(3) if there is any hitches with the arm when it is moved around by hands,

(4) if the belt is not slackened when checked disassembled and

(5) if the arm does not hit the brake when it is moved around while the electromagnetic brake is turned off.

Check if the Manipulator's arm is not colliding with the peripheral units around.

Replace the motor driver.

Replace the Drive Unit.

Replace the motor in case of motor/encoder failure.

Check on the cable connection between the control and  peripheral  units; 

Check  on  the  connectors: EMERGENCY, D-I/O and Expansion I/O;

Check on the short-circuits with the connecting cables or/and inside the peripheral units.
