Subject: UTAC NS70-15 and NS70-29 input arm not in random pick up mode

Handler Model: NS-7000

Controller: RC520

Date: 30 Jun 2010


 

Symptom

Advise customer to recreate new setup files with imported original good setup file. Suspect previous setup files corrupted.
