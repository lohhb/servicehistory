Subject: NS-7000 Tower light no light and no buzzer sound

Handler Model: Silicon Labs  NS-7000(S/N: 18732) MMI 1.1c8, SPEL CT 2.81b  PSUNT286 

Controller: RC520

Date: 30 Apr 2010


 

Symptom

Tower light no light and no buzzer sound.


 


 

Action

Checked fuse on SKP 429 board, found out fuse blown.

Swapped Output I/O module 8 Channel(Lamp) in PSUNT286 with tray out, same symptom.

From Output I/O module 8 Channel(Lamp, SKH0755-B12), Output Bit 168 to 175 link to relays(KA100 to KA107) in SCP 429 to trigger tower light and buzzer (SKH0755-B07).

Checked tower light cable, found some exposed wiring, only OUT171, 24V and GND have continuity, the rest open.

Bring back tower light for verification.


 

Day 2:

-tower light bring back office for verification, functioning check ok

-tower light cable(about 2.5m) prepared in office

-Installed new tower light cable to SCP 429 board(6 wires + 2 buzzer wires as separate connectors)

- tower light verification found Green light not working in correct order, found that "Relay Logic at Lighting] for [Tower light setting] on GREEN was set to 'N Close', so set to 'N Open' in MMI.

 SR S/N: 012064


 

Cause

Broken tower light's cable.

Wrong tower light setting.
