Service History

Subject: NS-7080 Automat Mat4 & Mat10 tray finger bend and Index Device drop

Handler Model: Xilinx NS-7080W(NS7KW-01, S/N: 121286)

Controller: RC520

Date: 8 Apr 2010


 

Symptom

Automat error. Tray out error.

Device drop during auto run and pick up error arm 1 during height calibration. Now disabled arm1 for production run.


 


 

Action

Found Mat4 and Mat10 tray finger bend. Knock back finger.

Found transport (traverser) arm P1, P2 and home out.

Calibrate transverser robot 10, teach P1 and P2.

Verified Mat1 to Mat10 with traverser P1 and P2 by SPEL. OK.

Activate all vacuum of index arm. Found arm 1 site H weak vacuum.

As Index arm 1 moves from point to point, found site H vacuum cut off above test site.

Suspect index arm tube for Arm 1 H site broken.

Found site H tubing cracked at joint.

Enabled both arms, Auto Contact Height calibration. OK.

Running check auto 1 and automat Mat4 and Mat10. OK.

Released to production.


 


 

Cause


 


 

Remarks

selrb 9 for Elevator. (Calib 3)

selrb 10 for Traverser (Calib 2)


 

Check points for Index Arm Device drop:

- vacuum cup condition, device drop more frequently in hot mode?

- o-ring worn out on plunger

-Check diameter/tolerance of guide pins on socket guide plate

-Check any broken/bent tubing or tubing pressed by cable tie on SLK

-Check air fitting/elbow or gasket not loose on SLK

-Check if guide pins bent/worn out

-Check vacuum filter dirty

-Check vacuum filter put properly

-Check seal for vacuum put properly and not torn

-Check plastic casing for vacuum filter not broken

-Check nest guide pin's hole

-Check o-ring on SLK

-Check metal hooks of Socket Layout Kit (SLK)

-Check correct brand and specification of latch use for (Index) Test Hands

-Check metal latch worn out at Index Test Hand

-Check Index teaching position P50 and P51

-Check if Index Arm crash before or Index Arm bent(Index Arm alignment check)

- if device never drop, adjust vacuum switch sensitivity


 

Correct metal latch of parts No is TL-29A, this is supplied by Tochigiya.
