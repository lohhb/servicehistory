Subject: NS7080W Output arm crashed issue

Handler Model: NS7080W  (SDT, S/N: 121126, EPH#08)

Controller: RC520

Date: 25 Mar 2008


 

Symptom

Initially encountered “Detection Sensor Error”(unloading side). Customer while troubleshooting crash output Hand A. Hand A replaced but now, open pitch is N.G. According to SDT, Hand A is not 80mm pitch. Suspected side/moving plate also bend.


 


 


 


 

Action

Checked alignment for Output Hand A using Shuttle Alignment jig. Minor alignment on Output Hand A. Checked hand A condition.

Run dummy units BGA 4.5x4.5 for verification. Ok

Service report reference: 010270
