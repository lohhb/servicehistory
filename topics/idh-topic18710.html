Service History

Subject: In Hot mode, after One Cycle, disable one of the shuttle.

After that start with Continuous Start, Input arm will place devices onto the disabled shuttle continuously loading 3 to 4 devices onto the disabled shuttle 2 (double stacking of devices).

After that WAR0599 robot controller (RC520) error occurs for Input Shuttle 2 which is the disabled shuttle

Handler Model: NX1032XS (S/N: 182347)

Controller: RC520 (S/N: 00504), DUBOX030 (S/N: 100004), PSUNT353 (P892) 

Date: 22 Nov 2012


 

Symptom

In Hot mode, after One Cycle, disable one of the shuttle.

After that start with Continuous Start, Input arm will place devices onto the disabled shuttle continuously loading 3 to 4 devices onto the disabled shuttle 2 (double stacking of devices).

After that WAR0599 robot controller (RC520) error occurs for Input Shuttle 2 which is the disabled shuttle.


 


 

Action

EPSON release HMI ver 1.50 to patch the following:

1) Rectified the bug of the shuttle mode "One Side" - double stack device on unuse shuttle followed by torque error.

2) Rectified BackupSan tool which cannot backup

3) Changed the alarm map position of "Tray Ionizer Warning"

Handler.map in C:\NX1032X\Error\English


 


 

Cause

HMI bug in Ver 1.42.
