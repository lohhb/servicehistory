Subject: NS-7000 Contact Height Calibration problem with Single Site by using Dual Site with 1 site disabled

Handler Model: NS-7000

Controller: RC520

Date: 7 Jul 2009


 

Symptom

Customer encounter a contact height calibration problem on a NS-7000 and need advise.

When selected single site and above 60kgF, height calibration can only be carried out in torque control mode and not in direct contact mode.  But after height calibration, can run in direct contact mode without error.


 

Description:

Handler Configuration: -

Index max contact force = 120Kgf


 

MMI Ver.: 2.0J


 

Site used: Single (using dual site with 1 site disable)

Contact parameters:

Pin count = 1,932

Force per pin = 35kgf

Force per device = 67.62kgf

Contact Force = 67.62kgf


 

We try to do a contact height calibration(Direct contact mode) we are unable to proceed

with the error message "Select Torque control when the test site is single and the contact force is 600N or more"


 

We are able to perform the calibration with "Torque control mode"

But what trouble us is why there is this error when the Index max contact force is set at 120Kgf

while the Contact force for calibration is only 67.62kgf? (Refer to attached "contact error.jpg" for better understanding)

Can this be avoided?

Please advise on this.


 


 

Action

Refer to User manual for allowable setting range of contact force for standard 1200N.

Contact Force that set by customer is 67.62kgf but according to User manual, Instantaneous Max is 61.2kgf, which indicates that the contact force is out of range.

9 Jul 2009 email to Hidaka

Customer not able to use direct contact mode for the below config during height calibration. After select torque control mode, they were able to complete height calibration.

But after completed height cal, they switched back to direct contact mode, handler still able to run.

Why during height calibration cannot use direct mode but running mode can use?


 

9 Jul 2009 Check the setting at the NS-7000 in office

Finding:

1) Default MMI range for pin count is 5-1000, so change range value to 1,932.

2) "Direct Contact" cannot use same error msg as given by Elson

3) Contact mode changed to "Direct & Torque contact" then can do Auto Contact Height Calibration

4) Test Arm 1 and 2 showed contact force around 240N.
