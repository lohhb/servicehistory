Service History

Subject: NS7KW-03 at Xilinx -

1) Device drop on index arm 2 after test

2) Misplacement issue at unloader


 

Handler Model: NS-7080W

Controller: RC520

Date: 6 Feb 2009


 

Symptom

1) Device drop on index arm 2 after test

2) Misplacement issue at unloader


 


 

Action

1) During assessment, found out that device always drop at site 'H' (index arm 2).

Nest replaced, vacuum sensitivity adjusted and contact height carried out.

Furthermore, found out that 'O' ring on the SLK was wrong (slightly bigger and thinner), this may cause the device drop due to air leakage.

Thus, 'O' rings replaced with new one.

Performed pick-up and release calibration on Index Arms.

Fine-tuned index vacuum sensors' sensitivity. Site H of index arm 2 still have frequent device stuck error in socket.

Remove SLK and found out site H using KA's nest but the rest using FA's nests, request to change to FA.

Change SLK o-rings as some of the O-rings are different in dimensions.

Verified with x5 trays of dummy units ¨C No issue.


 


 

2) Upon fine-tuning, found out that release height very high (2.4) which cause device to drop mid-air, which indirectly caused misplacement. Release height fine-tuned to 1.3 and misplacement dissolved.


 

For 1) and 2), 5 trays of dummy ran without problem.


 


 


 

Cause


 


 

Remarks

Check points for Index Arm Device drop:

- vacuum cup condition, device drop more frequently in hot mode?

- o-ring worn out on plunger

-Check diameter/tolerance of guide pins on socket guide plate

-Check any broken/bent tubing or tubing pressed by cable tie on SLK

-Check air fitting/elbow or gasket not loose on SLK

-Check if guide pins bent/worn out

-Check vacuum filter dirty

-Check vacuum filter put properly

-Check seal for vacuum put properly and not torn

-Check plastic casing for vacuum filter not broken

-Check nest guide pin's hole

-Check o-ring on SLK

-Check metal hooks of Socket Layout Kit (SLK)

-Check correct brand and specification of latch use for (Index) Test Hands

-Check metal latch worn out at Index Test Hand

-Check Index teaching position P50 and P51

-Check if Index Arm crash before or Index Arm bent(Index Arm alignment check)

- if device never drop, adjust vacuum switch sensitivity


 

Correct metal latch of parts No is TL-29A, this is supplied by Tochigiya.
