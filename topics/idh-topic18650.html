Service History

Subject: NX1032XS Input Shuttles Heaters' circuit Breaker 3A tripped when temperature around 60~70 DegC

Handler Model: NX1032XS (S/N: 182347)

Controller: RC520 (S/N: 00504), DUBOX030 (S/N: 100004), PSUNT353 (P892)

Date: 11 Oct 2012  (Recovery DVD Ver 1.0,  FD12S164)

Date: 18~19 Oct 2012


 

Symptom

Input Shuttles Circuit Breaker (CB) 3A tripped when temperature around 60~70DegC.
 - Advised by EPSON to replaced SSR (Solid State relay)
- Advised through email by Serizawa-san to replace Circuit Breakers for Input Shuttles' Heaters from 3A to 5A.

 


 

Action

Serizawa informed that shuttle CB capacity is not match for shuttle heater wattage.

Actual situation now that is shuttle's CB is using 3A for CB203 and CB204.

However, correct CB type is 5A.

CB203 and CB204 should use 5A type.


 

5A breakers removed from handler in office NS-8040 and installed to NX1032XS, tested ok at 130DegC.


 

At 130DegC, dry run with devices did not trip breaker, but Index Arm 2 Heads starts to lose temperature when first plunge contact to Socket Guide Base jig(Default Socket Heater is unchecked/Disabled) and temperature control alarm occurred.

Indexs' heaters lose heat to Socket Guide Base jig.
 - Retry+Start but still temperature controller alarm. Retry+Start but still temperature for Heads never go up. Retry+Start, afew times also same problem
- Enable Socket Heater 1 & 2, then no problem.

 


 

Cause

Wrong rating of circuit breakers used for Input Shuttles for its' heaters.


 


 

Remarks

22 Oct email by Serizawa-san:

Regard to test head temperature drop,

As your information, test hand did not heating cold Socket Guide Base.

Because, test head is contacting to Socket Guide Base during test time.

This is heating time not enough and also test head heater capacity not include Socket Guide Base heating.

So, Socket Guide Base heating need by socket heater.

Could you confirm using socket heater?

If test head temperature drop situation will not solve using socket heater,

We have to consider for test head temperature drop.
