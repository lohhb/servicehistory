Service History

Subject: STM NS-8080 Input Shuttle floating error, empty tray pickup error

Handler Model: NS-8080 (NS8080-02, S/N: 181581)

Controller: RC520

Date: 18 Apr 2009


 

Symptom

1. Input shuttle device floating error.


 

2. Empty tray pick up error.

3. Found that the 3.6X3.6 contactor head nest was broken, now they temporary used 3x3 nest for production.- (1pcs broken)


 


 

Action

1. Found that the input arm hand X way A & C which pitching was slightly out, and B & D is no problem.

-Input arm alignment mechanically adjusted with shuttle jig.

-Fine tuning on loader and input shuttle.


 

2. Tried to adjust the hardware and MMI pick up setting, but the error still persist.

-Swapped the empty tray pick up hand with the evaluation machine.


 

Cause


 

Remarks

Sometimes, for some empty trays, the tray pocket for vacuum suction is not at centre of tray, so need to change "Pickup Up Position" (from usual 67.90mm) in [Tray Form], so that empty tray can be pickup.

So it may not be tray pad worn out or Empty tray arm position is out.
