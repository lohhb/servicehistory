Error 2   Syntax error

Causes

A SPEL statement has been improperly entered. Some causes of this error are:

Bad Keyword.

Incorrect parameters.

Bad function name or variable name.

A space or tab between function name and open parenthesis.

You are referencing an identifier created with a #define statement but the case does not match. #define macros are case sensitive.

Remedies

Correct the statement.

Tip To get help for any SPEL keywords while editing a program, put the insertion point anywhere in the word and press the F1 key.
