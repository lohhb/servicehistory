Service History

Subject: Silab NS-7000 s/n:18732 Input Arm Open/Close Error

Handler Model: NS-7000 (S/N: 18732)

Controller: RC520

Date: 23 Aug 2009


 

Symptom

Input Arm variable pitch intermittent open/close error.


 


 

Action

Execute MCAL command for Selrb7, Open/Close motor take long time to complete MCAL.

Input Arm homing takes longer time to complete due to open/close motor.

Swapped stepping motor driver for Input arm open/close with output arm, problem did not migrate.

Concluded Inarm open/close stepping motor OK.

Follow up again next day due to no access to spare part in office.


 

24 Aug 2009

1) Replaced stepper motor for input arm open/close.

2) Execute MCAL command in SPEL for Robot 7 ¨C OK.

3) Verified and adjusted opening pitch of 80mm.

4) Cycle run with dummy units in variable pitch OK.


 


 

Cause

Faulty stepper motor.

Stepping Motor with Gear    R33N700002600 


 

Remarks

This problem is not related to previous PG Board problem.
