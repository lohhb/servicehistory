Symptom

UTAC reported RS485 time out error issue on this handler(running 1x2 dual site hot temp, device I4748_19x19_570BGA). Every few mins will happen the error. Even after power cycle the handler, HOME and START.

 

After I check the jam history, found many WAR 01210 RS485 command error from morning 6+am.

I did not find anything unusual on the input and output arm, only hand H suction cup in bad condition, so replaced it.

After replacing the suction cup, run handler in dry cycle mode for 30 mins. No problem.

Request for dummy units and ran 3 trays. No problem.

Release to production and monitored 5 output auto 1 trays. No problem.

 

So in the end unable to see the problem and determine root cause.
