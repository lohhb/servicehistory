Subject: EEMS  NS-8080 8 Hands hand up sensor error after hand height calibration and hand pitch issue

Handler Model: NS-8080 (NS88-04 s/n: 181655) 8 Hands High UPH

Controller: RC520

Date: 28 Oct 2010


 

Symptom

EEMS NS-8080 8 Hands (NS88-04 s/n: 181655) hand up sensor error after hand height calibration and hand pitch issue.


 


 

Action

Verified input arm hand pitch.

Re-aligned hand pitch using 8 hands shuttle alignment jig.

Carried out hand height calibration on input arm.

Changed back to customer's semitron suction cup and release to production and fine-tune input arm pickup height on loader.

Intermittent pickup error on loader for loader pick up height. Advised customer to change new semitron suction cups if problem  persist.


 


 

Cause
