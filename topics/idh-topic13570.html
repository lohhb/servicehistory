Subject: NS-7000 Input Arm on loader tray pick up offset by 1 row of device

Handler Model: NS-7000

Controller: RC520

Date: 4 Jan 2011


 

Symptom

IDT another NS-7000, input arm on loader tray pick up offset by 1 row of device.


 


 

Action

First row pickup is always +y by 1 row, second row pickup, is actually 1st row of tray.

Tray matrix verified and correct. This is a production setup file.

Hotplate, shuttle and unloading, pick and place no offset.

Advised to offset Loader_Org, Loader_x and Loader_y y point accordingly.

 After minus 23mm for y point, pick up is ok.
