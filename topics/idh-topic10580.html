Subject: NS-7000 power tripped a few times

Handler Model: NS-7000 (S/N: 18724)

Controller: RC520

Date: 19 Jun 2009


 

Symptom

Handler always tripped when START is being depressed in ambient mode.  Unable to simulate it in ambient mode.


 


 

Action

 Suspect is the facility problem so I on heater and press START and it tripped whenever I put to hot and START.  Had advised IDT to change all to 'C' breaker, as Japan recommended all breaker to be type 'C'.


 

Strange question is why NS02/01 in PM room is running without problem? (NS02 is still in PM room and I was told to run it in 85deg and 105deg dry run till I knock off, before I knock off, handler is running fine)


 

Note: PM room using incoming 'B' breaker and outgoing 'B' breaker,   test floor using incoming 'C' breaker and outgoing 'B' breaker.
