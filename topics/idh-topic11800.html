Service History

Subject: SCS NS-6040 Windows 2000 CPU Upgrade encountered:

1) unable to install GHOST recovery

2) "Motion initialization error" occurred after loading SPEL

3) Power down status shown on the MMI status windows and control panel no light

4) Buzzer always sound once depressed ON button


 

Handler Model: NS-6040 (S/N: 15411)

Controller: SRC520

Date:


 

Symptom

Unable to install Ghost version recovery CD.

Suspect IDE cable and HDD not compatible so both items changed but still unable to install. Using Easy restore recovery CD found that CDROM driver was not installed and recovery aborted.

Changed a new CDROM and recovery CD able to install. "Motion initialization error" occurred after loading SPEL.


 


 

Action

Due to original HDD faulty therefore unable to retrieve backup data.

All 3 DU removed to write down the CT gain values.

After enter the values, error still exist.

Reset the DU and resit the MIB, problem solved.

Power down status shown on the MMI status windows and control panel no light found due to restoring of SPEL system.

Solved by reinstalled the recovery CD again.

During homing, "pose data values not specified error occurred.

All robots carried out mcorg and mcal. Dry run without abnormalities.

MMI upgrade to ver. C7.14-SSY.

Buzzer always sound once depressed ON button found that unloader 1 remote I/O unit faulty.


 


 

Cause

1) Unable to use Ghost recovery CD due to faulty CDROM drive.

2) "Motion initialization error" occurred after loading SPEL could be due to wrong/missing SPEL system data

3)Power down status shown on the MMI status windows and control panel no light due to restoring of SPEL95 System data into SPEL CT.

4) Buzzer always sound once depressed ON button found that unloader 1 remote I/O unit faulty.


 


 

Remarks

After CPU upgraded, SPEL System restored in SPEL CT. MMI status windows show "Power Down".

This is because the NS-6000 SPEL95 System data is not compatible to be restored into SPEL CT.

Solution:

Restore DU 1, 2 & 3 data in SPEL CT

Restore Robot Manipulator data in SPEL CT
