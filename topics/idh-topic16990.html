Subject: NS-8080 Output Hand G servo error and RS485 error

Handler Model: NS-8080 High UPH 8 Hands (S/N: 181725, NS8080-05)

Controller: RC520 (S/N: 03-05006)

Date: 27 Aug 2011


 

Symptom

UTAC – NS8080-05 output hand G servo error and RS485 error(phone support)


 


 

Action

1. Frequent RS485 error happened after pick up error on output shuttle 2.
2. Output hand G servo error.
- vacuum when output hands pick up at -70kpa. Vacumm a bit weak.
- previously hand offset was set too low by tech when handler cannot pick up on output shuttle.
- check hands height at rib position.
- perform hand height calibration.
- check hand z-movement not so smooth for output hand G and E. Temporary adjusted top shaft better but hand E still rough. However, hand E did not have servo error registered in servo log.
- hand g pick up on device a bit off centre. Adjusted by the 2 screws on hand tip.
Since more pick up error happened on shuttle 2, problem could be from index arm placement. Asked Ray to service up the SLK, socket, nest, contactors.
Check back with Ray on 28 Aug, no feedback from Line and the last thing he did was clean the contactors.
