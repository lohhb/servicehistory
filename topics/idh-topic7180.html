Service History

Subject: Index Arm1 tilted from test site, index rotation shaft's guide bush
was loose

Handler Model: TR60 (S/N:121175) NEC

Controller: RC520 ( S/N: 01-20947) 3 DU

Date: 2 Jun 2008


 

Symptom

Index Arm1 tilted from test site, index rotation shaft's guide bush
was loose.


 


 

Action

2 June 08 update for NEC TR60:
 - handler was was convertedto TSSOP8_COLD_-43 on 1 Jun 08
 - SRC520 controller error, SPEL CT error history shows error T-4302
for Manipulator 11, 10, 9, 8 all Joint 1.
 - Index Arm1 tilted from test site, index rotation shaft's guide bush
was loose.
 - temporarily applied Loctitle 242 to index rotation shaft's guide
bush and tighten bolts
 - remove Index Arm 1 & 2 assembly.
 -mount alignment jig for Index Arm1 & 2
 -mount "T" shape jig at test site
 - robot 1 teaching for P2:Z(0) and P1:Z(0) (jog & teach for X & Y) =>
using test site in -Z direction and "T" shape jig
 - robot 2 teaching for P2:Z(0) and P1:Z(0) (jog & teach for X & Y) =>
using test site in -Z direction and "T" shape jig
 -remove "T" shape jig
 -put in soat boat jig where the Index Arm pickup device position and
teach for P1 & P2
 -put in output shuttle jig and teach for output shuttle in position, P31
 -bolts tightening check at index arms
 -dry no tray run
 -run handler at 60% speed for production as there is no Loctite 648
(for index rotation shaft's guide bush) and Loctite 272 (for bolts
index)
 -will wait for HOT temperature device run to apply Loctite 648 and 272
as both can dry faster in Hot temperature.
