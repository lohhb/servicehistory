Service History

Subject: NS-6040 communication time out error on LTX MX tester at SCS

Handler Model: NS-6040 (NS6040-127)

Controller: SRC520

Date: 22 Dec 2008


 

Symptom

NS6040 (127) communication time out error on LTX MX tester.


 

Action

Fault was confirmed on 127 when running on LTX MX tester, but there is no problem on 13 and 49.  Dummy tester carried out on 127 found no problem.

The following was replaced with no avail:-

a) SRC power supply

b) created new setup file

c) SRC from 49

d) IFUNT interface card in SRC520

e) IFUNT interface box behind handler

f) reload type41.hex driver by using Loader.exe

g) IFUNT cable

Had advised SCS tester group to double check if the tester program is written according  to EPSON standard or not.

SCS also complained NS7000-16 and NS7000-17 had the same problem on this tester last time but was left unresolved.

From the above serial numbers, suspect that this NS6040-127 , NS7000-16 and NS7000-17 may have some hardware similarity which could have cause the problem.  Furthermore, NS6040-127 is the latest batch of NS6040 that SCS purchased.  Had already sent mail to get feedback from JAPAN on any hardware difference between the newest NS-6040 and old NS-6040.
