Q8 Want to adjust about device pickup height of Input arm / Output arm hand (2 hand or 4 hand).

[ Case study ]

Some hands press on device when device pickup by each hand differed pickup height.

Some hands pump up device when device pickup by each hand differ pickup height.


 

Q9 Want to adjust about counter air volume of Input arm / Output arm

[ Case study ]

Some hands blow off the device when it release on the tray due to counter air too strong.

Some hands can not separate the device when it release on the tray due to counter air too weak.
