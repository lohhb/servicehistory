Service History

Subject: NS-7080W after installed with Automat encountered:

1) Bin with Auto1 ONLY encountered Error 3 Subsidiary 13, error message is Tray lock cylinder (backward) error for Tray Unloader 3 Sensor

2) Input Arm rotator Hands cannot home

3) Torque error on the Index, Input shuttle and Empty tray

Handler Model: NS-7080W(S/N: 121040, NS7KW-06)

Controller: RC520

Date: 16 Aug 2010


 

Symptom

NS-7080W(S/N: 121040, NS7KW-06) that is installed with Automat in this August by EPSON sub-con) during dummy device run with FT256 17x17 Quad, Binning only with Auto1 encountered Error 3 Subsidiary 13, error message is Tray lock cylinder (backward) error for Tray Unloader 3 Sensor , which is either(SQ157) doesn't turn ON or The tray lock doesn't move backward. I did a printscreen of the error but forgot to copy to my thumbdrive.After add Binning to include Auto2, then no error.

How come this error comes up when now there is no Auto3 but Automat?


 

Kindly check with EPSON why?Wasted time trying to troubleshoot, looking for this cylinder & I/O module. This morning, after installed FT256 17x17 Quad site C/K, during Homing, Empty tray arm, Index Arm homed, but Input Arm move to inbetween Input Shuttle 1 & 2 stopped. Found Input Hand Rotator’s dog plate hit onto hardstopper, so cannot complete homing.In SPEL, try to MCAL rb7 & rb8 but but dog plate hit to hardstoper.Also restart handler but same error. Change to different setup files also same problem.Had to remove the stopper plate on Input rotator and MCAL, ok. Changed HOFS from -250 to 1370 and put back stopper plate.Restart handler & HOME, ok. However, the Input Hand rotator dog plate was totally out away from the sensor. In SPEL CT changed the rotator HOFS back to -250. Able to HOME. Dry run handler ok.  Back track to last week after setup & power up this handler, when HOME handler, encountered RC520 error for Empty tray Arm torque error, Input shuttle torque error, Index Arm torque error & etc. Restart a few times also same problem. Came back after lunch and don’t know why no more torque error. However, when trying to dry run without C/K, Automat error for slot plate protruding out of Automat occurred, had to adjust the sensor SQ212.

There is also an Automat error which I forgot what it was, need to check the MMI error.

Action

18 Aug 2010

-Encounterd F-5055 Servo overload B for Index Arm2, redo AutoContact height, ok


 


 

19 Aug 2010

ICOS installation by ICOS vendor

-Calibration & interfacing with handler

-Handler's 2L1(230Vac) & 2L2(0V) voltage tap for use to ICOS pc, UPS and for 24Vdc PSU

(Initially customer tap 110Vac, cause handler power trip due to overcurrent)

20 Aug 2010

-Install CP320 19x19 Octal kit

-During AutoContact height, Index Arm 1 & 2 after device pickup from Input Shuttles several device drop

-Check suction cups at Contactors, condition no good, worn out. Replaced 16 suction cups for both layout kit

-Check metal plungers for contactors, found small o-rings already harden and causing air leak. Plunger with o-ring directly drop through contactor, normal condition, the o-ring stop plunger from going throug the contactor

-Insufficent new o-ring for plunger, so only able to use Index Arm1 layout kit

-Pick and place alignment at Input side with CW rotation

-Pick and place alignment at output Shuttle to Auto1 with CCW rotation, device misplacement

-Found rotation drifting, placement rotation is different

-HOME handler, do marking at Output Hand, observe rotation for pick from shuttle and place on Auto1, rotation is not consistent

-Pause handler, check that Input Rotation Hands is lock but Output Rotation is not locked(no torque), able to rotate Output Hands.

-Front panel power off & on few times, sometimes during front panel power on Output Rotation Hand is not lock, suspect either rotation stepper motor is faulty or stepper driver faulty

23 Aug 2010

FG320 19x19 Octal kit

For Ouput Rotator Motor is free sometimes during Pause:

-swapped another stepper motor but same condition of Rotator motor is free sometimes during Pause

-swapped stepper driver DRV21 for Output rotator hand to stepper DRV20 for Input rotator hand, problem migrate

-replaced small o-ring for metal plunger x16(due to Index Device drop after pickup device in AutoContact height)
