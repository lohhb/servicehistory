0331-NS7K  (Compiled by Loh HB)

Previous Top Next

Service History

Subject: 1) Customer feedback index time longer than NS8080, about 3s.
2) Even at 0s test time, index arm is not moving.
3) Handler transferred from calamba, unable to adjust speed. Greyed off.
4) Previous visit by EPSON engineer found system or data file corrupted because new HDD, system and data restored, speed unable to adjust.

Handler Model: NS-7000

Controller: RC520

Date: 15~16 Dec 2011


 

Symptom

STE NS7000 on call support
1) Customer feedback index time longer than NS8080, about 3s.
2) Even at 0s test time, index arm is not moving.
3) Handler transferred from calamba, unable to adjust speed. Greyed off.
4) Previous visit by EPSON engineer found system or data file corrupted because new HDD, system and data restored, speed unable to adjust.


 


 

Action

1) To carry out recovery of HDD with no/minimal restoration step by step.
2) However, long index time was not due to speed settings but index arm was waiting for input to load device. It was not slow but input arm is always moving to a bypass/waiting point before loading onto shuttle.
3) Recovery carried out but input/ouput stepper unable to home.
4) Hardware, wiring, parameters, IO settings/addressing all checked and compared, no abnormalities.
5) Used HDD from another handler and able to mcal for input/output but rotator for both remains unable to mcal.
6) #2 handler power on and able to mcal for all.
7) Recovery carried out again and this time, input/output able to mcal but rotator still unable to mcal.
8) During mcal, rotator arm is vibrating and does not mcal at all. Checked sensor and dogplate. Saw sensor unable to cut dogplate.
9) Dogplate removed and installed again. Rotator able to home.
10) Dry cycle and handler released to customer.


 

Cause


 

Remarks

Speed setting in MMI cannot be adjusted due to someone change setting in ArmCondition.def in C:\NS7000\config\Range.

So change the setting in "ArmCondition.def" for "Speed=100,100,10" & "Accel= 100,100,10"

Refer 0308-NS7K.
