Error 5055   Servo overload B.


 

The servo CPU detected an overload with the motor.
Check on the settings for the WEIGHT for the load value.

Check the Manipulator

(1) if the arm moves around smoothly when it is moved around by hands,

(2) if the arm's movement is tight and secure when it is moved around by hands,

(3) if there is any hitches with the arm when it is moved around by hands,

(4) if the belt is not slackened when checked disassembled and

(5) if the arm does not hit the brake when it is moved around while the electro-magnetic brake is turned off.

Check if the Manipulator's arm is not colliding with the peripheral units around.

Check on the hardware settings in SPEL CT.

Check on the main power voltage (INPUT voltage at the Drive Unit) and make sure that it is within the range.

Lower the operating duty factor (to under 50%).
