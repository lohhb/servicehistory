Service History

Subject: Uniwire disconnection upon power ON

Handler Model: NS-7000

Controller: RC520

Date: 23 Dec 2010


 

Symptom

UNISEM uniwire disconnection upon power ON.

Rear control panel cable exposed wire.


 


 

Action

Replaced rear control panel, OK.

Shuttle 1&2 sensor board error with red dot flashing on input side.

Floating sensor and gate sensor check wrt SPEL IN. OK.

Physical wring check, OK

I/O Module check on input side, OK

Sensor board replaced, same error.

REMOTE I/O reset and replaced, same error.

Shuttle 2 I/O module check, found dipswitch setting wrong.

Set according to good handler condition, no error.
