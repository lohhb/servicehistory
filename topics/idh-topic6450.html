Service History

Subject: CPU Board upgrade for NS-6040

Handler Model: NS-6040 (S/N:15620)

Controller: SRC520

Date: 14 Jul 2008


 

Symptom


 


 

Action

Upgrade on NS60-24T Type A SRC520 controller.

1. Upgraded MMI to v2.0c1 and SPEL to v2.53i, backup using backupsan, backup setupfiles through import/export function.


 

2. Replaced to new CPU Board, MIB Board x03 pcs, System panel II and 2.5 HDD.


 

3. Perform recovery, restore system using backupsan and restore setup files through import/export.


 

Encountered:

1) Head 6 thermo sensor faulty due to wire damaged

2) Emergency stop error due to  MIB2 and MMI3 no short connector fixed,

3) Found out handler unable to execute clean out at last input tray.

Check Input I/O Bit 198  from SPEL, unusual signs by depressing clean out button.

Swap PC I/O card from another handler ok.

Inform to customer, PC I/O card (Contec I/O board) faulty.


 

Dummy run QFN 5x5 device and release handler.


 


 

Cause

Faulty PC I/O card (Contec PO 32/32 L).


 


 

Remarks

For MIB 2 & 3 boards, dummy connector short pin 6-10, short pin 9-11. If no dummy connector E-stop error will occur.
