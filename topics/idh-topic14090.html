Service History

Subject: UTAC  NS8080-01 8 hands Input Arm frequent error

Handler Model: NS-8080 (NS8080-01) 8 Hands High UPH

Controller: RC520

Date: 17 Dec 2010


 

Symptom

UTAC  NS8080-01 8 hands Input Arm frequent error.


 


 

Action

Checked handler observer data.

Only found 1 count of RS485 2Ack error on input arm.

Lubricate all hands on input arm.

Found production enable input arm unitset device slow down of 0.5sec and index arm vacuum check wait on shuttle 0.8sec.

Rectified input arm, index arm settings to default and reduce input arm hands pickup height on loader.

Cycle run with units and found index arm 2 site F frequently have device pickup error on input shuttle. Found site F vacuum reading -30KPa when pickup device on shuttle. Upon error happen, able to retry as device was already on contactor and vacuum reading went up to -70KPa.

Remove arm 2 SLK and trigger vacuum for all sites, found out vacuum very weak and some vacuum units showing only -1, -2 and -3KPa.

Service vacuum pump filters and check again. Vacuum still low.

Increased vacuum volume on those affected vacuum units.

 Cycle run handler Ok.

 Released to production and monitor for 30mins OK.
