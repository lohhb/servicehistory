Service History

Subject: NS-8000 Center Single Site Testing

Handler Model: NS-8080(UTAC)

Controller: RC520(HMI 2.0)

Date: 31 Oct 2008


 

Symptom

HMI Software version installed into NS8080 evaluation system in UTAC is version 2.0.

We confirm this version should cover center single site testing which is requested Broadcom.

HMI version 1.8 & onward(incl. version 2.0) should have this function.

However, the reality in UTAC is it does not work well and found the error “W0422 Shuttle 1 sensor error (loading side) and W0522 Shuttle 2 sensor error(loading side).

SEIKO EPSON is investigating root cause of this error code and we are waiting for their finding.


 


 

Action

Perform single site handling setup for NS-8080 after received change-kit from K.A. Adjust floating sensors based on 2 sites configuration ( Shuttle 1 SQ112 & SQ113 and Shuttle 2 SQ114 & SQ115) at loading side.

Encounter both Shuttle 1 & 2 with sensor error.


 

Perform single site handling setup for NS-7000, exactly the same adjustment did on NS-8080 earlier on, did not encounter any shuttle sensor error.


 

18 Nov 2008:

Use sensor No.3 and No.5 for Center Single Site handling for NS8080.

Regarding for release position, NS8040 and 8080 is different position

NS-8040: Release on shuttle is center of Y direction

NS-8080: Release on shuttle is inside of Y direction.
