Service History

Subject: Sensor detection error occurred for Input Shutter upon Start pressed

Handler Model: NS7080 (Xilinx, NS7K-04)

Controller: RC520

Date: 31 Jan 2008


 

Symptom

Upon start, sensor detection error occurred for input shutter, running on octal site. 


 

Action

Found that transmitter and receiver  for Input Shuttle were not aligned properly and one of the receivers was not correct, that is, they used quad site receiver instead of octal site receiver.

All sensors for Input Shuttles re-aligned and problem do not exist anymore.

Fine tuning was carried out for this package too.


 


 

Cause

Input Shuttles' Floating sensors were not use correctly(Octal setup but end user use Quad site shuttle sensor) or not aligned correctly.
