Subject: MMI display "Door open" error after using a CPU upgrade Recovery CD on a normal NS-6040

Handler Model: NS-6040 (S/N: 18288)#51 Type 1

Controller: SRC520 (S/N: 01161)

Date: 7 Aug 2008


 

Symptom

Cleaned up of the handler (de-greasing and greasing of all parts).

Tried to on the handler but to no avail.

Initial problem is HDD problem, CMOS cannot detect HDD, thus HDD replaced.

After installation, able to detect HDD.

Used recovery CD (for CPU upgrade) to install Win98 and MMI and SPEL CT into new HDD, but upon startup of MMI, always have “Door open” error.

Tried a few times to un-install and re-install all software again but to no avail.


 


 

Action


 


 

Cause

The CPU upgrade recovery cd cannot be use on a normal NS-6040.


 


 

Remarks

HDD used was a 3.5" IDE 3.2GB.

Handler was refurbished in Mar 2010 and upgrade with PC-686BX with Win98.
