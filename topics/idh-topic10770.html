Subject: SCS NS-7000 MMI H7.11-SSY issues

Handler Model: NS-7000

Controller: RC520

Date: 4 Oct 2009


 

Symptom

SCS have some issues on the NS7000 software MMI ver H7.11-SSY.


 

1)     When “Tray arrival time up error” occurred, only “Alarm Reset” button was blinking at the control panel. No other options were available. Normally, there should be a “Retry” and “Alarm Reset” buttons option.

 

 –   In “Jam 1101” error code, only “Alarm Reset” was activated. This issue can be solve by activating the "Retry" option in the error file. But is error specially design for SCS software version?


 

 

2)     In normal circumstances, if a person reset or one cycle the system during production (Hot Mode/Quad site) and disable any of the sites. MMI Error “Jam 0111” will show on the screen. However, MMI Error“WAR0199” and SRC Error “I2031” appeared on screen if a person reset or one cycle the system during production (Hot Mode/Quad site) and disable either site B or site D. As for disable either site A or site C were Ok.
