Service History

Subject: Xilinx NS-7080W Shuttle thermal expansion

Handler Model: NS-7080W

Controller: RC520

Date: 12 Mar 2009


 

Symptom

10 Mar 2009 Floating Device on Unloading Tray Report


 

Customer:        Xilinx Asia Pacific Pte. Ltd. (XAP)


 

Problem:          Floating device on Unloading Tray


 

Handler:          NS7080W


 

Action Plan:    Select the newest handler as a “Golden Handler” (GH) with 4 packages

selected from XAP process team to be fine tuned by GETECH Systems

Solution.


 

S/N:                 121298


 

4 Packages:     FT256 17x17 @ 100 degrees with CW/CCW rotation

                        FG400 21x21 @ 100 degrees with CW/CCW rotation

                        FG320 19x19 @ 100 degrees with CW/CCW rotation

                        CP132 8x8 @ 100 degrees with CW/CCW rotation


 

Duration:         24th February 2009 - 5th March 2009


 


 

Findings w.r.t device floating issues as follows:


 

1)   Thermal Expansion: - This causes overall floating on unloading trays.

2)   Fix Tray: - Tray clamp design allows slight play of top right or bottom right alignment from handler rear. But it should be bottom right. In the event operator place tray top right, device will float.

3)   Position drift: - Substantial drift on Y direction but X direction drift is quite bad on CP132 8x8.

4)   Shuttle vibration: - This cause irregular pick up. Not centering causing floating during placement.


 

Please refer to next page for detail information on each finding.


 


 


 


 


 


 


 


 


 


 


 

Item 1: - Thermal expansion


 

1)   Heater “ON”. With XAP initial wait of only 60seconds.

2)   When device pick up, can see pick up not center in x-direction.

3)   When place on unloader tray, device floating in y-direction.

4)   Offset of x on shuttle pick up is -0.1mm.

5)   After this tray running complete, found about 20 devices floating on Auto Unloader 1.

6)   After this tray complete, allow 600seconds waiting time (actually 600seconds + 1st tray complete running is about 1800seconds total waiting).

7)   New offset of x -0.6mm, after 1800seconds, device pick up center and placement is “Good” on Auto Unloader 1. No floating of 1 full tray (360 devices).

8)   Initial wait time with shuttle in “HOME” position for thermal expansion.

9)   Conclusion is XAP need to increase initial wait from 60seconds to 1800seconds followed by normal alignment offset. Need to wait for thermal expansion stability before doing alignment offset.


 

Item 2: - Fix Tray


 

1)   Fix Tray has fixed guide pins for empty tray to be bottom right alignment.

2)   Possibility of operator to insert empty tray top right alignment.

3)   This case, device will float.

4)   Conclusion is need to highlight this human error possibility.


 

Item 3: - Position Drift


 

1)   This is a known problem of NS7000 capability to handle small packages as encountered in UTAC before.

2)   Both x and y direction, can been seen position drift between first row and last row.

3)   Substantial / marginal float in y direction.

4)   Position drift however, is not acceptable x direction on automat and fix tray.

5)   First row placement is ok. Last row placement, can be seen x direction more positive.

6)   Fix Tray Org and X-way point data, x-point reduced 0.4mm.

7)   Automat Org and X-way point data, x-point reduced 0.4mm.

8)   This drift is causing placement drift especially in CP132 8x8 package.

9)   This is possibly because device is smaller and more tray matrix as compared to FT256, FG400 and FG320.

10)             Running check conduction and able to achieve 1 full tray no device floating on Fix tray and Automat.


 


 


 


 


 


 

Item 4: - Shuttle Vibration


 

1)   Slight shuttle vibration causing pick up not center.

2)   Vibration is more especially when Input/Output arm moving at speed and accel 100%.

3)   When contactor arm pick from input side, also cause the vibration.

4)   With handler legs fully locked, able to reduce vibration.

5)   Conclusion is all handler legs must be lowered and tightened. As the test time generally for XAP product is long, to use “Auto Speed Down” option.

6)   For shuttle vibration, it seems it is happening to XAP handlers. Few handlers, shuttle motor gain settings changed to reduce vibration. Please check with Ogata-san for this issue.

Just a few things to note:

1) Initial Wait Time for these 4 packaged has been changed from 60seconds -> 1800seconds.

2) Anything to be done ONLY after 1800seconds upon switching from AMBIENT ->HOT.

3) During this initial wait time, please ensure handler in "HOME" condition. This is to ensure the output shuttles are in chamber.

4) Please note to operators/technician that for Fix Tray, the tray alignment is bottm right from the handler rear. If aligned top right, there will be device floating.

5) Please note to operator loading of empty tray into Automat is also critical. Slight tilt of tray in automat may cause device floating too especially for CP132.


 


 

Action

To verify thermal expansion with dial gauge.

Temperature @ 130 degrees and interval of 10 mins for data collection.

Output shuttle not in chamber.

Total 6 hours for data collection and found only -0.044mm expansion on shuttle.

Output shuttle itself is only @ 50 degrees.

To use EPSON method of shuttle teaching jig in chamber to verify again.
