Technical Support Notes

Subject: Rasco Pitch use on NS-6000/NS-7000/NS-8000

Handler Model: NS-6000/NS-7000/NS-8000 series

Controller:

Date: 24 Sept 2008


 

Symptom


 


 

Action

We would like to inform you for each of the handler on the possibility on using Rasco Pitch as follows.

According to your information, you want to cater for Rasco Pitch (X x Y=36 x 60mm)

1. For NS6000

As NS6000 does not test Square Quad site, it is impossible.


 

2. For NS7000

If we make SLK and C/K, you can test.

However, as min pitch for Input/output arm is 40mm, handler pick up/put on device one by one.


 

3. For NS8040

If we make C/K, you can test.

However, as min pitch for Input/output arm is 40mm, handler pick up/put on device one by one.
