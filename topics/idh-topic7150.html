Subject: NS-3000 ATC check on alarm for "Temp Ready" signal

Handler Model: NS-3000 ATC

Controller: RC520

Date: 1 Sep 2009


 

Symptom

NS-3000 ATC temperature ready check on alarm for "Temp Ready" signal.


 

Action

Suzuki-san, NS-3000 ATC check on alarm when "Temp Ready" (Temperature Ready) wire disconnected from ATC controller for MMI Active Cooling monitor set to Tester Start /Handler Start at ON/OFF condition.

(ATC Active Cooling always monitor when running).


 


 

Cause


 


 

Remarks

Temperature ready signal Function can only be seen in ONLINE mode.

KG(S) to confirm with AMD(S).

But temperature ready signal bit for each Head can be seen on the SPEL Input I/O.

ATC I/O in SPEL CT:

  

 

SPEL Input Bit

SPEL Label

Remarks

Head 1   Arm 1

40

ACT_COL1

All 4 I/O bit must be ON when ATC "Temp Ready" is connected (2 pin white connected behind ATC controller)

Head 2   Arm 1

41

ACT_COL2

Head 5   Arm 2

42

ACT_COL3

Head 6   Arm 2

43

ACT_COL4

When ATC have alarm the respective Bit will be OFF.

eg. when ATC Head 6 error, Bit 43 will be OFF.
