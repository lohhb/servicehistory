Service History

Subject: NS-8040 unable to use Recovery CD CD FD08S120 Ver 2.0 (11/18/2008) on 2.5” SATA HDD

Handler Model: NS-8040 (S/N: 181670, NS-003)

Controller: RC520 (S/N: 03-02120)

Date: 20 Jan 2012


 

Symptom

Infineon – NS-003 unable to use Recovery CD CD FD08S120 Ver 2.0 (11/18/2008) on 2.5” SATA HDD.


 


 

Action

During HDD Recovery by using Recovery CD FD08S120 Ver 2.0 (11/18/2008) , initially Ghost restoring data but suddenly writing error.


 

In DOS mode, it display
“Abort: 10008, Unexpected end of file
Batch file A:\REST.bat missing
Strike a key when ready...”


 HDD in this recovery is 2.5” SATA, previously recovery cd FD08S120 Ver 2.0 tested on 3.5” IDE HDD was ok.
Awaiting reply from Hidaka-san.


 

30 Jan 2012

Tried on our office NS-8080HW on Recovery CD FD08S120 Ver 2.0 (11/18/2008) with SPI-8550-LLVAS with DVI VGA port & 2.5” SATA 160GB HDD, ok.

Hidaka also confirm NS-8000 Recovery CD Ver2.0 and ,Ver2.1 can use on both CPU both SPI-8550-LA and SPI-8550-LLVAS with either 3.5” IDE 80GB HDD or 2.5” SATA 160GB HDD


 

EPSON(J) think that problem could be due to faulty CD-ROM drive.


 


 

Cause

CPU Board SPI-8550-LA + 2.5” 160GB HDD SATA in 3.5” cradle cannot use FD08S120 Ver 2.0

(There is no problem with the CD-ROM drive)


 

Follow up, refer 0416-NS8K.


 


 

Remarks

Recovery CD Ver 2.0  (FD10S083, 11/18/2008) can use for cream color RC520.

For SPI-8550-LA/SPI-8550-LLVAS (3.5" IDE or 2.5" SATA HDD), also for NS-8080HW


 

We have different Recovery CD which type controller is using.

White color: Ver2.0,Ver2.1

Black color: Ver 4.1
