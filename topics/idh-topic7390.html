Service History

Subject: HMI unable to start up completely

Handler Model: NS-8040(UTAC, NS80-26 S/N:181540)

Controller: RC520

Date: 8 Oct 2008


 

Symptom

HMI unable to start up completely upon power on handler. HMI Status windows always showing SPEL executing.


 


 

Action

Check SPEL ok no error.

Launch SPEL then launch HMI, error message displayed “Open error on C:NS8000\RUN folders. Suspect setup file corruption, change to another setup file then restart HMI.

Boot up ok, but error displayed” open error for Microsoft Jet database” (ODBC)  Inform Line engineer to check with engineering group due to this error is not due to handler.


 


 


 

Cause

1) HMI Status windows always showing SPEL executing due to setup file corrupted.


 

2) handler.mdb  corrupted in C:\NS800\Env  caused ” open error for Microsoft Jet database” (ODBC).


 


 

Remarks

Need to Copy & paste handler.dsn and handler.mdb in C:\NS800\Env   from another handler.
