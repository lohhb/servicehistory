Service History

Subject: NS-8000 Chamber ionizer controller EP3

Handler Model: NS-8000 series

Controller: RC520

Date: 20 Jul 2010


 

Q&A

Need to check with u regarding NS-8000 chamber ionizer controller EP3.

 

1.It can detect ion balance in the chamber right?

Ans:

No, it is no detection ion balance, just check cleaning condition by own sensor.

 


 

2.If the ion cannot balance due to pins dirty, the controller will trigger cleaning alarm right?

Ans:

Yes, anyway it can be detected cleaning alarm.

 


 

3.If the fan is turned off but ion balance still within spec, there won't be any alarm right?

Ans:

It has Ionizer ON/OFF check. So, it is same as EP5 condition

 


 

4.What is the ion balance +/- range the controller will detect before alarm? I only know for EP4 ESD class 0 is +/- 5v. And it can be set in software.

Ans:

It depends on environment condition. It may have cleaning alarm +/-35 to 45 V


 


 

If pins dirty, the ions cannot discharge properly and make the ion imbalance inside the chamber. So sensor will detect ion imbalance and alarm as cleaning alarm right?


 

If not, how the handler know that the pins are dirty?

 

Also, you mention it may have cleaning alarm +/-35 to 45v. So this is range of ion balance in chamber correct? ion balance in chamber if more than 45v means pins are dirty and need to clean so cleaning alarm is relationship to ion balance?


 

Actually, it is detected decay condition.

Ion balance value is just for calculate.

(I’m sorry but I also not sure that how to calculate the value.

This is replied by supplier.)
