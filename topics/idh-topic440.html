Service History

Subject: NS80-99 Operation panel IO no response

Handler Model: NS-8040

Controller: RC520

Date: 5 Dec 2007


 

Symptom

Cannot HOME or START from Front and Rear Operation Panel.

Remote I/O card red ALM LED lighted. 


 


 

Action

Swap with another Remote I/O card to confirm card faulty.

Suspect card was swapped as there was marker pen marking on the card.


 


 

Cause

Faulty Remote I/O card.


 


 

Remarks

Use multi-meter to check below when handler is power off:

1. Between D-G NO short-circuit.

(Resistance: >200 Ohm( Value will be depended on equipment))

2. Between 0-24V NO short-circuit.

(Resistance: >few kOhm (Value will be depended on equipment))


 

SPEL "Main" prg if not selected for Auto mode will also cause Operation panel no response.

In NS-8000 series, there is no PC IO card, so Operation panel operation is linked to Input Remote IO Module EIN30 (16 CH below Loader), wiring diagram SKH0811- E02 .
