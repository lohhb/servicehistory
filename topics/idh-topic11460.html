Service History

Subject: NS-8040 During ONLINE Production hang up

Handler Model: NS-8040(NS-102)

Controller: RC520

Date: 29 Dec 2009


 

Symptom

NS-102 with WFWLB183/217 setup for production run to replace NS-002, NG.

Handler issue Input Arm out of range error during Hand D trying to pickup device at loader tray(after place device at Input Shuttle 1 & 2).

HMI error: Input Arm","WAR0199","00","00","00","00","Robot controller (RC520) error occurs.

SPEL error: I-2031    The subscription of Array Variable is out of the user-defined range.

Handler hang up.

Symptom is similar to error occurred when NS-101 go ONLINE, but OFFLINE no hang up.

Re-install HMI F8.12-SIS, changed I/F to INFINEON typeA.


 


 

Action


 


 

Cause

Bug for Real Time Command error.

HMI ver 2.5 will remove the bug.


 


 

Remarks

The following errors in SPEL:

1) I-2031    The subscription of Array Variable is out of the user-defined range.

2) T-4102    The Drive Unit power is not turned on. 

3) T-4003    Communication error within the Motion Control Module.

4) Error G-7101    Unknown Error !!

are the same that occurred in 0087-NS8K and 0125-NS8K
