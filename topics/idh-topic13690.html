Subject: UTAC  NS8080-04 chamber temperature display on HMI 520 deg

Handler Model: NS-8080(NS8080-04, S/N: 181748)

Controller: RC520

Date: 23 Oct 2010


 

Symptom

Reported by Ray that chamber temperature display on HMI 520 deg.

After power cycle handler, encounter JAM1514  serial port open error.

All temp on HMI display as "---" dash.


 


 

Action

Check and found temp control unit 2 dip switch setting wrong. Set from '5' to '2'.

Reboot handler ok, all temp got display but chamber temp 520 deg.

Measure chamber temp sensor wiring resistivity shows 0.5 ohm.

Check NS8080-07 chamber temp sensor wiring shows 110 ohm.

So confirm on temp sensor faulty.

Handler temporary able to run on ambient mode only until warranty replacement.

SR:011941


 


 

Cause

Faulty temperature sensor of chamber.
