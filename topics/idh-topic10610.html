Subject: EEMS NS-7000 Input Hand Open/Close error during MCAL

Handler Model: NS-7000 (S/N: 181187)

Controller: RC520

Date: 1 Jul 2009


 

Symptom

Input Hand Open/Close error during MCAL.

Unable to complete MCAL motion, open/close hand jerk during execution of SPEL motion command. This problem was not seen at output Hand open/close.


 


 

Action

Swapped 2 PG cables  to PG Board C-873.

Check PG Board in controller for burn and setting.

Swapped stepper driver DRV20 for Input open/close.

Retension timing belt for open/cose stepper motor.

Replaced stepper motor, ok.

Swapped back from 3.5" HDD to original 2.5" HDD.

Dry run handler, ok.


 


 

Cause

Faulty stepper motor.

Stepper Motor with gear (Spec: PMM33A-MG10, R.code: R33N700002600, 7NS05-32)
