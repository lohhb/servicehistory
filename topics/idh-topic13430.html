Service History

Subject: NS-7080W Index Arm 1 unable to pickup device

Handler Model: NS7080W (S/N:181826)

Controller: RC520

Date: 13 Oct 2010


 

Symptom

Output arm mechanical alignment using 80x60 shuttle jig. Running check found index arm 1 unable to pickup devices.


 


 

Action

Rubber cups and vacuum check carried out but problem persist.

After removed layout kit found that one of the hook bending angle has been widen.

Changed another set of layout kit, problem solved.


 


 

Cause

Hook on socket layout kit bend caused vacuum leak which resulted in device pickup error.
