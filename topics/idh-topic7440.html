Service History

Subject: NS01 unable to pick up tray and floating from hotplate to shuttle

Handler Model: NS-8040(Infineon, S/N: 181492, NS-01)

Controller: RC520

Date: 20 Nov 2008


 

Symptom

Unable to pick up tray and floating from hotplate to shuttle.


 

Action

1.   Empty Tray for NS01 unable to pick up empty tray.

2.   Rotated 90 degrees temporary for running.

3.   Reported by KA that NS01 input arm y-axis showing symptoms of drift.

4.   Found NS01 empty tray arm pad using rectangle type which should NOT be the case. Only NS100, evaluation handler is rectangle type.

5.   Reported to engineer that line swapped empty tray arm pad of NS100 and NS01.

6.   Changed back pad and both handler able to pick up tray.

7.   Checked input arm pick and place. KA offset HOTPLATE matrix y-start from 20mm  change to 20.5mm.

8.   This should NOT be the case. Offset can be done on hotplate offset and input arm has no drift.

9.   Changed hotplate offset from 0 -> -0.4mm and 1 full tray running check. Found no floating on shuttle. Pick and place on input arm found good and verified by Infineon engineer to be good.
