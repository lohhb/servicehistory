Remarks


 

Distribute board SKP479(Type A & Type B) :

Board to distribute the motor power / encoder / I/O cables among the Z-axis actuator,main board, and driver module. Two LEDs are mounted on this board to indicate the current pressure of vacuum sensor and current control of Counter Air (C-Air) valve of the Z-axis actuator.


 

Control Board SKP478 (Main board) :

One main board can control actuators' motion and I/O for 8 axes in total. Driver module on the main board controls the motor of each Z-axis actuator.


 

Driver module( Driver Board SKP480  +  Resolver Board SKP481) :

One driver module can control the Z-axis actuators' motor for 2 axes.
