Subject: ASE II NS-6040(S/N: 15906) PC-686BX Win98 unable to use EPSON Recovery CD

Handler Model: NS-6040 (S/N: 15906)

Controller: SRC520

Date: 5 Apr 2011


 

Symptom

ASE II NS-6040(S/N: 15906) PC-686BX Win98 unable to use EPSON Recovery CD.


 


 

Action

5 Apr 2011

Customer informed using EPSON Recovery cd FD04S015 , SRC520CE controller hang up and also facing “A fatal exception 0E has occurred at 0028:00000015.The current application will be terminated.”(blue screen).Also when copy backup system data, projects and  setup files to floppy disk, encountered disk write error(blue screen).Hang up due to recovery cd unable to support 60GB HDD.Recovery using old image file to this 60GB HDD, due boot up encountered fatal exception error and SPEL execution error.Using EPSON Recovery cd  FD04S015 on 20GB, okDuring bootup encountered fatal exception error and SPEL execution error.Image another NS-6040 and restore to 60GB HDDDuring 60GB HDD launch MMI and SPEL, encountered SPEL execution error.Suspect PC-686BX CPU Board spoilt.


 

6 Apr 2011

Replaced to new CPU Board(from stock's RC520)Recovered data back to 60GB HDD, able to launch MMI & SPEL95 without problem.But backup in floppy corrupted, no spel pos data.Reteach SPEL position.SR no: 012166


 


 

Cause

PC-686BX CPU spolit.
