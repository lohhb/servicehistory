Service History

Subject: NS-8080 Device pickup error at loader

Handler Model: NS-8080 (S/N: 181632)

Controller: RC520

Date: 11 Feb 2010


 

Symptom

Input arm Hand C once pickup device, the device drop off.

When device been pickup, If Hand C able to pickup device successfully, the vacuum

reads -54kpa but when moved above shuttle, it reads -82kpa.


 


 

Action

Found Hand C partial torn at the center of air tubing.

Air tubing replaced and running check carried out with no abnormalities.


 


 

Cause

Input Hand C tubing broken caused device dropped.
