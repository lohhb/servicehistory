Service History

Subject: Issue during NS-6040 CPU Upgrading on NS6040 S/N: 15627

Handler Model: NS-6040(STATS ChipPac Ltd)

Controller: SRC520

Date: 17 Oct 2008


 

Symptom

Issue during NS-6040 CPU Upgrading on NS6040 S/N: 15627.

1) After CPU Board upgrade, when power up the handler, no display on the monitor.

Found that the CPU board faulty.

2) Reset and Pause buttons didn't light up.


 

Action

UPH conducted from 0sec to 6sec and backup the system before CPU upgrading.

850MHz CPU board was installed on the SRC controller.

When power up the handler, no display on the monitor found that the CPU board faulty.

Faulty CPU board replaced.


 

System BIOS (IRQ 5 & 10  Legacy ISA) was configured and Win98, MMI and SPEL were installed using CPU upgrading CD.

Upgrading of the MMI to ver C7.14-SSY and restoration of the backup files.

Reset and Pause buttons didn't light up.

Operation panel items at user I/O in SPEL were all lighted up thats abnormal.

SPEL system file was tried to restore again but problem persist.

Re-installation of software failed.

Tried to resit the PC I/O connector and check out the fuse condition.

Lastly, resit the PC I/O board and problem solved.

UPH conducted from 0sec to 6sec after CPU upgrading.

Dummy run on 14x20 with alignment carried out at  Auto 1-3 and Fix 1-3 out therefore have to re-align.
