Subject: Upgrade MMI to 2.0c12. Encountered "Jam0715 Tester interface control abnormal"

Handler Model: NS-6040 (UTAC NS60-15T)

Controller: SRC520CE

Date: 11 Dec 2007


 

Symptom

Upgrade MMI to 2.0c12. Encountered "Jam0715 Tester interface control abnormal".


 


 

Action

Check and replace switching power supply from hardware store and Ifunt100 interface unit

not OK.

Re-sit and reload ifunt_40.hex file, OK.

Load ifunt_50.hex file and verify handshake/binning, OK.


 


 

Cause

Faulty switch power supply in SRC520 controller

Faulty I/F unit IFUNT100.
