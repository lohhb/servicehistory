Subject: NS-3000 ATC MMI software patch Ver A10.12-AMD for thermal Index Head temperature display

Handler Model: NS-3000 ATC

Controller: RC520

Date: 16 Jul 2009


 

Symptom

AMD NS3000 ATC MMI A9.11-AMD have time stamp error.

        3 files with time stamp error:

        C:\WINNT\SYSTEM32\COMCTL32.OCX

        C:\WINNT\SYSTEM32\COMDLG32.OCX

        C:\WINNT\SYSTEM32\TABCTL32.OCX


 

Current ATC MMI A9.11-AMD (This version have time stamp error).

NS-3000 ATC need to solve the following:

-    MMI disable Index Heater

-    MMI Time Stamp error


 

Manually disable Head 1, Head 2, Head 5 and Head 6 temperature display on MMI by editing C:\NS3000\Config\Table\Temp.def

Change Temp.def content to the following to disable Head temperature display:

-1, 0, Head 1

-1, 0, Head 2

-1, 0, Head 5

-1, 0, Head 6


 

With new MMI Ver A10.12-AMD (SPEL Ver2.81b), it is suppose to automatically disable Heads temperature display in MMI.

ATC controller LB300i  has it own temperature display, when ATC in use the Heater in Index Head is not in used.


 


 

Action

Install revised MMI Ver A10.12-AMD from EPSON but only solved the thermal head temperature display ON/OFF for ATC enable or disable issue.


 

Communication error of RS232 for ATC, resit connector at Yamatake board.


 

19 Nov 2009

Received solution of time stamp error.

Please see the procedure as follows;


 

Note: The procedure is done before install software to Handler.

- Find NS3000\exe\Tool\DEL_NS.bat_

- Change the name to DEL_NS.bat

  Removed "_" from file name.

- Execute the file(Double click the file)

- Install new software
