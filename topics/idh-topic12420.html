Subject: NS-8080 During Initial or Continuous Start, Input Shuttle 2 Sensor error (loading side)

Handler Model: NS-8080(03-NS8000; S/N: 181634)

Controller: RC520

Date: 21 Apr 2010


 

Symptom

During Initial or Continuous Start, Input Shuttle 2 Sensor error (loading side).


 


 

Action

Shuttle 2 Sensor error (loading side):

Test-site X-pitch 80mm, Y-pitch 60mm

Package UF-I9045-OCTAL-14x14-128QFP

HMI M9.10-SSY-sp

SPEL CT 3.11ds

Wide Hotplate

PSUNT 351

1) During Initial start or Continuous start, checking on Input Shuttles detects Shuttle 2 sensor error(loading side)

2) Inline 4 fiber sensor(for Octal; 4x2) input shuttle 2, SQ-127 setting check and power tunning with device as this sensor have very low reading (1 Device Exist Sensor + 6 Device Floating Sensors)

3) Check I/O module EIN8 (64), found with paper labeling and marking on black connector(Strangely this connector is labeled Shuttle1).

4) Found one orange wire's pin connection to this black connector

5) Connect this orange wire to the correct pin location

6) Offline dry run 9 trays dummy devices in Ambient, ok

7) Offline dry run 9 trays dummy devices in Hot, ok

Service Report S/N: 011777


 


 

Cause

Wrong pin connection on black connector labeled Shuttle 1.


 


 

Remarks

Shuttle sensor of Input has 1+8=9


 

In case of Shuttle sensor error on Wide type:

Show on the LCD display W0522+W0523

W0522 is 7 sensors on the Standard handler

W0523 is 2 sensor of additional Wide type.


 

Shuttle1

NS-8080:  Floating sensor is 6 units (SQ112,113,120,121,122,123)

                  Existing sensor is 1 unit. (SQ116)

NS-8080W Floating sensor is 8 units (SQ58,59,112,113,120.121.122.123)

                  Existing sensor is 1 unit. (SQ116)

               Shuttle sensor error: W0422


 

               Option: Additional floating sensor ( This is X direction. SQ160,161)              

                           This error shows W0423.

 

Shuttle2
