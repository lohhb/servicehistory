Service History

Subject: UTAC IFX BGA8.5x8.5 high conti failure on NS8040 quad site

Handler Model: NS-8040

Controller: RC520

Date: 23 Nov 2010


 

Symptom

UTAC  IFX BGA8.5x8.5 high conti failure on NS8040 quad site.


 


 

Action

Total 5 setup running same package.

1 setup with HMI ver.2.6 and 1 setup with HMI ver.1.5  over 90% yield.

The other 3 setup with HMI ver.1.5 low yield.

Found handler with HMI ver.2.6 read 240Kpa on the EP regulator.

The handler with HMI ver.1.5 only 125Kpa.

 Awaiting Hidaka san to reply on this software difference.

Customer feedback during meeting, they already tried swapping handler, C/K, sockets but still cannot resolve.

They had also checked handler index arm alignment and coplanarity ok.

Lastly, UTAC agreed to swap the handler (HMI ver.2.6) together with the C/K, loadboard and sockets with the handler of worst yield, monitor overnight production and feedback the next day.


 


 

Cause

According to software engineer, this is software mis-calculation for HMI version 1.5. HMI version 2.0 onwards problem already fixed.
