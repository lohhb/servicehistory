Service History

Subject: Field upgrade NS-8080 to High UPH 8 Hands

Handler Model:  NS-8080 (S/N: 181594, H14-NS88)

Controller: RC520

Date: 17 Oct 2011


 

Symptom

ASE 8 IO upgrade on H14-NS88 s/n:181594 by MIKUNI.


 


 

Action

17 Oct 2011

- Precheck on handler condition and backup handler data.
 - Remove 4 hands assembly on input/output arm.
 - Remove belt holder.
 - Remove linkage and cam follower on X-axis(input/output arm)
 - Remove existing vacuum lines(input/output arm)
 - Relocate X-axis home sensor location(input/output arm)
 - Remove X-axis hard stoppers on both ends(input/output arm)
 - Installed 8 IO assembly on but input/output arm.
 - Installed new belt holder(input/output arm)
 - Installed new linkage.
 - Installed new hard stoppers(input/output arm)
 - Replaced Y-axis home sensor plates(input/output arm)
 - Split vacuum lines for 8 IO vacuum supply for both input/out arm.
 - Installed RS485 communication card (Serial I/O Board ; R35N800022500 (COM-2DL-PCI)), in RC520.


 

18 Oct 2011

- Harddisk recovery (for High UPH 8 Hands) and restore handler data.
 - Replaced fixed tray plate.
 - Using dial gauge to check fixed tray plate parallelism/perpendicular with output arm by mounting dial gauge on output arm(checking on fixed tray X and Y axis guide pins)
 - Check fixed tray tray presence sensor in SPEL.
 - Reduce PSU1 supply to 24.7Vdc.
 - Check both input and output vacuum and counter air in Z-Teach program.
 - MCAL openclose for both input and output arm. Check in SPEL input bit 56 INA_HD_OS and 60 OTA_HD_OS. Check 40mm between individual hands.
 - Relocate loader position to -60mm(Config require for 8IO)
 - Using dial gauge on input arm to check X and Y axis of loader lane.
 - Adjust loader tray lifter parallelism with ref. to tray.
 - Reteach Tray arm loader position.
- Master Calibration for both input/output arm.
 - Hand pitch check for both input/output arm.
 - Auto teaching for both input/output arm.
 - Verified tray arm pickup position.
 - Cycle run handler.
 - RS485 2Ack error found on Output arm follow by Hand H servo error.
 - Unable to recover, informed to EPSON by mikuni members and requested for Hand actuator replacement.


 

20 Oct 2011

- Troubleshoot on Output Arm Hand H servo error.
 - Error recovered but simulated error during cycle run by Pause+Reset and Start. RS485 2Ack error and Hand Servo error Hand H occur.
 - Suspect Hand H wiring problem.
 - New hand H replacement arrive.
 - Replaced Output Arm Hand H.
 - Simulation of error OK, no occurance.
 - Housekeeping of wirings on Hand H.
 - Check vacuum and counter air.
 - Master calibration and Hand pitch adjustment on output arm.
 - Cycle run handler.

21 Oct 2011

- Convert handler to Octal 4x4.5 package.
 - Fine-tune pickup and placement on handler.
 - Found out Output arm pickup Y pickup/placement offset more than 1mm due to Hand pitch adjustment not done with reference to Hand A after Hand H replacement.
 - Redo auto teaching on output arm.
 - Found loader pickup starting of tray and end of tray not level.
 - Attached dial gauge on input arm and check on loader lane co-planarity. Readjust loader lane.
 - Redo Auto teaching on input arm.
 - Fine-tune in Hot mode for pickup/placement on loader, hotplate, input shuttles, output shuttles and unloader trays.
 - Installed HMI v1.9 and release to customer.
