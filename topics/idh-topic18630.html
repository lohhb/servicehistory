Service History

Subject: Reloader 2 unable to unload empty tray to operator side

Handler Model: NX1032XS (S/N: 182347)

Controller: RC520 (S/N: 00504), DUBOX030 (S/N: 100004), PSUNT353 (P892)

Date: 11 Oct 2012  HMI 1.0 ( FD12S163), (Recovery DVD Ver 1.0,  FD12S164)


 

Symptom

NX1032XS Reloader 2 unable to unload empty tray to operator's side in Auto mode.Using HMI [I/O Monitor] to trigger also unable to move empty tray out at the Reloader 2.

Reloader 2 only can move empty tray into the handler.


 


 

Action

- Swapped ribbon cable for Reloader 2.

- Replaced 16 Bit Output I/O module "Loader" in PSUNT


 

EPSON send HMI Ver 1.03 has been modified for Reloader 2 movement.


 

Install HMI 1.03 and tested, Reloader 2 able to function correctly.


 


 

Cause

HMI 1.0 software bug caused Reloader 2 unable to function correctly.
