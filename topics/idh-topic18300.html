Service History

Subject: NS8040(181206) encountered hardware conflict on pci slot 2 during boot up

Handler Model: Evaluation System NS8040 Handler (S/N: 181206)

Controller: RC520

Date: 16 Mar 2012


 

Symptom

NS8040(181206) encountered hardware conflict on pci slot 2 during boot up.


 


 

Action

20 Mar 2012

1) Installed IFUNT100 driver but conflict persist.

2) Installed HEX file to IF UNIT but software unable to detect GPIB card.

3) Replaced GPIB card.

4) In Bios, no conflict message on PCI slot 2.

5) Installed IFUNT100 driver, able to detect new hardware and no conflict.

6) Installed HEX file, able to detect GPIB card and able to load HEX file to IF UNIT.

7) Able to select GPIB, RS232C and DIO communications with no error.


 


 

PT sensor for index arm head 5/6 is N.G.
 Temperature feedback is drifting. Sometimes -14, 4 or even 520 degrees.

Part code should be 8NS38a-12, 8NS38a-04, 8NS38a-10 according to NS8040 part list manual Rev.1 page 143


 


 


 

Cause

Spoilt IFUNT100B card (R35N700012300, SKP383) in RC520.
