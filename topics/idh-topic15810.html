Controller: RC520

Date: 7 Apr 2011


 

Symptom

UTAC - NS8080-01 output arm hand G hand up sensor error issue.


 


 

Action

- Swapped back hand home sensor from hand E to hand G.

- Replaced new z-axis module hand G.

- Replaced new driver/encoder board for EG/FH.

- Check new hand G hand height by Auto teaching.exe. Carried out hand height calibration.

- Check new hand G XY pick up position at output shuttle ok.

- Run dry cycle mode. Release handler.

Warranty S0474 closed.
