Service History

Subject: NS-8080 encountered the following error:

1.  “The uniwire 1 transmission line is disconnected”

2.  Index door not close

3. Chamber fan error

Handler Model: NS-8080 (S/N: 181648)

Controller: RC520

Date: 15 Mar 2010


 

Symptom

NS-8080 (S/N: 181648) is down. Handler giving the following main error :

1.  “The uniwire 1 transmission line is disconnected”

2.  Index door not close

3. Chamber fan error

Index door & Chamber fan signal belongs to same Input I/O Module EIN30.


 


 

Action

-Reset Remote I/O board, then restart handler and try to homing found the error still persist.

-Check I/O modules in power RDY and LINK  power is power on.

-Open top cover of index area, resit all I/O modules power connector error still persist.

-Check the input I/O address -ok

-I/O check found the index door signal wasn't feed back.

-Swapped the FANDRV output module from other handler the error still persist.

-Swapped the INDEXIN1 input module from other handler problem solved.


 


 

Cause

Input I/O module EIN30(232) causing the problems.

Remote I/O Input Unit 8CH ------R33NS700001500


 


 

Remarks

Use multi-meter to check below when handler is power off:

1. Between D-G  NO short-circuit.

(Resistance: >200 Ohm( Value will be depended on equipment))

2. Between 0-24V  NO short-circuit.

(Resistance: >few kOhm (Value will be depended on equipment))


 

Email by Hidaka, 25 May 2011:

To change EIN30 label of 232  Input Module 8 CH (located at Index area)

EIN30    -> OP IN

EOUT30 ->OP OUT


 

EIN30(23) of 8 CH above Index connects the following:

1) IN280, SKIP ENABLE

2) IN281, not use for NS-8040 (NS-8080/8160 Chamber Fan Driver Alarm)

3) IN282, ionizer for chamber(test area)

4) IN283, Ionizer for Loader & Unloader (Ionizer Alarm 2A & 2B)

5) IN284, Index Cover Unit

6) IN285, not use in NS-8040 (NS-8080/8160 Test Hand spring compliance sensor)

7) IN286, Index Cooling Fan for Z motor

8) IN287, Index Cooling Fan for U motor


 

Refer Drawing Name. Skip enable SW / Sheet No. SKH0811-G16-3 / Drawing No. Remote I/O 16
