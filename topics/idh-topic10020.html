Service History

Subject: NS-8040 HMI unable to load completely

Handler Model: NS-8040 (S/N: 181452)

Controller: RC520

Date: 9 May 2009


 

Symptom

Infineon informed NS-8040(S/N: 181452) unable to load HMI completely.


 

Action

Use NS-8040 Recovery CD Ver 1.56 to recover HMI F8.12-SIS and SPEL CT 3.11ds to HDD in demo NS-8040 in getech office

Use  BackupSan restore backup files for NS-8040(S/N: 181452).

Bring recovered HDD to Infineon.

Attempt to re-install HMI but HDD cannot boot up.

Swapped to new recovered HDD able to boot up.

Run dummy devices for (V93K)_WFWLB217_Q_AMB, ok


 


 

Cause


 


 

Remarks

Maybe setup file corrupted.

In HMI, can try to select different setup file and delete the previously load setup file.
