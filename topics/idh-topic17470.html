Service History

Subject: NS-7000 Output Arm Hand Open/Close issue

Handler Model: NS-7000 (S/N: 18732, NS70-01)

Controller: RC520

Date: 11 Apr 2012


 

Symptom

Silabs – NS70-001(18732) output arm open/close issue.


 


 

Action

Replaced back new type stepper motor driver CSD5807N2-P and tried recommended

 “Run 5, Stop 3.5 setting but open/close movement still unable to function properly.

 Tried other settings same.


 

3 May 2012

1.Check output shuttle 80x60 pitch with shuttle alignment jig. Hand C alignment out.
2.When variable hand closed, hand B and D already touched but hand A and C still got gap.
3.Sometimes 100% speed can hear knocking sound when variable hand close.
4.Adjust output arm HOFS open/close from -280 to -500 to open up variable pitch.
5.Fixed back new motor driver CSD5807N2-P, set 100% speed, output shuttle variable 0.5mm and cycle run handler, ok.
6.Release handler for monitoring.
*Also found hand A and B lower than C and D. SR:013193.


 

6 May 2012

 encountered 2 times output arm unable to open/close to position.


 07 May 2012

swapped motor driver from output to input.

Encountered hand open/close error and open/close pitch unable to open/close to position at loader and input shuttle.
Swapped stepper motor driver CSD5807N2-P with Getech used driver PMD07C.
Customer accepted used driver as purchased item.
SR:013196


 


 

Cause

Faulty Stepper motor driver CSD5807N2-P, so use Getech used driver PMD07C.  .
