Service History

Subject: NS-7000 Recovery CD questions

Handler Model: NS-7000

Controller: RC520

Date: 11 Sep 2009


 

Symptom

Questions on NS-7000 Recovery CD.


 


 

Action

Recovery CD Ver1.10 and Ver1.20 are based on same contents but recovery software is different.

Ver1.10 used Easy restore for recovery software.

Ver1.20 used Ghost for recovery software for CPU Board SPI-6941.


 

1) Recovery CD Ver. 1.00  FD04S50, can this is applied on CPU Board PC-686BX ?

Recovery CD Ver 1.00 FD04S50 CANNOT be used on CPU Board PC-686BX.

This  is for CPU Board "SPI-6941-EPP" with PG Board C-864v1 x03pcs on ISA slots is old type PG Board.


 

2) Will the PG Boards(C-863, C-864v1 and C-873) of different model such as C-864v1 affect the recovery CD version to be used?

C-863 per board support maximum 8 Axes

C-864v1 per PG Board support maximum 8 Axes

C-873( same as NS8000 PG) per PG Board support maximum 16 Axes

There are 2 types of CPU board for NS-7000 series handlers.

Old type CPU board "PC-686BX(PC-EPP850)" is not support ( Can not use )

recovery CD.

The latest CPU board is "SPI-6941-EPP.


 

Ver 1.00 FD04S50 is for latest CPU board of "SPI-6941-EPP" and old PG of

"C-864V1".


 

Ver 1.10 FD05S050 is for latest CPU board of "SPI-6941-EPP" and latest PG board of "C873" as one board type. 


 

Ver 1.20 FD07S034 is for latest CPU board of "SPI-6941-EPP" and latest

PG board of "C873" as one board type for NS7000 and NS7080W.


 


 

Note:

Information from EPSON Robot:

Please get RC520 recovery CD-ROM of " Windows 2000 Rev. SP4 recovery"

of N0. "R5-4000e-e date 2003.12.16".

This recovery CD only recovers up till Windows 2000 installation.

After Windows 2000 recovery, Windows 2000 boot up, continue Windows 2000 reserve resources for MIB 1 & 2, PG 1~3 and etc by following NS7000 Maintenance manual Rev 1 or 2.
