Error 5041     Torque error in the Low-power state.


 

The Manipulator's error (irregular motion) is detected in the condition where the upper limit of the torque is being controlled in Low-power state.

Check the Manipulator

(1) if the arm moves around smoothly when it is moved around by hands,

(2) if the arm's movement is tight and secure when it is moved around by hands,

(3) if there is any hitches with the arm when it is moved around by hands,

(4) if the belt is not slackened when checked disassembled and

(5) if the arm does not hit the brake when it is moved around while the electro-magnetic brake is turned off.

Check if the Manipulator's arm is not colliding with the peripheral units around.

Check on the power cable connection.

Check on the power cables inside the Manipulator for any loose pins, cable disconnection or short-circuit. Replace the motor driver.

Replace the Drive Unit.

Replace the motor in case of motor/encoder failure.


 

Check:

1) CT Data

2) Motor Driver Module ( servo driver )

3) Motor (smooth when motor shaft turn by hand, any break unit)

4) Signal Cable connection

5) Belt tension

6) Cable short/open/loose on manipulator

7) Any dummy inertia (NS-6040 and UBS Input/Output Arm)

8) SSR ( Solid State Relay )

9) Greasing
