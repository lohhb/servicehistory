Service History

Subject: Tray arm robot controller(SRC) error occurs during production

Handler Model: NS-8080 (S/N: 181416)

Controller: RC520

Date: 8 Sep 2009


 

Symptom

Customer found that tray arm robot controller(SRC) error occurs during production


 


 

Action

-Checked the servo motor and encoder header is proper plug on controller.

-Do axis auto calibration, error still persist

-Replaced the motor driver (100W), error solved.

-dummy run -ok.


 


 


 

Cause

Faulty 100W Motor Driver(Driver Module) caused tray arm robot controller(SRC) error.
