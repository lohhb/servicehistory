Error 53   The specified file cannot be opened.

Causes

The .OBJ or .PNT file name specified in an XQT statement does not exist in the robot controller.

The file name specified in a CHAIN statement does not exist in the robot controller.

An #include statement refers to a file that does not exist.

Remedies

In EPSON RC+, XQT and CHAIN must use program group names. Each program group has an OBJ file and a PNT file. Check your programs to make sure you are using the correct syntax for these commands.

Check your #include statements.
