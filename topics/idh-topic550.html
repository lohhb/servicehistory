Subject: MMI and SPEL upgrade for NS70-30

Handler Model: NS-7000 #275 (S/N: 18996) (UTAC)

Controller: RC520 (S/N: 01-20285)

Date: 26 Dec 2007


 

Symptom

Encountered run-time error during MMI upgrade installation, MMI unable to launch.


 

Action

Recovered system using Recovery CD 1.10(FD05S050) (SPEL CT 2.81b, MMI is 1.1c10).

Upgrade SPEL to 2.81b1h

Upgrade MMI to 1.1c13.

Restored system, pose and setup files.

Dry cycle handler, O.K
