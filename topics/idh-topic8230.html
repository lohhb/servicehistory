Subject: NS-7160W Chamber Fan not moving when Chamber Heater ON

Handler Model: NS-7160W

Controller: RC520  MMI Ver 1.0d

Date: 19 Feb 2009


 

Symptom

NS7160w for UTAC customer report:

1) From MMI Maintenance->Equipment controller, can manually turn on fan.


 

2) If turn on heater, chamber fan normally will start moving but now, turn on heater, chamber fan not move.

Upon switch to high temp, chamber fan cuts off immediately and stop moving.


 

Sometime have error message upon power up:

Error code 42, Subsiadiary 15

Unit name : Temp controller.


 

Suspect possible yamatake problem or some hardware failure.

Chamber will always on when switch from Ambient->Hot.

Even if temperature has reached, chamber fan is still moving.


 

Please check with Japan what other check area and what files they need? I think hardware wiring should be ok because can turn on chamber fan in equipment controller.


 


 


 

Action

Advised to check SQ195 connection.
