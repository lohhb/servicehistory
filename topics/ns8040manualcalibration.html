*MCAL for selrb 7 or selrb 8, use a ruler or shuttle jig check for 80mm Hand pitch, if NG use the [Robot Manipulator] adjust HOFS (How about MCAL then check 80mm pitch, use Jog&Teach to Open/Close Hand then measure for 80mm then calpls 0 and calib 1)

·     To Initialise Absolute Motor, in SPEL CT, click [Setup]->[Robot Manipulator]->[Reset Enc] tab. Tick the Joint to initialize and click [Reset] button.


 


 

·     The Limit Pulse value is only for reference from UTAC NS80-18 (181363) and the limit pulse value used on NS80-89 as calibration pulse.

·     Push & hold affected Axis to the mechanical “negative” limit, then perform calibration using the above tabulated Limit Pulse.

·     After calibration, teach Index Arm Z, U and Y. Confirm Shuttles positions.

·     Alignment check in HMI.


 

Remarks

Install SPEL CT 3.11ds will solve Encoder Backup Alarm error,

or use Recovery CD FD07S109  ver1.56 (12/11/2007) will also solve Encoder Backup Alarm error. Therefore, AutoTeach of motors can be used.

This is similar to 0004-NS8K and 0007-NS8K.
