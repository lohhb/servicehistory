Service History

Subject: NS8080 Temperature Priority Mode different from NS-7000 series

Handler Model: NS-8080/NS-7000 series

Controller: RC520

Date: 22 Jul 2009


 

Enquiry

Xilinx Alan Teoh asked why new NS-8080 handler software temperature priority mode is not same as NS7080W handler software:


 


 

Reply

For NS8080, in temperature offset window, in priority mode, when we choose

1) <Speed Priority Mode> means that test arm 1 will wait with devices in nests when test arm 2 is testing. Input shuttle will be in chamber. – Temperature has been found to be more stable. (Using chamber temperature to control K1)


 

2) <Temperature Priority Mode> means that devices will be in input shuttle 1 pockets in the chamber, while test arm 2 is testing. – This mode has been found to be harder to control temperature as device usually fail K1 (Too hot).

 

For NS7KW software when set to temperature priority mode, input shuttle is outside chamber, ensuring better temperature control as it is not impacted by chamber temperature.

We need the NS8080 handler to follow the temperature priority mode of the NS7080W handler software.


 

24 Jul 2009 email from Hidaka-san:

we evaluated temperature priority mode NS7 style and NS8 style.

(Detail of evaluation data, please see attached file "NS8000 temperature priority mode.pdf".)

We found both of styles have good point and weak point.

However, we decided to use NS8 style temperature priority mode in NS8.


 


 

Remarks

1) For NS-8080 standard, Temperature priority mode (Input Shuttle standby Inside chamber).

2) For NS-7000 standard, Temperature priority mode (Input Shuttle standby Outside chamber).


 

Weak point of NS8K  Temperature priority mode:

- Necessary to change Shuttle temperature offset for long and short test time.

  The reason is because Input shuttle go to chamber in standby while test arm in contact to socket. It affect chamber temperature, so it is difficult to adjust temperature by waiting time on the shuttle.


 

Weak point of NS7K Temperature priority mode:

- Cold material(Output Shuttle) go into chamber during test process, it does not stablise chamber temperature.
