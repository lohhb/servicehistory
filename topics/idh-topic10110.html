Subject: NS-7000 Output arm hand D, vacuum cannot turn on and hand up sensor error

Handler Model: NS-7000 (S/N: 18724)

Controller: RC520

Date: 29 Apr 2009


 

Symptom

Vacuum error at output arm hand D.

Hand up sensor error at output arm hand D.


 

Action

Vacuum error at output arm hand D

- found out that the there is a missing washer from hand D, after inserting the washer, no more vacuum error upon hand height calibration.


 

Hand up sensor error at out put arm hand D

- home sensor was replaced due to crack.

- suspect that cables is causing the hand up sensor error, had already did some tidy up of cable and is still  under monitoring.

No. : 7NS05-34

Code : R33N700002800

Description : Beam sensor ( R ) for actuator

Specification : PM-R24


 

Note:

new HDD taken from Vincent to clear the bad sector issue.  System recovered to original using ghost CD from Silab and back-up san.

This handler cannot use normal recovery CD as there is some hardware conflict. All earlier NS7000 do not come with recovery CD. CPU Board is PC-686BX.


 

5 May 2009 Output arm hand D hand up sensor error

Hand was replaced and manual installation of Windows 2000 and drivers.

80x60 pitch checked for output arm. Additional sensors removed from input shuttles. Due to hot plates main body removed to aid removal of additional sensors, thus, teaching carried out.


 

7 May 2009 Input arm open/close unable to open

Suspect timing belt loose, thus, tightened and to monitor during buy-off
