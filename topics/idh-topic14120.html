Service History

Subject: NS-8080 8 Hands Hand up sensor does not turn on error

Handler Model: NS-8080 (NS8080-01) 8 Hand High UPH

Controller: RC520

Date: 21~23 Dec 2010


 

Symptom

Hand up sensor does not turn on error.

Even with all hands up, error occurs.


 


 

Action

Found SPEL Input Bit 52 (for 8 Hands all at home position) for Output Hands intermittent ON/OFF.

Error occurs when Input Bit 52 OFF but it will ON by itself, or by touching output hand assembly, by opening door etc..

By pushing main axis of x/y, IN52 is found flickering.

Removed covers to check inner wiring/cable condition but suddenly no 24V supply from control board.

Handler 24V should be ok as the other areas have 24V supply.

To swap control board and continue verification of IN52 intermittent.


 

Removed output arm control board, found FUSE open.

Replaced control board.

Check wire, connection for driver, encoder and distribution board.

Power ON moving with hand, no flickering of IN52.

Dry cycle dry no tray.

 No hand up sensor error.

Released for overnight dry cycle running to confirm for intermittent hand up sensor error.
