Service History

Subject: Unable to completed Input Hand D Hand Height Calibration

Handler Model: NS-7000

Controller: RC520

Date: 7 Jun 2011


 

Symptom

During Input Hand Height Calibration, unable to complete for Hand D.
 


 

Action

Input Hand D vacuum activated while moving down and go up, able to see vacuum reading reached -53kpaFound vacuum switch unit sensitivity for SET1 was set to -99 kpa.Change SET1 to -50 kpa.

Able to complete  Input Hand Height Calibration.


 


 

Cause

Wrong setting of vacuum detection threshold to -99 kpa, correct setting is -50 kpa.


 


 

Remarks

Other check points for Hand up sensor error during Hand Height Calibration:

- check filter (of actuator) is clean or place correctly

- air tubing any torn/broken/bend

- vacuum sensitivity is set correctly for vacuum switch unit(eg. set to -50kpa and not 50kpa)

- vacuum suction tip/cup is flat on the rib surface

- cone (guide pad) is secured at correct height

- gasket missing on the rubber cup(suction tip) near the screw thread

- check cable for stepper motor

- replace PG board

PG# 1 for Input Hand A, B, C & D

PG# 2 for Output Hand A, B, C & D

PG# 3 for Input & Output Open/Close


 

For C-873 there is only only PG Board in RC520.

For C-863 there are 3 PG Boards in RC520.

For C-864v1 there are 3 PG Boards in RC520.

C-863 & C-864v1 are NOT Compatible.
