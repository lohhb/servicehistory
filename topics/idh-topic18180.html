Subject: NS-8040 unable to use Recovery CD CD FD08S120 Ver 2.0 on 2.5” SATA HDD

Handler Model: NS-8040 (S/N: 181670, NS-003)

Controller: RC520 (S/N: 03-02120)

Date: 18 Nov 2008


 

Symptom

Error when using NS-8000 Recovery CDFD08S120 Ver 2.0 + CPU Board SPI-8550-LA + 2.5” 160GB HDD SATA in 3.5” cradle on Infineon NS-8040(NS-003, S/N: 181670)


 


 

Action

Use Recovery CD FD10S083 Ver 2.1 (05/28/2010) on CPU Board SPI-8550-LA(15 pin VGA port) with 2.5” SATA 160GB HDD in 3.5” cradle, ok.

Install SPEL CT 3.11ds and HMI F10.14-SIS.

Restore SPEL System & Pos and HMI setup files.

Dry run handler, ok.


 

Do Recovery on another 2.5” SATA 160GB HDD in 3.5” cradle, ok.

Service report S/N: 012184


 


 

Cause


 

Remarks

This is a follow up on 0412-NS8K.
