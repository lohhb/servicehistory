Service History

Subject: Handler hang

Handler Model: NS-6040(CARSEM)

Controller: SRC520

Date: 9~12Sept 2008


 

Symptom

Seiko 03 Input/output pickup head moves up (HOMING) but main axis and other robots did not move.

Seiko 04 Intermittent hang up, Error message “Unable to write to Drive C”.


 

Action

Seiko 03

1.   Input/output pickup head moves up (HOMING) but main axis and other robots did not move.

2.   Possibly pickup hand is unable to “HOME” so main axis and other robots did not move.

3.   To check for PG Board condition as all 4 hand unable to complete “HOME”

4.   Swapped Seiko 04 HDD to separate software or hardware issue.

5.   With Seiko 04 HDD, able to see other axis HOME.

6.   Almost confirmed  SPEL system corruption.

7.   To re-install SPEL.

8.   Restore system files.

9.   Un-install, Re-install, restore backup, all cannot.

10. To try recovery system.

11. HOME with no restore.

12. Motor on all robots with no restore.

13. Restore, and HOME to check if it's backup system file corruption.

14. In case of system files corruption, to calibrate and re-teach all points.

15.      CT gain and offset to be keyed in manually.


 


 

Seiko 04

1.   Intermittent hang up.

2.   Error message “Unable to write to Drive C”

3.   Suspect HDD bad sector/cluster.

4.   SPEL requires temp HDD space to write data during real time execution.

5.   Previous issue is also hang up. Recovery performed.

6.   Recommend to change HDD.

7.   Remove CPU Board and perform maintenance on CU.

8.   Cleaned pins, connectors.

9.   Re-sit RAMS and CPU Board.

10. After 2 days, so far no hang up.

11. Unable to clone HDD due to time constraint

12. Possibly dust collection at pins caused intermittent connectivity.

13. To be monitored.
