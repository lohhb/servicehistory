Service History

Subject: NS7000 Input & Output Hand rotator upgrade

Handler Model: NS-7000

Controller: RC520

Date: 28~29 Nov 2012


 

Symptom

SCS NS7000 Input & Output Hand rotator upgrade.


 


 

Action

28 Nov 2012

1) Hardware installed, Config.exe generated, CONST.INC settings changed.
2) During hand height calibration, both input/output arm unable to MCAL.
3) Selrb 2 and 3, unable to MCAL.
4) Selrb 7 and 8, able to MCAL.
5) Hand up position, vibrating.
6) Hand down position, always moving up and down.
7) SPEL INPUT able to see Input Hand D sensor unable to turn on.
8) Wiring and connectors checked. Input arm Hand D sensor pin disconnected.
9) Both arms still unable to execute MCAL.
10) Spel Manipulator for Mnp2 and Mnp3, shows 6 axis enabled.
11) With Rotator, should be only 4 axis enabled.
12) Update manipulator settings, able to MCAL.
13) Spel Manipulator for Mnp2 and Mnp3 now shows 4 axis enabled.
14) Fixed/Variable and Rotator arm uses PG board differently. Update manipulator settings is required.
15) Continue height calibration and device running check.

 

29 Nov 2012

- Removal of open/close arm
- Fitment of rotation arm
- Check individual hands for vacuum, counter and sensors.
- Change setting and parameters in Config.exe and SPEL
- Carried out Hand Height calibration and pitch alignment
- Carried out alignment check using QFN 5x5 dummy devices.
- System Backup
