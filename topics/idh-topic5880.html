Service History

Subject: Unable to pick up device for Index arm 1 and Index arm 2

Handler Model: NS-8040(S/N: 181452, Infineon)

Controller: RC520

Date: 4 Apr 2008


 

Symptom

Unable to pick up device Index arm 1 and Index arm 2 after mounting orientation set screw on Index Arm 1 & Arm 2.


 


 

Action

Change kit (C/K) for LQFP176 24x24 (site: 2x2) delivered.

The Contactors were not orientation fool proof, o/p shuttles without guide pins.

1) Create setup file.

2) Adjust Shuttles' sensors position for 2x2 sites, X- pitch 64mm, Y-pitch 63.5mm.

3) AutoContact Height. Device unable to pick up Index arm 1 and Index arm 2.

4) Remove Contactors, check handler vacuum strong, ok

Swap problem contactor head with another head. Same problem

5) Check orientation set screw, found not flush with contactor head after secure lead to vacuum leak. Found both head same problem. After secure, OK.

6) Meeting KG, GETECH, Infineon on change kit design issue alignment check.


 


 

Cause

Orientation set screw for Index Arm 1 & Arm 2 found not flush with contactor head after securing, lead to vacuum leak. Found both head same problem. After secure, OK.
