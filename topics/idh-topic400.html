Service History

Subject: NS-7080W System Hang(Loan handler from Xilinx)

Handler Model: NS-7080W

Controller: RC520

Date: 1 & 2 Nov 2007


 

Symptom

SCS NS-7080W system hang.

Recovery by customer using Recovery CD but Jetheater unable to activate.


 

Action

Re-install MMI.

Customer performed recovery of HDD and setup file import corruption with Jet Heater unable to turn on.

Do not suspect hardware failure as HDD crash is unlikely to damage Jet Heater.

Wiring and Jet Heater verified by customer as “satisfactory”.

SCS did not version up -> MMI v.G6.10-SCX(probably) leading to unable to import setup file and turn on Jet Heater.

Setup file from MMI v.G6.10-SCX may not be able to use in MMI version of recovery.

Not MMiv.G6.10-SCX means MMI is not Jet Heater capable on software side.

“AUTO-RESTART” of RC520 and handler HANG led to HDD illegal shutdown.

Problem started since last week. Subsequently HDD crash.

Recovery was done yesterday but same problem persist and HDD crash again.

 SCS spare HDD was used to verify HDD is not issue and found “AUTO RESTART” problem. PSU swapped, same problem and CPU Board was swapped and so far “satis”.

To monitor for problem by dry cycle overnight. If problem recurs, GETECH engineer to support 03/Nov, Saturday.

CPU Board faulty.


 


 

Cause

CPU Board was faulty.


 

Remarks

Air flow is 100L/min for jetheater.
