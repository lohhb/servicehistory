Subject: Mcorg, T-5041 torque error in low power state

Handler Model: NS-6040(UTAC)

Controller: SRC520

Date: 29 Aug 2008


 

Symptom

Mcorg, 5041 torque error in low power state.


 


 

Action

After “Home” button pressed, lndex arm moved up and drop down.

After motor on, sfree, using hand to move index arm x axis, index 1 drop down.

(be careful for NS-6000, if press Break Release switch after SFREE command, the Index arms can just drop down.

Therefore, hold the Index Arm when pressing Break Release switch)

Brake switch ok.

Mcal same problem.

Mcorg, torque error in low power state.

Dry run no problem.

Suspect possible motor brake system faulty, servo driver, signal relay board, liner guide.

Hot Plate heater on, no problem possibly because able to overcome counter torque.

Advised customer to run production but not 100%. Monday to down for checking.

Encountered tester time up error upon start lot.
