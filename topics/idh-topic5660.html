Service History

Subject: NS-8080 Hand up sensors error

Handler Model: NS-8080(STATS, evaluation)

Controller: RC520

Date: 22 Jan 2008


 

Symptom

Setup QFP 14x14 devices and contact height calibration.

Input/output arms sensor up error occurred.

Hand up sensors did not turn off at all hands.


 


 

Action

Auto Hand Height calibration carried out but fail – after Hand A finished height calibration, it does not continue calibrate rest of the Hands.

( C:\NS8000\Tool\AutoTeach.exe   then select [In/Out Arm 3 Point Teach] tab )

MMI reinstalled but problem still persist.

Hand A vacuum still ON after Hand Height Calibration.

Found that the metal cones not fully push up causing some vacuum leak.


 


 

Cause

Metal cones not fully push up causing some vacuum leak.


 


 

Remarks

Ensure vacuum sensitivity threshold is set to -30kpa to -50kpa.
