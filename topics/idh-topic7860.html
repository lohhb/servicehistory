Service History

Subject: MMI counter reset to 0 upon "TRAY END"

Handler Model: NS-7000(Amkor)

Controller: RC520

Date: 7 Nov 2008


 

Symptom

MMI counter reset to 0 upon "TRAY END".


 


 

Action

1.   Counter reset upon “TRAY END”.

2.   Software already in MMI 2.0k.

3.   Found problem to be counter reste during start of new LOT and not as reported “TRAY END”.

4.   Found start mode settings to clear counters in MMI.

5.   Removed start mode settings and monitor.

6.   ATSG will feedback if problem comes back.


 


 

Cause

Found Start Mode settings to clear counters in MMI was selected.
