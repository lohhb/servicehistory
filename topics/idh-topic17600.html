Service History

Subject: Phone support for Xilinx NS-7080W Index encountered error 5050 Overspeed in the Torque Control Mode for FG676 package when using Direct & High Contact Force

Handler Model: NS-7080W (S/N: 121298, NS7KW-07)

Controller: RC520  MMI M8.10-SXC

Date: 28 Mar 2013


 

Symptom

Phone support for Xilinx NS-7080W(NS7KW-07) Index Arm 1 & 2 encountered error 5050 Overspeed in the Torque Control Mode for FG676 package when using Direct & High Contact Force.

FG676_27x27_OCTAL_XAP package, 8 site (4x2)

X pitch 127mm, Y pitch 63.5mm


 


 

Action

Customer informed actions taken are as follows:

- Alignment check on both Index Arms.

-Found Index Arm 2 alignment out to test site and shuttle (X-Axis).

- Fine tune alignment and verified.


 

However, error appeared again during production.

both Arm 1 and 2 but still same problem using Direct Contact Mode or Direct & High contact force mode


 

Advise customer,

- check Index Arm hit to any parts/abnormal noise & motion

- check for any loose bolt at Index Arms

- guide pins for output shuttles and socket guide base jig are secured

- size & tolerance of guide pins for output shuttles and socket guide base jig

- grease Index Arm linear guides & ballscrews, High Contact Force linear slide

- check tubing & valve units for High Contact Force

- check P50 & P51 Y positions for rubbing

- Use [Retry]+[Start] in AutoContact Height

-  check for abnormal noise during Auto Contact Height


 

5 Apr 2013 onsite

Customer confirmed FF1152 & another package no problem.

Set to Direct Contact Force will also have this strange abnormal jump motion of Index Arm after contact to socket guide jig.


 

- check P50 & P51 Y positions for rubbing

- Index Arm linear guide and ballscrews check smoothness of motion & for rubbing

- check P50 & 51 Z & U height using Quad SLK, around -97mm, ok

- AutoContact Height using [RETRY]+[START], ok but Index Arm 2 pickup & release height to shuttle are 41.50mm and 43.00mm respectively as compared to Index Arm 1 having 33.50mm & 35mm

-Dry run in Direct & HCF, Index Arm 1 & 2 during contact to socket guide base have abnormal-jump(spring up effect) motion before HCF cylinder come down(Dummy Contact is OFF).

-With device running also same.

-Check with NS7KW-08 that is also running FG676_27x27_OCTAL_XAP but Index Arm 1 & 2 contact with socket guide base(but index arm does not have jump up motion) then followed by HCF cylinder coming down.

-Copied setup file FG676_27x27_OCTAL_XAP from NS7KW-08 and import to NS7KW-07

-Dry run for 1 hour, no abnormal motion or noise

-Running check with 3 tray of devices, ok

- Setupfile corrupted in this case

- SR: 012521


 


 

Cause

Setup file for FG676_27x27_OCTAL_XAP corrupted.


 

Remarks

For  FG676_27x27_OCTAL_XAP, the tray pocket for vacuum suction is not at centre of tray, so need to change "Pickup Up Position" in [Tray Form], so that empty tray can be pickup.

Sometimes, for some empty trays, the tray pocket for vacuum suction is not at centre of tray, so need to change "Pickup Up Position" (from usual 67.90mm) in [Tray Form], so that empty tray can be pickup.

So it may not be tray pad worn out or Empty tray arm position is out.
