Service History

Subject: NS-8080 High UPH  output hand B home sensor doesn’t turn off/on error

Handler Model: NS-8080 (S/N: 181665,NS8080-01)

Controller: RC520 (S/N: 03-02140)

Date: 30 Jan 2012


 

Symptom

UTAC NS-8080 High UPH  output hand B home sensor doesn’t turn off/on error.


 


 

Action

- Check input and output arm vacuum threshold –35, –40, –70 ok.

- Found output shuttle 1 pickup at –2.10, output shuttle 2 at –1.60. Maybe too low.

- Dry cycle handler for an hour ok. After pause and start again, encountered 2Ack error and output hand B unable to home.

- Checked again SPEL input bit #52 led light did not turn off when pulled down output hand B. Other hands ok.

- Replaced output hand B home sensor.

- Performed hand height calibration for output arm ok.

- Checked SPEL input bit #52 led light able to turn off when pulled down hand B.

- Dry cycle run handler at 100% ok. Did not receive dummy units for cycling.


 


 


 

Cause

Faulty Home sensor of Output Hand B.
