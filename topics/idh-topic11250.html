Service History

Subject: NS-7000 Jam 0303 Device drop error Arm 1 and Jam 0304 Device drop error arm 2

Handler Model: NS-7000 (S/N: 18721)

Controller: RC520

Date: 2 Dec 2009


 

Symptom

Loan NS-7000 (S/N: 18721) Jam0303 Device drop error arm 1 and Jam0304 Device drop error arm 2. When error occurs, devices are still on the contactor.


 


 

Action

Checked index counter air volume, vacuum, and vacuum unit sensitivity ok.

Also tried swapping IO module for Index input bit and error still persist.

Even when disable arm 1, only run index arm 2, both arm 1 and 2 device drop error will

happened.

Tried running only each of the 4 sites with the other 3 sites disable and still faced the same issue, arm 1 and arm 2 device drop error.


 

We are suspecting software issues right now as hardware seems ok. We will try to get a new HDD for recovery back to older version of MMI and SPELCT tomorrow.


 

3 Dec 2009

- Found index arm 1 and arm 2 tubing fitting at chamber top loose. Tighten and re-adjusted index vacuum unit sensitivity.

- Reset Remote IO card on RC520 controller.

- Remove SPELCT input and output remote IO addresses and add again.

- Swapped input arm INARM1 input IO module.

- Swapped new 20GB hardisk and recovery to MMi 1.1c10/SPELCT 2.81b. Restored system, pose data and re-created new setip file for Quad site BGA 11x11.

- Change setup to Dual site 1x2 TQFP 7x7.


 

After all the above, device drop error on arm 1 and 2 still occurred.


 

4 Dec 2009

- Index arm 1 and 2 device drop issue resolved after replacing remote IO card from UTAC consignment spare.


 


 

Cause

Faulty Remote I/O card.
