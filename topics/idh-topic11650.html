Service History

Subject: NS-7000 Rear operation panel no response

Handler Model: NS70-30 (S/N: 18996)

Controller: RC520

Date: 8~11 Mar 2010


 

Symptom

NS-700 during Class 0 Installation, cycle run handler and found out rear control panel no responds.


 


 

Action

Checked DIP switch settings on Contec PC i/o card in RC520. Still same condition.

Swapped Contec PC I/O card from NS70-29, condition worst as both front/rear control panel no responds.

Replaced new Contec PC i/o card into RC520. Condition same.

Removed RS485 communication card and at the same time re-sit connectors on connection board in PSUNT.

Power on, both front/ rear control panel working.

Installed RS485 communication card into RC520 again, front/rear control panel working. Concluded connectors on rear panel connection board giving the problem.


 


 

Cause

Connectors on rear operation panel connection board giving the problem.
