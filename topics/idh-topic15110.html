Service History

Subject: UTAC – NS80-40 ATC single site support HMI ver.X10.10-SUS upgrade

Handler Model: NS-8040 ATC (S/N: 181880)

Controller: RC520

Date: 15~16 Feb 2011


 

Symptom

UTAC – NS80-40 ATC single site support HMI ver.X10.10-SUS upgrade


 


 

Action

15 Feb 2011

Upgrade ver.2.6 to X10.10-SUS. Handler will not alarm if Head 1 thermal controller is switch off when running single site on thermal head 5.

However, handler still cannot prompt alarm when Head 5 controller is off during dry cycle running. Feedback to Hidaka san.


 

16 Feb 2011

- Checked and found incorrect temperature offset used for shuttle and hotplate.

 Revert to original offset settings.

- Handler started production after PE finished verification on tester program.

- Monitored production run. Lowest temperature observed on thermal controller is 95.3 degree, highest is102.2 degree during running which is within range.


 


 

Cause


 


 

Remarks

PS600 Power Supply for Temperature Controller.

ATC Chiller Neslab ThermoFlex 1400.

LB600i Temperature Controller.

TH170 Head mounted on Index Arm Head 5/6.
