Service History

Subject: UBS31 Index arm, Input arm and output arm unable to pickup error. Operation Panel no response

Handler Model: UBS

Controller: RC520

Date: 21 Jan 2008


 

Symptom

During production, encounter index arm, I/O arm unable to pick up alarm. Found no vacuum.

Vacuum pump breaker found trip. Restore breaker switch to ON for vacuum pump.

EP Regulator display 0 kPa.

Operation panel no response. Only Power ON and OFF button working. Unable to start handler.

Fuse found in good condition.

Remote I/O resit/reset and same problem.

Air In and Tray In I/O module swap. Problem solve.

Restore Air In and Panel In I/O module, no recurrence.


 

Action

Resit Remote IO card problem still same.

Swap Air In and Panel In I/O module ok. Swap back both modules again ok.

Suspect intermittent signal failure/oxidation led to Operation panel no response and EPR no air.


 

Cause


 

Remarks

Index Arm vacuum is supplied by vacuum pump in UBS. If Index arm no vacuum, check breaker for vacuum pump is turn on.

Input & Output Arm vacuum is supplied by factory air supply.
