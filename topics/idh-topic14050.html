Subject: Infineon NS-8040 (NS-103) HMI unable to startup properly

Handler Model: NS-8040 (NS-103)

Controller: RC520

Date: 10 Dec 2010


 

Symptom

Found HMI unable to startup properly and hang, only showing handler's taskbar.


 


 

Action

Abort HMI, and found out background SPEL having

"SRC Fileload error (Getkey)!!:

section=TorqueControl:Key=Y

Dimension: Task=32:

ErrorFilename=c:\NS8000\run\contact.dat.

ErrorFilename=c:\NS8000\run\binassignoff.dat


 

Select default setup file CSP5x5, HMI able to startup.

Once select another setupfile same error happened.

Found out total x6 setupfiles with corrupted contact.dat and binassignoff.dat.

Deleted affected setupfiles and told customer about problem.


 


 

Cause

HMI setup files corrupted causing HMI unable to startup(hang up).
