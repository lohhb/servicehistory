Subject: NS-8080 encountered Output Arm unable to Home, "2Ack error"

Handler Model: NS-8080 (NS8080-8) 8 Hands High UPH

Controller: RC520

Date: 7 Jun 2010


 

Symptom

NS-8080 High UPH handler Output Arm unable to HOME, 2Ack error.


 


 

Action

1.   Using Z-Teach program, output arm 2Ack error. Some kind of communication error.

2.   Swap RS-485#1 and RS-485#2. Able to HOME input arm with output arm port. RS-485 card should be ok.

3.   Further investigation carried out by KH/Hidaka-san.

4.   Japan to send control board, distribution board A and actuator unit.
