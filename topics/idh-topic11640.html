Service History

Subject: NS-7080W Index Device drop error especially 100 degC above testing

Handler Model: NS-7080W(NS7KW-06)

Controller: RC520

Date: 5 Mar 2010


 

Symptom

Device drop error especially 100degrees above testing.


 


 

Action

Found device to be stuck in socket after testing and causing the device drop error.

During testing, sensor is still ON. After testing, as index arm rises, device dislodged and stuck in socket.

50% speed is ok. Any speed faster will have device drop error.

Checked P51 and offset y-point -1.2mm.

Verified in AMBIENT 70% speed and ok.

At 100 degrees testing, index arm unable to pick up device from shuttle. During down motion, sensor is ON but after arm rises, sensor cut off and device pick up error.

Suspect -1.2mm is too much affecting pick up from shuttle.

Reduced -0.8mm, dummy run and ok. Released to production.
