Subject: NS-8080 8 hands Ver1.3 software and Fixed Tray Plate replacement

Handler Model: NS-8080

Controller: RC520

Date: 29 Mar 2010


 

Some of announcement about NS-8080 8 hands Ver 1.3 software and Fixed tray plate replacement.


 

1. We have released Ver1.3 on 3 Mar and 16 Mar.

 But, both of software is same contents.

 So, there is no need to install Ver1.3 software again when you replace  the fixed tray unit. 


 

2. When you replace the fixed tray units, you should be change Y axis of  pose data on the fixed tray by manual.


 

I'm sorry to confuse the software announcement and inconvenience.


 

Note: How to change Pose data

-Set Auto teaching Jig on the Fixed tray position -Open the SPEL software -Click "P"

-Select "Jog & Teaching"

-Output arm move to Fixed tray teaching position by manual.

-Check current position by display of Jog & Teaching -Change the Y axis value of pose data following current position.

-Change the value total 9 position(Fixed tray 1,2,3 have 3 points each.) -Execute "Auto teaching on the fixed tray."
