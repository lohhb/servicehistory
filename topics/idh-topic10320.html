Service History

Subject: T-4190 communication with the module VSRCPG, Manipulator 7 joint 0"

Handler Model: NS-7000 (S/N: 18606)

Controller: RC520

MMI: 1.1c4

Date: 5 Jun 2009


 

Symptom

Power on handler and encounter "T-4190 communication with the module VSRCPG, Manipulator 7 joint 0"error in SPEL.


 


 

Action

5 Jun 2009

Suspect missing stepping motor driver or faulty PG Boards which caused this error. Will continue when replacement stepping motor driver arrive next week.


 

8 Jun 2009

Some issue with NS-7000 s/n: 18606(Intel).

Previously, removed x3pcs of PG boards C-863 from s/n:18606 and tried in Silabs s/n:18732.

Found out Silabs is using PG Boards C-864v1, but we used PG Boards C-863 and tried it in Silabs.

Intel NS-7000 (S/N: 18606) original is using C-863 PG Board x03 pcs on ISA slot.

Silab NS-7000(S/N: 18732) original is using C-864v1 PG Board x03 pcs on ISA slot.

NS-7000 (S/N: 18606), encounter Spel error "T-4190 Communication error with the module, manipulator:7 unit:0", so we changed it back to C-864v1 boards and system,OK.

 

Last week, replaced back x3pcs of C-863 back to s/n 18606 and encounter the same error as in Silabs.

We worried we had damaged x3pcs of C-863 PG Boards.


 

9 Jun 2009

Change jumper the S1 setting for PG3 according to the pdf file for EM086A1700F NS-7000 Maintenance Rev.6 manual, now the error no more.

Handler able to home and dry run.


 


 

Cause

Wrong jumper setting of S1 on PG3 Board for C-863 PG Board caused  "T-4190 Communication error with the module VSRCPG, manipulator:7 unit:0"


 


 

Remarks

C-863 PG Board and C-864v1 PG Board are not compatible because of PG board to Stepper driver wiring different in handler.


 


 

For  "T-4190 Communication error with the module VSRCPG, manipulator:7 unit:0", check the following:

- PG Board DIP switch (jumper) setting

- Correct model of PG board used(C-863, C-864V1, C-873)?

- Resit PG Board

- Resit CPU Board

- PG setting in SPEL Hardware


 

Refer EM086A1700F NS-7000 Maintenance Rev.6 for jumper setting of PG Board for C-863, C-864v1 and C-873, in Chapter 7.4 PG Board (Replacement of Boards in Control Unit) pg 122 to 126.

C-873 is used on PCI slot.
