Service History

Subject: UTAC NS-6040 T4216-Servo real time cmd overrun issue

Handler Model: NS-6040 (NS60-60, S/N: 15542)

Controller: SRC520

Date: 5 Feb 2010


 

Symptom

Feedback by UTAC handler long history of problems and stopped production 6 months ago.

Latest issue was replaced faulty 2.5" HDD and recovery with  NS-6040 Recovery CD FD03S028 Ver 2.0 but encountered T-4216 Servo Real Time Command overrun.


 

In SPEL95, Robot 1~6 Robot status "???".


 

NS-6040 Recovery CD FD03S028  Ver 2.0 content:

(EasyRestore) Win98, MMI 2.0c1, SPEL 95 Ver 2.53i

To use with Pentium 200MHz CPU board


 

Action

-Uninstalled Spel95 and MMI and reinstalled Spel95 version 2.53i and MMI 2.0c12.

Resolved T-4216 Servo Real Time Command overrun problem.


 

-Restored system and POSE data and MCORG all robots but encountered input and output arm F-5041 Torque error in low power state.

-Swapped DU 2 with DU3 with all servo packs remain in respective axis and problem migrated to shuttle 1 and 2. Confirmed DU2 faulty.

-MCORG and Mcal input arm ok but output arm unable to complete Y axis MCORG and Mcal.

-Swapped input and output Y axis servo pack but problem still remain on output arm.

-UTAC provide spare DU2 and output arm Y axis 400W motor.

-2nd set DU2 also encountered same F5041 torque error in low power state issue and found that the SSR (Solid State Relay) in DU2 was missing.

-UTAC provide 3rd set DU2 and replaced with output arm Y axis 400W motor ¨C OK.

-MCORG output arm and dry cycle run handler in 100% speed and release


 

Service report 011919


 


 

Cause

Corrupted SPEL95 and MMI.

Faulty Drive Unit 2.

Faulty 400W Servo Driver for Output Arm Y Axis.
