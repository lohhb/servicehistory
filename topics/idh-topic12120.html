Subject: AMD – UBS 15 , Error 3001 The specified value exceeds the parameter’s upper limit  for INA_HOME after pressed HOME

Handler Model: NS-UBS

Controller: RC520

Date: 17 Sep 2012


 

Symptom

AMD – UBS 15 , Error 3001 The specified value exceeds the parameter’s upper limit  for INA_HOME after pressed HOME.

Starting, have “Door open” issue, check all doors still same, reinstall MMI and SPEL still the same, the end we found one of the door sensor malfunction, so I short the sensor, it works.

But, when we press “Home” button, it show the message below:

Error Task : 3

Error Func : INA_HOME

Error Line : 8

Error Code : 3001

Error Mesg : The specified value exceeds the parameter’s upper limit


 


 

Action

Re-install SPEL CT 2.81b.

Restore SPEL System.

Re-install MMI A12.16-SAS.

Dry run, ok


 


 


 

Cause

Customer did SPEL CT unistallation, reinstallation and System restore causing homing error.
