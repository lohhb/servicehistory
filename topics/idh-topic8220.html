Service History

Subject: Device drop on Index Arm 2 using new Socket Layout Kit

Handler Model: NS-7080W

Controller: RC520

Date: 9~10 Feb 2009


 

Symptom

Xilinx complained that device drop again on index arm 2 sites B, D, F and H using the new Socket Layout Kit (S/LK) that was passed to them via Goto san. SLK NS7KW-LK13 (used to work on this issue last week) was fitted to verify problem and everything was good, just need to do fine-tuning (that is, no device drop when using LK13).


 

Action

 Fine tuning was carried out as there were a lot of floating errors and misplacement at unloader. There was also another error on hand up sensor (input and output arms), whereby after much troubleshooting, it was found out that the individual hand grounding cable was the root cause, thus adjusted and no more error thereafter.

  After solving the above problem, the new SLK was put back and the error still persist.  It was noticed that there was a gap between the compliance unit and SLK, thus, KA lift up both brackets (using 0.5mm washers temporarily) that held the latch and dummy run at 100% without problem.  KA will make shims to replace the temporary washers.


 

Tray measurement was carried out for unloader limit sensor project (limit is 12 trays as requested by Xilinx).


 


 

Cause


 


 

Remarks

Check points for Index Arm Device drop:

- vacuum cup condition, device drop more frequently in hot mode?

- o-ring worn out on plunger

-Check diameter/tolerance of guide pins on socket guide plate

-Check any broken/bent tubing or tubing pressed by cable tie on SLK

-Check air fitting/elbow or gasket not loose on SLK

-Check if guide pins bent/worn out

-Check vacuum filter dirty

-Check vacuum filter put properly

-Check seal for vacuum put properly and not torn

-Check plastic casing for vacuum filter not broken

-Check nest guide pin's hole

-Check o-ring on SLK

-Check metal hooks of Socket Layout Kit (SLK)

-Check correct brand and specification of latch use for (Index) Test Hands

-Check metal latch worn out at Index Test Hand

-Check Index teaching position P50 and P51

-Check if Index Arm crash before or Index Arm bent(Index Arm alignment check)

- if device never drop, adjust vacuum switch sensitivity


 

Correct metal latch of parts No is TL-29A, this is supplied by Tochigiya.
