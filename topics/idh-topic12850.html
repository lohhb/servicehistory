Subject: NS-6040 Software Hang up

Handler Model: NS-6040

Controller: SRC520

Date: 12 Nov 2010


 

Symptom

SCS NS-6040 MMI 10.23 hang whenever PAUSE+RESET.

Irregardless of new or old CPU.

All hang after PAUSE+RESET

Control panel no response.


 


 

Action

Found SPEL has error and stop at START.prg and trying to call a function but no found.

Found program with function is in working memory but unable to link and compile. Thus, all functions in this program not found in programming window.

Upon compile, always has memory error.

Reinstall MMI, same.

Found program memory 2046, increase to 4096.

Able to compile, verified and no hang.

SCS to continue verify other functions and feedback.


 


 

Cause

Customised MMI software bug caused software hang up.
