Service History

Subject: STM NS-8040 Device drop after test/stuck in socket

Handler Model: NS-8040

Controller: RC520

Date: 18 Feb 2009


 

Symptom

Device drop error after test. Found device always in socket.


 


 

Action

Ambient testing NO PROBLEM.

This only happen in HOT MODE. To request KA to check on socket/change kit.

Thermal expansion of kit, device socket etc...

To check on air tubing for possible crack leading to vacuum leak.

Device condition error on output shuttle.

Tried lower release height but to no avail.

Advice to raise contact height to try and check contact air pressure.


 


 

Cause


 


 

Remarks

Check point:

- vacuum strong?

- any broken tubing?

- tubing loose?

- Both Index Arms device dropped?

- same test site having device dropped?

- remove socket guide base jig, does device drop when/during/after contact?

- P50 & P51 Y position check

- Socket guide base jig's guide pin dimension & tolerance? can remove guide pins?

- socket tolerance? use vacuum pen place device in socket and remove device?
