Symptom

Output Arm Hand D cannot HOME, Stepper Motor Driver never generate power, Stepper Motor Driver is hot.


 


 


 


 

Remarks

Replace a Stepper Motor Driver form new buyoff handler S/N: 18728.

Technically, R33ZTA2080200 is the code for NS6000 and R35N700003600 is the code for NS7000.

However, to avoid any confusion, KG(S) use R33ZTA2080200 to prepare the item with their system.

R35N700003600 is not a mistake and can use it with no problem.

According to Epson both are for stepping motor driver (PMD07C), but only "R33ZTA2080200" is actually used. So please use R33ZTA2080200, not R35N700003600.


 

Stepper Motor Driver (PG Driver) PMD07C for NS-6000 & NS-7000 was discontinued around July 2012.

Replacement stepper motor driver is CSD5807N2-P.
