Subject: NS-8080 High UPH 8 Hands input arm hand B sensor doesn't turn off, 2Ack unknown error and servo error

Handler Model: NS-8080 High UPH 8 Hands

Controller: RC520

Date: 4 May 2012


 

Symptom

NS8080-02 input arm hand B sensor doesn't turn off, 2Ack unknown error and servo error.

- Handler unable to home and start.


 


 


 

Action

- SPEL Input IO #48( for Input Hands HOME) not turn off when hand B was pulled down. Rest of hands ok.

- Replaced Input hand b home sensor(lot BA) with spare(lot BG).

- Performed hand height calibration for input arm.

- Cycle run handler ok.

- SR:013194, warranty S0515.
