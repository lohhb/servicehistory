Service History

Subject: "Initial Start Move Wait" in Tester I/F of MMI at Xilinx

Handler Model: NS-7080W

Controller: RC520

Date: 1 Nov 2006


 

Information

Regarding "Initial Start Move Wait" in Tester I/F of MMI,

this wait time is for Hot mode only.

When start handler with hot mode, first IC will contact to socket,

then, normally test start request will send to tester, but, first IC contact only wait to send start request until passed "Initial start move wait" time.

This function is to wait for temperature to be stable.

It was found that first IC normally will contact at lower temperature. So, handler have to wait for time to recover to the correct temperature.
