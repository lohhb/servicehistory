Subject: NS-7000 MMI unable to launch. Output Arm hang during homing.

Handler Model: NS-7000 (S/N: 18732)

Controller: RC520 (S/N: 01-01379)  CPU Board PC-686BX        3 PG Boards C-864v1

Date: 18 Sept 2011


 

Symptom

Silab NS7000 MMI unable to launch, output arm hang up during homing. (Phone support)


 


 

Action

- SPEL CT main and background function not set properly.

a) In SPEL, ensure "Main" set as "Main Function"

b) In SPEL, ensure "startup" set as "Background Function"


 - "output hand B sensor wire cut, hand A sensor block not in position".


 


 

Cause
