Subject: NS-6040 Shuttle 2 unable to home. Unable to complete MCAL and MCORG.

Handler Model: NS-6040 (S/N: 15477)

Controller: SRC520

Date: 12 Jun 2009


 

Symptom

Shuttle 2 unable to home. Unable to complete MCAL and MCORG.


 


 

Action

Calibrate Robot 6 for Shuttle 2 pulse 0 by calib command in SPEL command execution windows.

Pulse 0 position is when the home sensor is cut.

Dry run handler, ok


 


 

Cause


 

Remarks

If unable to execute MCORG, can try to set HOFS to 0  for the affected axis.

>HOFS 0,0,0,0  ' print screen HOFS first, not affected axis don't key in as 0

after that try MCORG that axis again.


 

If even calib command don't work, check:

- sensor

- senor's cable

- motor

- motor signal connector

- servo driver

- cable connection to Drive Unit
