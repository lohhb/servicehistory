Service History

Subject: NS-6040 Input Hand 1 unable to pickup device and Output Arm reached the limit of motion range

Handler Model: NS-6040 (S/N: 18579)

Controller: SRC520

Date: 23~25 Feb 2010


 

Symptom

NS-6040 Input Hand 1 unable to pickup device and Output Arm reached the limit of motion range.


 

Action

23 Feb 2010

-Teaching for teaching points and running check found input arm Hand 1 unable to pickup device and output arm homing sensor error. Input arm vacuum switches wring wrong connection and output arm U & Z height too low. Vacuum switch wring connection changed for input arm and output arm U & Z height re-teach.


 


 

24 Feb 2010

-Install Win98 recovery CD but unable to install. Tried a few different version of recovery CD but still unable to install. Install Win95 using clone CD from other system, teaching of teaching points all locations except for Index Arm teaching and mcorg done but handler unable to start. Tried another clone CD and it works fine but "Arm reached the limit of motion range" error occurred after output arm pick up device from output shuttle 1 before moving to auto1 unloader.

Tried bin to Auto 2 also same problem.

Restored back previous robot 3 pose data and it works fine.


 


 

25 Feb 2010

1) Encountered output arm limit error. Found out Output Arm(robot 3) Auto1,2,3, Fix 1,2 and 3 the value of, Z and U are is 0.

2) Re-do Output arm Z and U height teaching for Output Arm robot3.

Verified pose data in SPEL, Z and U axis data exist after teaching.

3) Verified output arm, running check ok

4) Replaced controller cooling fan. Power on and verified cooling fan OK


 


 

Cause


 

Remarks

T- 4001 "The Arm reached the limit of motion range" during cycling and running mode.

Usually, after HDD Recovery if Input or Output Hands encountered error 4001 "The Arm reached the limit of motion range", it is advisable to make sure that Input or Output Arm Z & U in mnp02.pos or mnp03.pos are not zero, usually Z & U are -20 to -25mm.

Use GenPos in SPEL to teach the Z & U height for Input Hands height teaching with jig place on Input Shuttle and for Output Hands height teaching with jig place on Output Shuttle.

For Robot 2 Input Arm, make sure Z & U are not zero, around -20 to -25mm for the following points:

P50 ' Load Tray

P51 'Input Shuttle1

P52 'Input Shuttle2

P53 'HotPlate1

P54 'HotPlate2

For Robot 3 Output Arm, make sure Z & U are not zero, around 20 to -25mm for the following points:

P81 'Auto1

P82 'Auto2

P83 'Auto3

P84 'Fix1

P85 'Fix2

P94 'Fix3

P95 'AutoMat

P96 'Fix4

P86 'Output Shuttle1

P87 'Output Shuttle2
