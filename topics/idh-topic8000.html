Service History

Subject: AMD UBS:- 2 issues, home sensor OFF and no error message/DU no error during hang up.

Handler Model: UBS

Controller: RC520

Date: 2 Dec 2010


 

Symptom

AMD UBS has 2 issues:

1) home sensor OFF and no error message/DU no error during hang up.

2) pickup hand vibrating and unable to home fully.


 


 

Action

For home sensor OFF, changed PG board and OK.


 

For hang up, Tester off-line running with device, OK

Die inspection enabled, after pick up from output shuttle 2, hang up. Shuttle 1 able to pick up. 3rd and 4th after pick up hang.

MMI event log, no error. SPEL error history shows T-4241, manipulator 3 joint 1,

The detected speed at the Low-power state is too fast.

Output arm picker head can MCAL but main axis unable to MCAL but able to HOME.

Swap servo driver, N.G.

Swap DU2 as event log shows error for both input/output arm. N.G

Request customer to send mcd file to edit.

After edit mcd file, dry run dummy units with die inspection enabled, no T-4241.

However, still unable to mcal but able to home for output arm main axis. For this, suspect possible hardware failure like motor or SSR or power supply. Customer to change during next PM cal.

Temporary to run production with new mcd file.


 


 

Cause


 


 

Remarks

Check points:

- check axis mechanical movement is smooth or not

- any loose screw/bolt

- replace servo driver

- signal connector of motor connected properly / broken wires
