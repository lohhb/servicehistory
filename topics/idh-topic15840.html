Service History

Subject:  Tray End at Front operation panel for NS-8040 unable to function

Handler Model: NS-8040(S/N: 181213, NS-100)

Controller: RC520

Date: 19 Apr 2011


 

Symptom

Customer informed Tray End at Front operation panel for NS-8040 unable to function.


 


 

Action

MMI I/O Utility check, Tray End & Tray Feed able to response.Unable to find any abnormalities.Also resit connector to I/O Module(EIN30) that is below Loader.Offline with devices dry run with Tray End & Tray Feed function for Retest mode, ok.


 


 

Cause


 


 

Remarks

There are two EIN30 I/O Modules.

One EIN30 16CH is below Loader, later stage this was relocated to near the vacuum valve for Empty Tray arm.

The other Input I/O module EIN30  8 CH is at Index area.
