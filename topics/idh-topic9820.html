Subject: Xilinx TR-60

Handler Model: TR-60 (S/N: 121274)

Controller: RC520

Date: 13 Mar 2009


 

Symptom


 


 

Action

Set up to run cold mode.  Cold mode fine-tuning carried out.  While running jam rate, 2 problems occurred:

a) soak boat stuck at the top of robot 10 as it hit a bracket at top most of robot 10,  thus return to room.  In between, open up the chamber door and verified this problem.  Upon return to room temperature, carried on running at 50% and soak boat have no problem and can fit nicely into the bracket.  The bracket might have contracted due to cold temp.   Maybe, in cold temp, try to fine tune the offset for robot 10.

b) A few times, device cannot be picked up from index arm (hand C) and retry a few times but to no avail.  After much verification, suspected that hand C and B is swapped somewhere.

Also, the device surface will get wetter after a few runs as the handler did not came with a defroster.

 Please check on this issue again on Monday and do the necessary swap.


 

13 Mar 2009

JAM @ soak boat when running -40degrees.

Found input buffer to be too high and soak boat stuck after transfer from test buffer.

To offset input buffer height in cold running to try again.


 

Cause

Found input buffer to be too high and soak boat stuck after transfer from test buffer.
