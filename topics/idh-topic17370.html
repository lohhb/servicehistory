Service History

Subject: NS-7000 after HDD Recovery

1)“Series port open error” Communication error with the ionizer controller

2)“The target temperature can not be set"

Handler Model: NS-7000 (S/N: 181260) STM

Controller: RC520

Date: 19 & 23 Dec 2011


 

Symptom

NS-7000 after HDD Recovery

1)“Series port open error” Communication error with the ionizer controller

2)“The target temperature can not be set"

 


 

Action

-Found an error 1 “Series port open error” Communication error with the ionizer controller.
-3360 ionizer controller sensor 1 alarm.
-Use IPA to clean the chamber ionizer discharge pin, error still persist.
-Check device manager COM port RS485 was missing.
-Need to install the RS485 drive and MKS Ion system software. On Pending due to CD drive at office.

-Error 2 “The target temperature can not be set.
-Check the thermo control board connector. Ok
-Calibration for the thermo control board but found address port open error.
-Swapped the yamatake board from system 2, error still persist.

 

23 Dec 2011

-“The target temperature can not be set error.
-Check the COM port and wire connectivity. -OK
-Swapped SRC controller from another handler and the error still persists.
-Found the Yamatake board DIP switch setting was wrongly set.
-Reset the DIP switch setting error gone.
-Dummy dry run . OK


 


 


 

Cause

1)“Series port open error” Communication error with the ionizer controller

 - due to  COM port RS485 device manager was missing.


 

2)“The target temperature can not be set"

- due to  Yamatake board DIP switch setting was wrongly set.
