Service History

Subject: NS60-06 Error T-4001 input arm limit error during initial start.

Handler Model: NS-6040(UTAC, NS60-06, S/N: 15542)

Controller: SRC520

Date: 28 Aug 2008


 

Symptom

MMI display T4001 error code, the Arm reached the limit of motion range.

Input arm hit soft limit error when initial start.


 


 

Action

Check  input arm hard limit pulse position, open up input arm J1 and J2 range, also cannot.

Advise customer to restore back old system and project files and re teach position again because error happened after input arm teaching.


 


 

Cause


 


 

Remarks

Can check the Input pickup hand Z & U height in mnp02.pos.

Usually, the Z & U pickup height for Input/Output Hand should be around -25mm.


 

T- 4001 "The Arm reached the limit of motion range" during cycling and running mode.

Usually, after HDD Recovery if Input or Output Hands encountered error 4001 "The Arm reached the limit of motion range", it is advisable to make sure that Input or Output Arm Z & U in mnp02.pos or mnp03.pos are not zero, usually Z & U are -20 to -25mm.

Use GenPos in SPEL to teach the Z & U height for Input Hands height teaching with jig place on Input Shuttle and for Output Hands height teaching with jig place on Output Shuttle.


 

For Robot 2 Input Arm, make sure Z & U are not zero, around -20 to -25mm for the following points:

P50 ' Load Tray

P51 'Input Shuttle1

P52 'Input Shuttle2

P53 'HotPlate1

P54 'HotPlate2


 

For Robot 3 Output Arm, make sure Z & U are not zero, around 20 to -25mm for the following points:

P81 'Auto1

P82 'Auto2

P83 'Auto3

P84 'Fix1

P85 'Fix2

P94 'Fix3

P95 'AutoMat

P96 'Fix4

P86 'Output Shuttle1

P87 'Output Shuttle2
