Service History

Subject: UTAC- ESD Class 0 installation on NS70-41 and LAN conflict will parallel port in BIOS cause RC520 hang up(Blue screen)

Handler Model: NS-7000

Controller: RC520

Date: 20~21 Aug 2009, 1 Sep 2009


 

Symptom

UTAC- ESD Class 0 installation on NS70-41.

1 Sep 2009

Unable to resolve LAN driver issue.

RC520 will hang(blue screen) upon installing driver and need to reset the RC520 controller. Awaiting advise from Hidaka san.


 

Action

Completed hardware installation and setup.

- close loop board and close loop cables

- 4x ION ionizer fans

- MKS controller and sensors for chamber, input and output

- MKS driver and software

- Upgraded MMI version

- PCI card in RC520 for MKS controller

- replaced new chamber ionizer controller


 

1 Sep 2009

- Ground loader and reloaders fingers with grounding cable.

- Install new front panel for MKS controller.

- Unable to resolve LAN driver issue.

 PC will hang upon installing driver and need to reset the RC520 controller. Awaiting advise from Hidaka san.


 

2 Sep 2009

Disable parallel port in BIOS and install LAN driver from Contec board driver cd.


 

Cause

LAN setting conflict with parallel port in BIOS cause PC hang upon installing LAN driver.


 

Remarks

For blue screen after installation of MKS software, load back last known good configuration of window 2k.

For Ethernet unable to recognize and had to disable parallel port in BIOS and install ethernet driver again.
