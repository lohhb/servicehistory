Service History

Subject: NS-8080 (8 Hands) "Home sensor does not turn on" error

Handler Model: NS-8080 8 Hands High UPH

Controller: RC520

Date: 25 Nov 2010


 

Symptom

"Home sensor does not turn on" error.

Input Arm, 3-4 pickup hand spline shaft condition not smooth. Mostly unable to retract up with servo off conditions.


 


 

Action

Dry run with dummy units, only occur in HOT mode and mostly after place device or pocket device presence check.

Saw from SPEL I/O, it seems home sensor is delayed, intermittent.

When error happens, 20-30seconds later, sensor ON.

PAUSE+START+PAUSE+START motion to trace which hand N.G.

Suspect hand A or C.

Connector check for loose connection by hand.

Dry run again, this time, SPEL I/O seems more responsive.

No home sensor error occurrence for few trays.

Released to production for monitoring. Suspect connector loose.


 


 

Cause

Loose connection on the 8 hands assembly.
