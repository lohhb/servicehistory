Service History

Subject: NS-7000 Empty Tray Reloader 2 having frequent Tray Separation Error and No Empty Tray.

Handler Model: NS-7000 (S/N: 181173)

Controller: RC520

Date: 10 Feb 2009


 

Symptom

EEMS - Empty Tray Reloader 2 having frequent Tray Separation Error and No Empty Tray.


 


 

Action

Handler having false alarm even with presence of empty trays at Empty Tray Reloader2. Suspect wiring problem, check sensors' wiring ¨C no disconnection. Adjusted Empty Tray Reloader2's cylinder middle position. Cycle run handler with empty trays ¨C No occurrence. After few hours, customer reported errors occur again and observe at SPEL's i/o window related sensor's i/o flickering. Swapped Empty Tray Reloader's RLD2IN2 i/o module with RLD2IN1. Cycle run handler with empty trays for 1hr and advise customer to monitor.


 


 

Cause

Faulty Input I/O Module for Empty Tray Reloader RLD2IN2.
