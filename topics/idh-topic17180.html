Service History

Subject: NS-8080 High UPH 8 Hands Output Hand C servo error & RS485 error

Handler Model: NS-8080 (S/N: 181666, NS8080-02)

Controller: RC520 (S/N: 03-02143)

Date: 9 Nov 2011


 

Symptom

UTAC NS8080-02 High UPH 8 Hands – Output Hand C servo error & RS485 error.


 


 


 

Action

Only Output Hand A & B with rubber cups, the rest of the Hands not in use(also no suction cup).

During Homing, Output Hands all hands able to home normally but Hand C go down in step motion(1 or 2 cm) and sometimes hit the rib.

Sometimes can complete Home but during dry running Hand C servo error or RS485 unknown error or RS485 2Ack error.

SPEL Input Bit 52 (8 Hands HOME check for Output Hands) check is ON after Home, normal.

Pull Hand A or B down, Input Bit 52 is OFF, pull Hand Input Bit 52 is still ON.

Already resit the 4 connectors for Output Hand C Home sensor, Vac&C-Air sensor, Motor & Encoder.

Pull all Actuators, Hand E ball spline movement abit unsmooth, Hand E will not spring up. Rest of the Hands are smooth for motion.

Auto Hand Height Cal for Output 8 Hands, can complete A & B but for C, totally never move but Hand C vacuum activated.

Error message was “vacuum sensor error. Set Vacuum pump and tuning on”.

Exit Auto Hand Height Cal GUI and perform Auto Hand Height Cal. Same problem as above.

-------------------------

1. NS8080-02 output arm c RS485 unknown error, servo error, hand up sensor doesn’t turn off.

- Output arm hand c unable to home. Jerking motion. Moved down and up during homing and hand height calibration.

- Found SPEL Input bit #52(For Output Hands HOME) led light did not turn off when pulled down hand C. Pulled down hand a and b ok.

- Resit home sensor connector ok but problem occurred again later.

->Suspect faulty Home sensor for Output Hand C

-------------------------


 

Cause

Suspect faulty Home sensor for Output Hand C.


 


 

Remarks

Warranty S0502 raised for Z-axis module C home sensor R33N800018200 (PM-L24-R).
