Information

Explanation on End to Start for NS-8000.


 

 Please see End to Start means as follows;


 

 End = End signal have received from tester.

 Start = Start signal send to tester.


 

 So, it is inclusive Index time, delay time, IF communication time and

 waiting for any processing.

 I'm sorry but it does not include test time.


 

 Ex: Why Oct sites takes 3.1 sec but Quad site takes 1.7 sec.

   For Oct case: There is release the devices to Input shuttle by Input

 hand and pick up the devices from output shuttle by output shuttle

 each 2 motion.


 

 For Quad case: There is release the devices to Input shuttle by

 Input hand and Pick up the devices by output shuttle by output hand

 each one motion.


 

 It is different for waiting for processing time between Quad and

 Oct sites.
