Service History

Subject: IDT Shuttle 1 1-1 top sensor  replacement

Handler Model: NS-7000

Controller: RC520

Date: 30 Jun 2009


 

Symptom

Output Shuttle 1 1-1 emitter top sensor replacement.


 


 

Action

Handler hang during starting up Windows 2000 in the color screen at around 75% loading. Power off handler, power up handler, after loading MMI completed, mouse cannot move, keyboard no response but NUMLOCK green LED flickers. Power off handler, power up handler, in BIOS set HDD-0 first boot device, CD-ROM 2nd boot device, floppy 3rd bootup device. Restart handler, no hang up and able to load to MMI but upon power on the operation panel, MMI display uniwire diconnection error. Checked all I/O modules power conditions, seem ok. Remove uniwire connector at controller and check wires, reset Remote I/O board.

Run handler with 1 tray of 10x10 devices, ok. Cause of hang up unknown.
