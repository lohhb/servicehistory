Service History

Subject: NS-7000 error F 5004 Initialization time out error with absolute encoder

Handler Model: NS-7000 (UTAC, NS70-06)

Controller: RC520

Date: 8 Apr 2013


 

Symptom

UTAC – NS70-06 encountered SPEL error F-5004 Initialization time out error with absolute encoder, manipulator 1 joint 2.
(phone support)


 


 

Action

Initial torque error on index arm and found SSR NG.

Replaced SSR and then encountered SPEL F-5004 error.

Replaced whole DU1 and MIB same. Only robot 1 status in SPEL window “???”.
Advised Ray to check Y-axis motor.

After replaced Index Y-axis motor, reset encoder and start spel, error gone.


 


 

Cause

Faulty Index Y Axis motor.
