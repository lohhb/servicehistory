Service History

Subject: NS-7000 ESD class 0 EP5 controller unable to control the ion balance for the chamber

Handler Model: NS-7000 (NS70-34, S/N: 181008)

Controller: RC520

Date: 9~11 Feb 2011


 

Symptom

Problem encountered:

1.This upgrade kit came with a new EP5 controller and non-sensor ionizer electrode for chamber. However, the ionizer electrode screw hole is size m3, and cannot secure to the m4 screw hole in the chamber.


 

2.The EP5 controller unable to control the ion balance for the chamber. Slight adjustment to the ionizer controller adjustment knob will see the ion balance going to +30 or ¨C30 volts. St Calamba encountered same problem when they installed the EP5 and Suzuki went down to troubleshoot this issue. Request Iwata san to share their findings.


 

3. When triggered the 128 and 129 ionizer fan SPEL I/O, fan and ionizer can work. But when pushed start button to run handler, ionizer controller unable to turn on itself. Need to verify again tomorrow if the chamber fan is enabled in temperature parameter. Maybe ionizer controller will turn on only when fan is ON.


 


 

Action

10 Feb 2011

- Verified with spare EP4 controller but still unable to control the ion balance in chamber.

- Checked MKS controller control 1 RJ-9 connector and found wrong wiring connection. Confirmed again on other ESD class 0 handlers for the correct connection.

As confirmed by Iwata san, he has spare wiring and will bring in the part tomorrow morning.

Continue buy off items. Add extra groundings to input/output arm and empty tray arm.

Completed resistivity check and partial ionizer cpm check(without the chamber).


 

11 Feb 2011

- Replaced spare MKS control 1 RJ-9 cable and verified chamber ion balance able to control within range.

- Replaced new set of chamber discharge needles. Completed chamber ionizer cpm check and manual trigger check.

- Release handler.

SR ref: 013155


 


 

Cause

MKS controller control 1 RJ-9 connector and found wrong wiring connection.
