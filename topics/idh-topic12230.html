Service History

Subject: NS-8080 8 Hands Z Axis servo error

Handler Model: NS-8080 8 Hands High UPH

Controller: RC520

Date: 5 Apr 2010


 

Announcement

5 Apr 2010 email from Hidaka

Advance notice of 8 hands Z axis servo error.


 

Problem: 8 hands Z axis servo error

This error cannot be recover by "Pause + Reset", it can be recovered by Power Off & Power On front operation panel.


 

Reason:

Software bug for 8 Hands Z Axis servo driver firmware.


 

Countermeasure:

Upgrade software (firmware for Z Axis servo driver in) Driver Board and Control Board


 

When we fix the shipping schedule of the special tools and software, we will inform to customer.


 


 

Remarks

25 May 2010

NS8080-02 input/output 8-hands Z-axis driver software upgrade and fix tray unit replacement

Upgrade with upgrade tools JTAGjet 2000I(signum system) from Japan.

After upgrade simulate Z-axis error and able to recover error with Pause+Reset. Run dry cycle at 80% speed for 2 hrs ok.

Also replaced old fix tray unit with new type to prevent output arm from hitting rear door cover.
