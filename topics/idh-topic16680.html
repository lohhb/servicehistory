Subject: Infineon NS-013 HMI & Windows 2000 sluggish response

Handler Model: NS-8040 (S/N: 181860)

Controller: RC520 (S/N: 03-10098)

Date: 16 Jun 2011


 

Symptom

Infineon NS-013 HMI & Windows 2000 sluggish response.


 


 

Action

NS-8040 Axes running ok but HMI(click on Unitset, Deviceset, Maintenance icon)& Windows 2000 response on the screen is sluggish.When trying to create and name the backup folder, keyboard no response.Unplug and plug USB connector for keyboard, NS-8040 HMI screen response back to normal. Plug keyboard to another USB port, ok.Backup data for NS-013.Swapped keyboard with NS-108, ok.Request customer to monitor HMI response for NS-013 & NS-108.


 


 

Cause

 USB keyboard not connected properly caused slow response of HMI & Windows 2000.
