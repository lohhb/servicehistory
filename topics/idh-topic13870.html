Service History

Subject: TR60 Chamber Fan error and Input Hand pickup error

Handler Model: TR60 (S/N:121175) Renesas

Controller: RC520 S/N: 01-20947)

Date:


 

Symptom

Renesas TR60 Chamber Fan error and Input Hand pickup error.

RETRY button always ON.

 


 

Action

1) Encounter chamber FAN error middle of production and input arm pick up error.
2) Chamber fan replaced.
3) Input arm pick up error due to pick up height too low and hit tray.
4) During MMI offset of pick up height for input arm, too high and encountered pick up error, handler hang.
5) When handler alarm for pick up error, tower light red, MMI shows running, control panel related buttons flashing BUT retry button always on.
6) Pressing other buttons no response. Only way is power off button.
7) Restart pc same.
8) As long as able to pick up device, no hang up. But when pick up error like half tray or too high, will hang up.
9) To check SPEL user I/O Input bit of RETRY button as it is always ON when error happens.
10) Upon restart handler and no running, from SPEL already can see Input bit of RETRY button always ON.
11) Suspect continuous feedback(short condition) leading to hang up (no response from all buttons) when error happens requiring retry button.
12) This explains why when error happens, all related buttons flashing but retry always on when no button was pressed.
13) Customer replaced RETRY button and ok.
14) When device pick up error, no more hang up.
