Subject: UTAC NS8080-01 8 hands handler having  "Hand up sensor error" with Output Arm

Handler Model: NS-8080 (NS8080-01) 8 Hands High UPH

Controller: RC520

Date: 10 Dec 2010


 

Symptom

Customer changed to Quad setup and only enable site C.

Output Arm using Hand E to pickup unit.

Same situation, error hand up sensor doesn't turn on happen when Output Arm move above Output Shuttle 1.


 


 

Action

Customer changed to Quad setup and only enable site C.

Output Arm using Hand E to pickup unit.

Same situation, error hand up sensor doesn't turn on happen when Output Arm move above Output Shuttle 1.

Tried pushed in all connectors on distribution board, error persist.

Tried adjust Output Arm hand actuators' HOFS in Zteach, error persist.

Fine adjustment in Output Arm in HMI Unitset. Increase Z speed to 70%. and introduce 0.01sec robot settling time.

Advise customer to monitor, will consult hidaka for advises.

Follow up refer 0273-NS8K
