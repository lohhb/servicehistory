Subject: NS8080-02 output arm hand H intermittently vacuum sensor off error

Handler Model: NS-8080 High UPH 8 Hands (S/N: 181666)

Controller: RC520 (S/N: 03-02143)

Date: 8 Jul 2011


 

Symptom

NS8080-02 output arm hand H intermittently vacuum sensor off error.


 


 

Action

NS8080-02 output arm hand H intermittently vacuum sensor off error.

- Replaced output z-axis module hand H.

- Checked hand pitch and align.

- Performed master calibration for output. Encountered hand c servo error unable to detect vacuum. Removed hand adaptor and blow filter ok.

- Input hand d sometimes cannot pickup from loader. Check hand d z-height and carried out hand height calibration.

- Check input and output HMI z offset and reset to 0.

- Release for production. Collect handler observer, log and table data.

Raise warranty S0495 for hand H.
