Service History

Subject: NS-8080  8 Hands Input arm hand up sensor doesn't turn on issue

Handler Model: NS-8080 (NS8080-03, S/N: 181695) 8 Hands High UPH

Controller: RC520

Date: 13~14 Jan 2011


 

Symptom

UTAC  NS8080-03(181695) input arm hand up sensor doesn't turn on issue

When input arm at shuttle 1 and 2 position, can see in SPEL  IO #48 delay to ON.


 


 

Action

Checked all z-axis module and found one of the ball spline o-ring stuck between the shaft spring.

Removed all o-rings from the input arm.

Can see improvement in the responds of IO #48.

Under monitoring.


 

14 Jan 2011

Handler when boot into HMI will prompt either input or output z-axis servo error. Unable to home or run. Check on SPEL IO #48, 49, 50, 52, 53, 54 not ON.

Found IO Arm IN1 module address set wrongly. Address same as IO Arm IN2.

Revert address back to original setting and resolved z-axis servo error.

Suspected someone might have accidentally changed it as the event log registered uni-wire cable 1 disconnection error around 8am.

For hand up sensor doesn't turn on error, Check all cables behind distribution board SKP479-1A and 1B and re-sit all connectors on them. Dry cycle handler and released for production. Handler still in production as of 7pm.


 


 

Cause


 


 

Remarks

Use multi-meter to check below

1. Between D-G  NO short-circuit.

(Resistance: >200 Ohm( Value will be depended on equipment))

2. Between 0-24V  NO short-circuit.

(Resistance: >few kOhm (Value will be depended on equipment))
