Service History

Subject: UBS#17 setting for CONTEC COM-2(PCI)H communication card

Handler Model: NS-UBS

Controller: RC520

Date: 30 Dec 2011


 

Symptom


 


 


 

Action

Setting for the CONTEC COM-2(PCI)H communication card, refer below:

NSUBS 17

S/N: 151078

MMI A11.15-SAS

SPEL 2.81b

Windows 2000 Device Manager Address Range

IFUNT 100 E400;E800

CONTEC COM-2 : E000

CONTEC PO-64 : EC00

COM 4 Buffer setting in Windows 2000:

BPS 57600

DATA BIT 8

PARITY NONE

STOP BIT 2

FLOW CONTROL NONE

RECEIVER BUFFER 32

TRANSMIT BUFFER 32

Installation notes on 22 Aug 2011:

-COM-2(PCI)H (RS232 2-port card) and cable installed in RC520.RS232 conflict problem found causing MMI hand up after reboot.
 - Change COM-2(PCI)H card PCI slot, problem solved. Do not use PCI-4
 PIO64 card on PCI-1, IFUNT200 card on PCI-2, COM-2(PCI)H card on PCI-3

-there is no need to add SPEL I/O port address in SPEL for COM-2(PCI)H card.
