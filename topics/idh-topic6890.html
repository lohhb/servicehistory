Service History

Subject: NS7000 detection sensor error at Output shuttle 2

Handler Model: NS-7000 (STATSChippac)

Controller: RC520

Date: 3 Jun 2008


 

Symptom

During production, Output shuttle detection error occurred intermittently.


 


 

Action

Alignment check on the Output shuttle sensors carried out.

Trying to swap timing sensor with side sensor found one of the timing sensor pin not sit in properly.

Problem solved after reconnecting the pin.


 


 

Cause


 


 

Remarks

Check points for Output Shuttle Sensor Detection Error:

- photoelectric switch(timing sensor)

- resit connector for timing sensor

- any uneven slide plate of output shuttle

- tunnel block missing on the output shuttle slide plate

- any hole on tunnel block

- Gate Sensor Board

- Input IO Module for Timing sensor

- Timing sensor positions( X direction and height)

- Use of retrofit kit, tick "Use Side Device Exist Sensor" in the Device Exist Sensor Window
