Service History

Subject:  “Error #49 Write Fault” during HDD recovery

Handler Model: NS-6040 (S/N: 18485), Type 1, 3.5" HDD

Customer No: H10-NS64

Controller: SRC-520CE

Date: 17 Dec 2007


 

Symptom

Upgrade to PC-686BX(PC)-EPP850 CPU Board for NS-6040 (H10-NS64), Type 1.

Encountered “Error #49 Write Fault” during HDD recovery when using 2.5” 60GB Seagate HDD.

Win95->Windows 2000

MMI 2.1b->MMI 3.0c

SPEL95 2.53i->SPELCT 2.81b


 


 

Action

Replaced 2.5" 60GB HDD to 2.5” 20GB Hitachi HDD.

Use Recovery CD (FD07S057, ver 3.0c, 2007/06/20) to perform recovery, ok.

Restore DU 1,2 & 3 data in SPEL CT.

Restore Robot Manipulator data in SPEL CT.

Copy & paste old pose data to SPEL CT project.

Handler able to dry run and dummy device alignment.


 


 

Cause

2.5" 60GB Seagate HDD is not compatible for Type 1 CPU Board upgrade.


 


 

Remarks

Refer procedure on NS-6040 CPU Board Upgrade PC-686 BX (Windows 2000 / SPEL CT)
