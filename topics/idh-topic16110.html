Service History

Subject: NS8080-03 output hand G hand up sensor error issue

Handler Model: NS-8080 High UPH 8 Hands (UTAC)

Controller: RC520

Date: 30 May 2011


 

Symptom

NS8080-03 output hand G hand up sensor error issue.


 


 

Action

- Meet up with Auw, Hung, Roel, Guna for discussion.

- Feedback by Guna, he already checked hand G and found Z movement not smooth, and applied grease to the shaft. All 8 sites also enabled and running fine from morning.

- Proposed by Japan to check and adjust handler’s 24V supply as a temporary measure. But production unable to release an hour system time as requested. Only 15 mins allowed to check on the 24v supply first.

- Checked voltage on the output arm, 26V. Collected handler data to send to Hidaka san.

- Released handler to continue production. Under monitoring.


 

NS8080-03 input hand B, output hand G hand up sensor error.

Japan to send 2x z-axis module and 2x distribution board.

Hidaka san to update impact of 24v difference and parts ETA.


 

26 Jun 2011

NS8080-03 output hand G hand up sensor error issue

- Measured PSU1 N1-P1 terminal at 26.3vdc.

- Measured output arm CN9 input supply at 26.0vdc.

- Tune down voltage at PSU1 to 24.7 (+/-0.2) vdc.

- Measured input arm CN9 at 24.5vdc, output arm at 24.5vdc. 24vdc adjustment completed.

- Check all metal suction cups for inout/output arm the same. Remove 1 metal cup to take measurement as per Hidaka-san request.(refer to attached photos)

- Check z-height of output arm all hands at rib position. Noticed some difference in height for some hands.(refer to attached photos)

- Check output arm HOFS in z-teach and perform hand height calibration for both input and output arm. Check again for all hands at rib position ok.

- Release handler for production.


 

Temporary measures for NS8080-03 are:

1)to replace input hand B z-axis module and input distribution board, output hand G z-axis module and output distribution board,

2)tune down to 24vdc. ETA of parts to be advise by Hidaka san.
