Subject: UTAC – NS8080-01 input arm hand F/H vibrating issue

Handler Model: UTAC NS-8080 (NS8080-01, S/N: 181665)

Controller: RC520

Date: 28 Apr 2011


 

Symptom

UTAC – NS8080-01 input arm hand F/H vibrating issue.


 


 

Action

- Replaced driver module for hand F/H.

- Replaced z-axis module hand F and H.

- Check new hand F/H hand height by Auto teaching.exe. Fine tune height with z-teach.

- Check new hand F/H XY release position at input shuttle. Fine tune hand H XY release position by adjusting hand tip.

- Checked vacuum and counter air for hand F/H ok.

- Run dry cycle mode. Release handler for production.

- Warranty S0478 closed.
