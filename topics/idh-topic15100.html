Service History

Subject: NS-7000 encountered

1) Cover open” door switch not functional

2) One Cycle button cannot light up for Rear & Front Operation Panel but the function is ok.

3) Start button for Rear Operation Panel cannot work.

Handler Model: NS-7000 (S/N:18825, NS-02) IDT(M)

Controller: RC520 (S/N: 01-01775)

Date: 22 Sep 2011


 

Symptom

1) Cover open” door switch not functional

2) One Cycle button cannot light up for Rear & Front Operation Panel but the function is ok.

3) Start & Home buttons for Rear Operation Panel cannot work.


 


 

Action

21 Sep 2011

1) Cover open” door switch not functional

Swapped Front Operation Panel PCB, confirmed that VD1 zener diode open caused the “Cover Open” but handler still running production without stopping.
Customer to purchase Front Operation Panel's PCB.


 

22 Sep 2011

2) One Cycle button cannot light up for Rear & Front Operation Panel but the function is ok.

Check SPEL CT Output Bit 189 whether able to ON One Cycle LED button but NG.
Swapped “LOADER” Output I/O Module in PSUNT286 from another NS-7000, confirmed that Output I/O Module spoilt. 


 

22 Sep 2011

3) Start & Home buttons for Rear Operation Panel cannot work.

Previously this handler's Maintenance Switch was disable.
Check jumper connection(START) to bypass Maintenance/Normal mode.
Jumper connector connection seem loose, press the jumper connector.
Put back jumper(on START) in PSUNT286.
START connector cable disconnected from "START".
Power up handler.
Able to Home and START handler.
Clone this 20GB HDD by Clozilla.
Restore image to another 20GB HDD.
Power up handler with this HDD, confirm Homing & start operation.


 


 

Cause

1) Cover open” door switch not functional - Found VD1 zener diode burnt at Front Operational Panel

2) One Cycle button cannot light up for Rear & Front Operation Panel but the function is ok. - Found “LOADER” Output I/O Module in PSUNT286 spoilt.

3) Start button for Rear Operation Panel cannot work - Rear Operational panel is spoilt
