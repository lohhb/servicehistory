Service History

Subject: IDT Penang NS-7000 Output Hands vacuum unit leakage

Handler Model: NS-7000 (S/N: 18781)

Controller: RC520

Date: 21 Oct 2009


 

Symptom

NS-7000 Output Hands vacuum unit leakage. 15~16 Oct 2009

1.   After receive ejector valve, replace it and swap to output arm. Run it in ambient mode did not found any pick up error. Then, try to run in hot mode and found the valve malfunction again. Check the hander main body and shuttle plate did not found any current leakage.

2.   Suspect one of the heaters having current leakage, because of the current leakage. It may cause the 24v parts malfunction (advice by Suzuki-san).


 


 

Action

Direct replaced one complete vacuum unit set for Output arm.

Vacuum sensitivity setting for Output Hands to -50kpa.

Trigger vacuum for all output hands without blocking rubber cup, all around -40kpa.

Block my hands, reading around -88kpa.

In ambient mode, with dummy devices after output hands pickup 4 devices and arm move to unloader 1 tray, device "drop" error occurred.

Check output hands, device still remain at all pickup hands, all vacuum reading at around -40kpa.

Remove all devices from pickup hands, vacuum strength seem ok.

Remove black tubing and block metal fitting, reading around 40kpa., same reading for the rest of the hands.

Alan tried to loosen 2 screws on this new vacuum unit set(to allow for air leakage) but still the same problem.

Replaced to back to original vacuum unit set. This original vacuum unit set the 2 screws also lossen by Alan to allow for some air leakage.

(Alan discovered yesterday, that by having air leakage in for vacuum unit set, in hot mode there won't be device pickup or drop error at output arm).

Check the factory supply compress air, reading was around 0.49Mpa.

At 100 degC, run 2 trays of dummies, ok.

 

At 125 degC, using 1000pcs of devices, electrical check with tester, ok.


 


 

Cause

According to Suzuki-san, Output Hands vacuum unit leakage due to air seal in vacuum unit not sit properly.
