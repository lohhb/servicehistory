Service History

Subject: NS-8080 Input Hand E device dropped and vacuum sensor off

Handler Model: NS-8080 High UPH 8 Hands (S/N: 182262, H42-NS88)

Controller: RC520 (S/N: 03-30206)

Date: 5 Mar 2013


 

Symptom

ASE – H42-NS88 Input hand E device drop and vacuum sensor off error


 


 

Action

According to Choon Soon, he power off operational panel, removed the input arm top cover and resit connector on B/D driver(which is not related to hand E) and immediately no more error.


 Some findings as below.
1. All the vacuum and purge air tubing which was under the top cover still have some space allowance. Which means the top cover was not pressing on them.


 2. Input arm vacuum reading when turned on was –10 ~-15Kpa without device.Vacuum reading within range when sucked device.


 3. Counter air was weak. Regulator was close to 0mpa. Set it to 0.2mpa. Counter air on time of input arm set from 0.3s to 0.1s.


 4. HMI vacuum threshold setting at –35,-40-70. Normal.


 5. Removed input arm assembly side cover and checked CN10 connector on control board. Hand E signal line IN2 connected from distribution board to control board CN10.

Slightly pull the CN10 cable and connector came off.

Resit CN10 connector properly again.
Under monitoring.
