Subject: NS8080 has the hang up problem with Random and busy shuttle mode

Handler Model: Evaluation NS-8080(EEMS, S/N:181416)

Controller: RC520 (HMI ver 1.71 , SPEL CT3.11ds)

Date: 23 Jun 2008


 

Symptom

EEMS NS8080 has the same hang up issue as NS7000 after clean out when the lot ends.

EEMS NS8080 has the hang up problem with Random and busy shuttle mode.

Now they tried switch to 'Z' pick up and busy shuttle mode, encounter the same hang problem.

Hang up condition:

- 37.5 x 37.5 package

- long test time 14~16secs

- single site busy shuttle

- 3x7 tray matrix, 2x7 hotplate matrix

- random mode or 'Z' mode for input arm

- happened on HMi version 1.71

s/n:181415 (NS-8040)
